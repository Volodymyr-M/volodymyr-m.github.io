Frequently Asked Questions

What is Ingantt?

Ingantt is a project planning tool that integrates with Google Workspace, allowing users to create, manage, and track projects effectively with AI-powered features.

How do I start planning with Ingantt?

To start planning with Ingantt, simply sign up, create a project, and use the AI features to quickly generate a detailed plan or manually enter your tasks and resources.

Can I import my existing MS Project files?

Yes, Ingantt allows you to import your existing MS Project (.MPP or .XML) files for seamless integration and project continuity.
