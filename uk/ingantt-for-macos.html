Часті запитання

Що таке Ingantt для macOS?

Ingantt для macOS — це професійний інструмент керування проєктами, який працює нативно на вашому Mac. Він дає змогу створювати й керувати планами проєктів із підтримкою штучного інтелекту та сумісністю з MS Project.

Чи можу я імпортувати свої наявні файли Microsoft Project?

Так! Ingantt повністю підтримує імпорт файлів .MPP та .XML із Microsoft Project. Дані проєкту, включно із завданнями, ресурсами й залежностями, збережуться під час імпорту.

Як працює генерація проєктів з ШІ?

Наш AI-асистент може генерувати повні плани проєктів із коротких текстових описів. Просто опишіть цілі й вимоги вашого проєкту, і ШІ створить структурований план із завданнями, графіками та залежностями, базуючись на найкращих практиках.

Чи існує безкоштовна пробна версія для macOS?

Ні. Версія для macOS — це одноразова покупка без підписки чи безкоштовного пробного періоду.

Як придбати Ingantt для macOS?

Відвідайте Mac App Store і придбайте там. Ви володітимете додатком довічно без будь-яких повторних платежів.
