Що таке переглядач діаграм Ганта?

Переглядач діаграм Ганта — це інструмент, який відкриває файл діаграми Ганта (зазвичай Microsoft Project .MPP або .XML) і відображає проект як інтерактивну часову шкалу. Справжній переглядач дозволяє прокручувати графік, масштабувати, розгортати або згортати фази, і наводити курсор на завдання для перегляду залежностей, дат початку та завершення, а також ресурсів. Ingantt — це переглядач Ганта, який працює на Mac, iOS, iPad, Windows, Android та в браузері.

Як переглянути діаграму Ганта без Microsoft Project?

Відкрийте файл в Ingantt. Він напряму імпортує рідний формат .MPP та MS Project XML, тож файл від колеги, який користується MS Project, відкривається як повністю інтерактивна діаграма Ганта — без ліцензії MS Project та без встановлення Windows. Використовуйте версію для браузера для одноразового випадку або встановіть безкоштовну пробну версію для Mac, iOS, Windows чи Android. Завдання, залежності та критичний шлях приходять без змін.

Чи є безкоштовний переглядач Ганта?

Ingantt пропонує 7-денну безкоштовну пробну версію без кредитної картки. Пробна версія включає повний перегляд і редагування .MPP та MS Project XML файлів на всіх платформах — нічого не заблоковано за розподілом "переглядач" проти "редактор". Після пробного періоду платні плани починаються від 5,99 $ на користувача на місяць.

Чи можу я надрукувати або поділитися переглянутою діаграмою Ганта?

Так. Ingantt експортує діаграму Ганта в PDF або PNG для друку, надсилання електронною поштою або вставляння в слайд. Експорт PDF зберігає макет часової шкали на будь-якому рівні масштабування. Якщо вам потрібно згодом відкрити цей же .MPP в іншому місці, перегляньте наш посібник з відкриття .MPP файлів без Microsoft Project.
