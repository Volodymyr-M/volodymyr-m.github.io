Як відкрити .MPP файл без Microsoft Project?

Скористайтеся Ingantt у браузері. Він читає бінарний формат .MPP напряму — без ліцензії Microsoft Project, без віртуальної машини Windows, без конвертаційного сервісу, що присилає PDF на пошту. Ingantt імпортує кожне завдання, підзавдання, залежність, призначення ресурсу, етап та критичний шлях і відображає проект як повну інтерактивну діаграму Ганта. 7-денної безкоштовної пробної версії достатньо для більшості одноразових потреб відкрити файл, а кредитна картка для початку не потрібна.

Що таке переглядач Ганта і чи є Ingantt таким?

Переглядач Ганта — це інструмент, який відкриває файл діаграми Ганта (зазвичай Microsoft Project .MPP або .XML) і показує проект як інтерактивну часову шкалу. Справжній переглядач дозволяє прокручувати графік, збільшувати й зменшувати масштаб, розгортати або згортати фази та наводити курсор на завдання, щоб побачити залежності, дати початку й завершення та ресурси. Ingantt — це браузерний переглядач та редактор Ганта, тож ви можете відкрити той самий файл на будь-якому пристрої поруч і за потреби експортувати зміни назад у MS Project XML.

Чи можу я відкрити .MPP файл або переглянути діаграму Ганта у браузері?

Так. Ingantt працює повністю у браузері — Chrome, Safari, Firefox або Edge — на Mac, Windows, Linux, Chromebook, iPhone, iPad чи Android. Увійдіть з обліковим записом Google, завантажте .MPP файл або оберіть його з Google Drive — і повний проект відкривається у браузері з масштабуванням, прокруткою та підказками при наведенні. Нічого не встановлюється локально й нічого не завантажується. Це правильний шлях, якщо ви на робочому комп'ютері, куди не можна ставити нове ПЗ, або на Linux чи Chromebook, де Microsoft Project узагалі ніколи не був доступний.

Чи можу я роздрукувати свою діаграму Ганта або експортувати її у PDF чи PNG?

Так, усі три варіанти. Ingantt друкує діаграму Ганта прямо з браузера й експортує у PDF та PNG за будь-якого масштабу — тож потрібен вам односторінковий огляд чи часова шкала на кілька сторінок у розмір стіни, макет залишається чистим. PDF зазвичай обирають для поширення лише для читання зі стейкхолдерами, яким не потрібно чіпати оригінал; PNG зручний для вставки у презентації, документи Notion чи повідомлення в Slack; нативний друк найшвидший, коли потрібна паперова копія.

Чи можу я зберегти .MPP файл назад після відкриття в Ingantt?

Не як .MPP — Ingantt експортує у MS Project XML, який Microsoft Project відкриває напряму. Бінарний формат .MPP є пропрієтарним, недокументованим і фактично формат лише для запису з боку Microsoft; MS Project XML — це власний формат обміну Microsoft і переноситься без втрат. Тож ви можете відкрити .MPP в Ingantt, внести зміни, експортувати як MS Project XML і передати .xml файл колезі, який досі користується Microsoft Project — він відкриє його так само, як будь-який файл проекту. Для поширення лише для читання серед людей, які взагалі не користуються Project, правильний експорт — PDF або PNG.

Чи втрачу я залежності, ресурси або критичний шлях, відкриваючи .MPP поза Microsoft Project?

В Ingantt — ні, імпорт без втрат для важливих полів. Завдання, підзавдання, залежності finish-to-start, призначення ресурсів, обсяг робіт і тривалість, етапи та обчислення критичного шляху — все передається з оригінального .MPP файлу. Ingantt використовує ту саму внутрішню структуру проекту, що й Microsoft Project, тож діаграма Ганта, яку будували пів дня в Project, виглядає так само після відкриття в Ingantt. Безкоштовні переглядачі, які показують лише список завдань або зводять проект до зображення, втрачають граф залежностей — а це зазвичай найважливіша частина плану.

Чи є безкоштовний відкривач .MPP або переглядач діаграми Ганта?

В Ingantt є 7-денна безкоштовна пробна версія без кредитної картки на старт, і пробний період включає повний імпорт .MPP, перегляд, редагування, друк та експорт у PDF/PNG/XML — нічого не заблоковано за стіною «тільки для перегляду» проти «редактора». Більшості, кому потрібно відкрити або переглянути .MPP файл один чи кілька разів, безкоштовної пробної версії повністю вистачає. Після пробного періоду платні тарифи починаються від $5.99 за користувача на місяць. Якщо ви працюєте з підрядниками або клієнтами, які досі користуються Microsoft Project, платний тариф дешевший за один місяць Microsoft 365 Project.

Чи потрібно знати управління проектами, щоб користуватися переглядачем Ганта?

Ні. Ingantt спроектований так, щоб стейкхолдер не-PM — керівник, який перевіряє дату постачання, клієнт, який звіряє етап, новий член команди на адаптації — міг відкрити діаграму Ганта й прочитати її, не вивчаючи нічого нового. Ви прокручуєте часову шкалу, щоб побачити, коли що відбувається, наводите курсор на завдання, щоб побачити, хто над ним працює, й розгортаєте фазу, щоб побачити завдання всередині. Жодного жаргону й жодного кроку налаштування, крім відкриття файлу.
