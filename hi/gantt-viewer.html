गैंट व्यूअर — किसी भी डिवाइस पर .MPP या गैंट फ़ाइलें खोलें

Ingantt में .MPP या Microsoft Project XML फ़ाइल ड्रॉप करें और यह एक इंटरैक्टिव गैंट चार्ट के रूप में खुल जाएगी जिसे आप ज़ूम, स्क्रॉल, फ़िल्टर और प्रिंट कर सकते हैं — Mac, iPhone, iPad, Windows, Android या ब्राउज़र में बिना कुछ इंस्टॉल किए।

02एक वास्तविक गैंट व्यूअर को क्या करना चाहिए

असली .MPP फ़ाइलें खोलता है

Microsoft Project .MPP और .XML फ़ाइलों को मूल रूप से पढ़ता है — कार्य, उप-कार्य, निर्भरताएं, मील के पत्थर और संसाधन अक्षुण्ण रहते हैं।

इंटरैक्टिव, छवि नहीं

समयरेखा को ज़ूम करें, चरणों के माध्यम से स्क्रॉल करें, समूहों का विस्तार करें, कार्यों पर होवर करें। एक वास्तविक गैंट चार्ट, स्क्रीनशॉट नहीं।

किसी भी डिवाइस पर

Mac, iOS, iPad, Windows और Android के लिए मूल ऐप्स, साथ ही ब्राउज़र संस्करण — एक ही फ़ाइल को कहीं भी खोलें।

प्रोजेक्ट प्रबंधन ज्ञान की आवश्यकता नहीं

अगर आप बस यह देखना चाहते हैं कि गैंट क्या कहता है — तारीख, निर्भरता, कौन क्या करता है — तो आप बिना कुछ नया सीखे यह कर सकते हैं।

03Ingantt को एक्शन में देखें

04गैंट व्यूअर — अक्सर पूछे जाने वाले प्रश्न

मैं Microsoft Project के बिना गैंट चार्ट कैसे देखूं?

Ingantt में फ़ाइल खोलें। यह Microsoft Project के मूल .MPP प्रारूप और MS Project XML को सीधे आयात करता है, इसलिए MS Project का उपयोग करने वाले सहकर्मी की फ़ाइल पूरी तरह से इंटरैक्टिव गैंट चार्ट के रूप में खुलती है — बिना MS Project लाइसेंस या Windows इंस्टॉलेशन के। कार्य, निर्भरताएं और महत्वपूर्ण पथ पूर्ण रहते हैं।

क्या कोई मुफ्त गैंट व्यूअर है?

Ingantt 7 दिन का मुफ्त परीक्षण प्रदान करता है जिसमें क्रेडिट कार्ड की आवश्यकता नहीं है। परीक्षण में सभी प्लेटफ़ॉर्म पर .MPP और MS Project XML फ़ाइलों का पूर्ण दृश्य और संपादन शामिल है। परीक्षण के बाद, भुगतान योजनाएं $5.99 प्रति उपयोगकर्ता प्रति माह से शुरू होती हैं।

क्या मैं जो गैंट चार्ट देख रहा हूं उसे प्रिंट या शेयर कर सकता हूं?

हां। Ingantt गैंट चार्ट को PDF या PNG में निर्यात करता है ताकि आप इसे प्रिंट कर सकें, ईमेल में संलग्न कर सकें, या स्लाइड में पेस्ट कर सकें। PDF निर्यात किसी भी ज़ूम स्तर पर समयरेखा लेआउट को सुरक्षित रखता है। यदि आपको बाद में उसी .MPP को कहीं और खोलने की आवश्यकता है, तो हमारी Microsoft Project के बिना .MPP फ़ाइलें खोलना गाइड देखें।

Ingantt के साथ अपने गैंट चार्ट देखना शुरू करें
