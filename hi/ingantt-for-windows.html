07सामान्य प्रश्न

Ingantt for Windows क्या है?

Ingantt for Windows एक पेशेवर प्रोजेक्ट मैनेजमेंट टूल है जो आपके PC पर मूल रूप से चलता है। यह आपको AI-असिस्टेड प्लानिंग और MS Project संगतता के साथ प्रोजेक्ट प्लान बनाने और मैनेज करने की सुविधा देता है।

क्या मैं अपनी मौजूदा Microsoft Project फाइलें इम्पोर्ट कर सकता/सकती हूँ?

जी हाँ! Ingantt, Microsoft Project की .MPP और .XML फाइलों के इम्पोर्ट का पूर्ण समर्थन करता है। आपके प्रोजेक्ट डेटा (कार्य, संसाधन, निर्भरताएँ आदि) इम्पोर्ट प्रक्रिया के दौरान संरक्षित रहते हैं।

AI प्रोजेक्ट जेनरेशन कैसे काम करता है?

हमारा AI असिस्टेंट सरल टेक्स्ट विवरणों से संपूर्ण प्रोजेक्ट प्लान तैयार कर सकता है। बस अपने प्रोजेक्ट के लक्ष्य और आवश्यकताएँ बताएं, AI सर्वोत्तम प्रथाओं के आधार पर कार्य, समय-सीमा और निर्भरताओं सहित एक संरचित प्लान बना देगा।

क्या Windows संस्करण के लिए कोई निःशुल्क परीक्षण उपलब्ध है?

जी हाँ। Microsoft Store से इंस्टॉल करने के बाद, 7 दिन का निःशुल्क परीक्षण उपलब्ध है।

मैं Windows के लिए Ingantt कैसे खरीद सकता/सकती हूँ?

Microsoft Store पर जाएँ और वहीं से खरीदें। आपको यह एप हमेशा के लिए बिना किसी आवर्ती शुल्क के प्राप्त होगा।
