Visualizzatore Gantt — Domande frequenti

Cos'è un visualizzatore Gantt?

Un visualizzatore Gantt è uno strumento che apre un file di diagramma di Gantt — di solito un Microsoft Project .MPP o .XML — e mostra il progetto come una cronologia interattiva. Un visualizzatore vero permette di scorrere il cronoprogramma, zoomare, espandere o ridurre le fasi e passare sulle attività per vedere dipendenze, date di inizio e fine, e risorse. Ingantt è un visualizzatore Gantt che funziona su Mac, iOS, iPad, Windows, Android e nel browser, così puoi aprire lo stesso file su qualunque dispositivo.

Come visualizzo un diagramma di Gantt senza Microsoft Project?

Apri il file in Ingantt. Importa direttamente il formato .MPP nativo di Microsoft Project e MS Project XML, quindi un file ricevuto da un collega che usa MS Project si apre come un Gantt completamente interattivo — senza licenza MS Project né installazione Windows. Usa la versione browser per un'esigenza occasionale o installa la prova gratuita dell'app Mac, iOS, Windows o Android. Attività, dipendenze e il percorso critico arrivano intatti.

Posso visualizzare un file .MPP su Mac?

Sì. Ingantt ha un'app macOS nativa che apre direttamente i file .MPP e MS Project XML su qualsiasi Mac recente. Non servono macchine virtuali, Parallels o installazioni Windows. Scarica Ingantt dal Mac App Store, apri il file .MPP e vedi il Gantt completo con fasi, attività, dipendenze e percorso critico. Se non vuoi installare nulla, la versione browser funziona in Safari, Chrome o Firefox.

Posso visualizzare un Gantt su iPhone o iPad?

Sì. Ingantt ha app iOS native per iPhone e iPad che aprono file .MPP e MS Project XML come la versione Mac. Su iPad ottieni una cronologia Gantt a larghezza piena con pinch-to-zoom e scorrimento touch; su iPhone il layout si adatta allo schermo più piccolo per mantenere leggibili attività e milestone. È il modo più rapido di guardare un Gantt che un collega ti ha appena inviato quando sei lontano dalla scrivania.

Esiste un visualizzatore Gantt gratuito?

Ingantt offre una prova gratuita di 7 giorni senza carta di credito. La prova include la visualizzazione e modifica complete di file .MPP e MS Project XML su tutte le piattaforme — nulla è bloccato dietro una "edizione visualizzatore" vs "edizione editor". Dopo la prova, i piani a pagamento partono da 5,99 €/utente/mese. La maggior parte delle esigenze di visualizzazione occasionale rientra nella prova gratuita.

Posso vedere un Gantt nel browser senza installare nulla?

Sì. Ingantt ha una versione browser che apre file Gantt in Chrome, Safari, Firefox o Edge su Mac, Windows, Linux, Chromebook o mobile. Accedi con un account Google, apri un file .MPP da Google Drive o caricane uno, e il Gantt si carica nel browser con zoom, scorrimento e dettagli al passaggio del mouse. Niente viene installato, niente viene scaricato.

Devo conoscere il project management?

No. Ingantt è progettato perché un non esperto — un dirigente che verifica una data di consegna, un cliente che controlla una milestone, un nuovo membro che si orienta — possa aprire un Gantt e leggerlo senza imparare nulla. Scorri la cronologia per vedere quando succedono le cose, passa su un'attività per vedere chi la sta facendo, espandi una fase per vederne il contenuto. Nessun gergo, nessuna configurazione.

Posso stampare o condividere il Gantt che sto vedendo?

Sì. Ingantt esporta il Gantt in PDF o PNG per stamparlo, allegarlo a un'email o inserirlo in una presentazione. L'esportazione PDF mantiene il layout della cronologia a qualsiasi livello di zoom, utile per consegnare una versione in sola lettura a uno stakeholder. Se devi riaprire lo stesso .MPP altrove in seguito, consulta la nostra guida per aprire file .MPP senza Microsoft Project.
