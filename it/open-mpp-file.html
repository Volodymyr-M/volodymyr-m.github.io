04Aprire file .MPP — domande di chi è bloccato

Come apro un file .MPP?

Il modo più veloce è aprire il file in Ingantt. Un file .MPP è il formato binario nativo di Microsoft Project, e per default solo Microsoft Project lo apre correttamente — ma MS Project gira solo su Windows, costa diverse centinaia di euro e probabilmente non è quello che vuoi comprare per un singolo file. Ingantt importa .MPP direttamente, funziona su Mac, iPhone, iPad, Windows, Android e nel browser, con una prova gratuita di 7 giorni senza carta di credito. Apri l'app, trascina il .MPP, e il progetto completo appare in circa 30 secondi.

Come apro un file .MPP senza Microsoft Project?

Usa Ingantt. Legge direttamente il formato binario .MPP — niente licenza Microsoft Project, niente macchina virtuale Windows, niente servizio di conversione che ti manda un PDF via email. Ingantt importa ogni attività, sottoattività, dipendenza, assegnazione di risorse, milestone e il percorso critico, e rende il progetto come un diagramma di Gantt completo. La prova gratuita di 7 giorni copre la maggior parte delle situazioni occasionali, senza carta di credito.

Come apro un file .MPP su Mac?

Microsoft Project non gira su macOS — non esiste una versione Mac. La risposta non è "installare Project su Mac", ma "usare qualcosa che legga .MPP nativamente su Mac". Ingantt ha un'app macOS nativa sul Mac App Store che apre file .MPP direttamente su Mac Apple Silicon e Intel. Scaricala, apri il .MPP, e vedi il Gantt completo con attività, dipendenze e percorso critico. In alternativa la versione browser funziona in Safari, Chrome o Firefox.

Posso aprire un file .MPP su iPhone o iPad?

Sì. Ingantt ha app iOS native per iPhone e iPad sull'App Store che aprono file .MPP e MS Project XML direttamente. Su iPad ottieni una cronologia ampia con pinch-to-zoom e scorrimento touch — la cosa più vicina all'esperienza Microsoft Project desktop su un dispositivo iOS. Su iPhone il layout si adatta allo schermo più piccolo per mantenere leggibili attività, date e milestone. Spesso il modo più veloce per aprire un .MPP che un collega ti ha appena inviato.

Posso aprire un file .MPP online nel browser?

Sì. Ingantt ha una versione browser che apre file .MPP in Chrome, Safari, Firefox o Edge su Mac, Windows, Linux, Chromebook o mobile. Accedi con un account Google, carica il .MPP o selezionalo da Google Drive, e il progetto completo si apre nel browser. Niente viene installato localmente. È la via giusta su una macchina di lavoro dove non puoi installare software, o su Linux/Chromebook dove Microsoft Project non è mai stato un'opzione.

Esiste un modo gratuito per aprire file .MPP?

Ingantt offre una prova gratuita di 7 giorni senza carta di credito, e la prova include l'importazione e modifica complete di .MPP — nulla è bloccato. Per chi ha bisogno di aprire un .MPP solo una o poche volte, la prova copre la situazione interamente. Dopo, i piani a pagamento partono da 5,99 €/utente/mese. Se lavori con .MPP regolarmente, il piano è meno di un singolo mese di Microsoft 365 Project.

Perderò dipendenze, risorse o il percorso critico?

In Ingantt no — l'importazione è senza perdite per i campi importanti. Attività, sottoattività, dipendenze finish-to-start, assegnazioni di risorse, lavoro e durata, milestone e il calcolo del percorso critico arrivano tutti dal file .MPP originale. Ingantt usa la stessa struttura di progetto che Microsoft Project usa internamente, quindi un Gantt che ti è costato un pomeriggio in Project sembra uguale dopo averlo aperto in Ingantt. I visualizzatori gratuiti che mostrano solo liste di attività o appiattiscono il progetto in un'immagine perdono il grafo delle dipendenze.

Posso risalvare il file .MPP dopo averlo aperto?

Sì. Ingantt salva i progetti di nuovo nel formato nativo .MPP di Microsoft Project, più .XML, PDF e PNG. Puoi aprire un .MPP su Mac, iPhone o nel browser, fare modifiche, salvarlo come .MPP e passarlo a un collega che usa ancora Microsoft Project — lo aprirà come sempre. Se vuoi solo guardare il diagramma senza modificarlo, la nostra pagina Visualizzatore Gantt copre il caso di sola lettura.
