Come apro un file .MPP senza Microsoft Project?

Usa Ingantt nel tuo browser. Legge direttamente il formato binario .MPP — niente licenza Microsoft Project, niente macchina virtuale Windows, niente servizio di conversione che ti manda un PDF via email. Ingantt importa ogni attività, sotto-attività, dipendenza, assegnazione di risorse, milestone e il percorso critico, e rende il progetto come un diagramma di Gantt interattivo completo. La prova gratuita di 7 giorni basta per la maggior parte delle situazioni di apertura file occasionali, e non è richiesta carta di credito per iniziare.

Cos'è un visualizzatore Gantt, e Ingantt lo è?

Un visualizzatore Gantt è uno strumento che apre un file di diagramma di Gantt — di solito un file .MPP o .XML di Microsoft Project — e mostra il progetto come una timeline interattiva. Un vero visualizzatore ti permette di scorrere lungo la pianificazione, zoomare avanti e indietro, espandere o comprimere le fasi, e passare il mouse sulle attività per vedere dipendenze, date di inizio e fine, e risorse. Ingantt è un visualizzatore ed editor Gantt basato su browser, così puoi aprire lo stesso file su qualunque dispositivo hai a portata — ed esportare le modifiche di nuovo in MS Project XML quando serve.

Posso aprire un file .MPP o visualizzare un diagramma di Gantt nel browser?

Sì. Ingantt gira interamente nel browser — Chrome, Safari, Firefox o Edge — su Mac, Windows, Linux, Chromebook, iPhone, iPad o Android. Accedi con un account Google, carica il file .MPP o selezionalo da Google Drive, e il progetto completo si apre nel browser con zoom, scorrimento e passaggio del mouse per i dettagli. Nulla viene installato localmente e nulla viene scaricato. È la strada giusta se sei su una macchina di lavoro dove non puoi installare software nuovo, o su Linux o Chromebook dove Microsoft Project non è mai stato un'opzione.

Posso stampare il mio diagramma di Gantt, o esportarlo in PDF o PNG?

Sì, tutti e tre. Ingantt stampa il diagramma di Gantt direttamente dal browser, ed esporta in PDF e PNG a qualsiasi livello di zoom — quindi che tu abbia bisogno di una panoramica su singola pagina o di una timeline grande come un muro su più pagine, il layout resta pulito. Il PDF è la scelta abituale per la condivisione in sola lettura con stakeholder che non devono toccare il file originale; il PNG è comodo per inserirlo in presentazioni, documenti Notion o messaggi Slack; la stampa nativa è la più veloce quando vuoi solo una copia cartacea.

Posso salvare il file .MPP dopo averlo aperto in Ingantt?

Non come .MPP — Ingantt esporta in MS Project XML, che Microsoft Project apre direttamente. Il formato binario .MPP è proprietario, non documentato ed è essenzialmente un target di scrittura solo Microsoft; MS Project XML è il formato di interscambio ufficiale di Microsoft e fa andata e ritorno in modo pulito. Quindi puoi aprire un .MPP in Ingantt, fare modifiche, esportare come MS Project XML, e passare il file .xml a un collega che usa ancora Microsoft Project — lo apre come apre qualsiasi file di progetto. Per la condivisione in sola lettura con persone che non usano Project affatto, PDF o PNG è l'esportazione giusta.

Perderò dipendenze, risorse o il percorso critico aprendo un .MPP fuori da Microsoft Project?

In Ingantt no — l'importazione è senza perdite per i campi che contano. Attività, sotto-attività, dipendenze finish-to-start, assegnazioni di risorse, lavoro e durata, milestone e il calcolo del percorso critico arrivano tutti dal file .MPP originale. Ingantt usa la stessa struttura di progetto che Microsoft Project usa internamente, quindi un diagramma di Gantt che ha richiesto un pomeriggio da costruire in Project sembra lo stesso diagramma di Gantt dopo averlo aperto in Ingantt. I visualizzatori freemium che mostrano solo liste di attività o appiattiscono il progetto in un'immagine perdono il grafo delle dipendenze, che di solito è la parte più importante del piano.

Esiste un apri-.MPP o visualizzatore di diagrammi di Gantt gratuito?

Ingantt ha una prova gratuita di 7 giorni senza carta di credito richiesta per iniziare, e la prova include importazione .MPP completa, visualizzazione, modifica, stampa ed esportazione PDF/PNG/XML — nulla è bloccato dietro una divisione tra "edizione visualizzatore" ed "edizione editor". Per la maggior parte delle persone che devono aprire o visualizzare un file .MPP solo una o poche volte, la prova gratuita copre la situazione interamente. Dopo la prova, i piani a pagamento partono da $5.99 per utente al mese. Se lavori con fornitori o clienti che usano ancora Microsoft Project, il piano a pagamento costa meno di un singolo mese di Microsoft 365 Project.

Devo conoscere il project management per usare un visualizzatore Gantt?

No. Ingantt è progettato perché uno stakeholder non-PM — un dirigente che rivede una data di consegna, un cliente che controlla una milestone, un nuovo membro del team che si orienta — possa aprire un diagramma di Gantt e leggerlo senza imparare nulla di nuovo. Scorri la timeline per vedere quando accadono le cose, passi il mouse su un'attività per vedere chi ci sta lavorando, ed espandi una fase per vedere le attività al suo interno. Non c'è gergo da decifrare e nessun passaggio di configurazione oltre l'apertura del file.
