Domande frequenti

Che cos’è Ingantt per Windows?

Ingantt per Windows è uno strumento professionale di project management che funziona in modo nativo sul tuo PC. Ti consente di creare e gestire piani di progetto con pianificazione assistita dall’IA e compatibilità con MS Project.

Posso importare i miei file di Microsoft Project esistenti?

Sì! Ingantt supporta completamente l’importazione di file .MPP e .XML da Microsoft Project. I tuoi dati di progetto, incluse attività, risorse e dipendenze, verranno conservati durante l’importazione.

Come funziona la generazione di progetti con IA?

Il nostro assistente IA può generare piani di progetto completi partendo da semplici descrizioni di testo. Descrivi i tuoi obiettivi e requisiti e l’IA creerà un piano strutturato con attività, scadenze e dipendenze basate sulle best practice.

È disponibile una prova gratuita per la versione Windows?

Sì. Dopo l’installazione dall’app Microsoft Store, è disponibile una prova gratuita di 7 giorni.

Come posso acquistare Ingantt per Windows?

Visita il Microsoft Store e acquistalo lì. Possederai l’app a tempo indeterminato senza costi ricorrenti.
