Informativa sulla Privacy

Data di entrata in vigore: 22 febbraio 2026 Ultimo aggiornamento: 22 febbraio 2026

Ingantt Development (“noi”, “nostro/a/i/e”) gestisce l’applicazione Ingantt (l’“App”) e il sito web https://ingantt.com (il “Sito Web”). La presente Informativa sulla Privacy descrive come raccogliamo, utilizziamo e condividiamo le informazioni quando Lei utilizza l’App su tutte le piattaforme supportate (iOS, Android, macOS, Windows e Web, incluso tramite Google Workspace Marketplace).

1. Informazioni che raccogliamo

1.1 Informazioni fornite dall’utente

  • Informazioni dell’account Google: Se effettua l’accesso con Google, riceviamo il Suo indirizzo e-mail, il nome visualizzato e l’URL della foto del profilo da Google. Sulle piattaforme native (iOS, Android, macOS, Windows), l’accesso è facoltativo e abilita l’archiviazione dei file su Google Drive. Nella versione Web, Google Sign-In è obbligatorio e viene utilizzato per l’archiviazione dei file su Google Drive e la gestione degli abbonamenti. Quando utilizza per la prima volta la versione Web, viene creato un record utente sul nostro server contenente il Suo indirizzo e-mail, il nome, la lingua e la data di registrazione.

  • Informazioni su abbonamenti e fatturazione (solo Web): Quando acquista un abbonamento tramite il Web, raccogliamo il Suo nome, indirizzo e-mail, indirizzo postale, numero di telefono e dettagli relativi al pagamento. Queste informazioni vengono trasmesse a WayForPay, il nostro processore di pagamento, per l’elaborazione della transazione.

  • Feedback: Quando invia un feedback tramite l’App, raccogliamo il testo del Suo messaggio. Lei può facoltativamente allegare il file di progetto attualmente aperto per aiutarci a diagnosticare eventuali problemi.

  • Input per la funzionalità IA: Quando utilizza la funzionalità “Create with AI”, il prompt testuale da Lei fornito viene inviato al nostro server backend per l’elaborazione da parte di Anthropic (Claude).

  • File di progetto: I file di progetto che crea e modifica rimangono sotto il Suo controllo. I file vengono archiviati localmente sul Suo dispositivo o nel Suo account Google Drive (in una cartella denominata “Ingantt Schedules”). Quando apre un file in un formato non nativo (come MPP, GAN, XER o altri formati di file di progetto supportati), il file viene trasmesso al nostro server per la conversione del formato. I file vengono elaborati solo in memoria e non vengono archiviati sul nostro server.

1.2 Informazioni raccolte automaticamente

  • Metadati dell’App: Ogni richiesta al nostro backend include la versione dell’App, il nome della piattaforma (iOS, Android, macOS, Windows o Web), la lingua del sistema e, se ha effettuato l’accesso, il Suo indirizzo e-mail.

  • Segnalazioni di crash e errori: Utilizziamo Sentry (ospitato nell’UE) per raccogliere segnalazioni di crash e log degli errori in produzione. Questi includono stack trace, messaggi di errore, versione dell’App e informazioni sulla piattaforma. Non raccogliamo intenzionalmente dati personali tramite le segnalazioni di crash.

  • Analisi web (solo Web): La versione Web dell’App utilizza Google Analytics 4 per raccogliere analisi di utilizzo standard, incluse visualizzazioni di pagina e dati di sessione. Google Analytics può impostare cookie nel Suo browser. Le versioni native per dispositivi mobili e desktop dell’App non includono tracciamento analitico.

  • Preferenze utente: L’App archivia le Sue preferenze locali (come il tema, le opzioni di visualizzazione, le larghezze delle colonne e l’elenco dei file recenti) sul Suo dispositivo utilizzando i meccanismi di archiviazione standard della piattaforma. Queste preferenze non vengono trasmesse ai nostri server.

1.3 Informazioni che non raccogliamo

  • Non utilizziamo i Suoi dati per addestrare modelli di intelligenza artificiale o machine learning.
  • Non pubblichiamo annunci pubblicitari e non integriamo alcun SDK pubblicitario.
  • Non raccogliamo dati sulla posizione, contatti, foto, input della fotocamera o del microfono.
  • Non utilizziamo identificatori del dispositivo a scopi di tracciamento.
  • Il backup su cloud di Android è esplicitamente disabilitato per l’App.

2. Come utilizziamo le Sue informazioni

Utilizziamo le informazioni raccolte per i seguenti scopi:

  • Fornire e gestire l’App: Elaborare le Sue richieste, abilitare l’archiviazione e la sincronizzazione dei file tramite Google Drive, convertire i formati dei file e fornire la creazione di progetti basata sull’IA.
  • Gestire gli abbonamenti (solo Web): Elaborare i pagamenti degli abbonamenti Web, verificare lo stato dell’abbonamento e gestire il Suo account.
  • Inviare e-mail transazionali (solo Web): Inviarle notifiche relative all’account, come messaggi di benvenuto, promemoria sulla scadenza del periodo di prova, conferme di abbonamento e avvisi di annullamento.
  • Migliorare la stabilità e le prestazioni: Analizzare le segnalazioni di crash, i dati sugli errori e la telemetria lato server per identificare e correggere i bug.
  • Comprendere i modelli di utilizzo (solo Web): Utilizzare dati analitici aggregati per comprendere come viene utilizzata l’App Web e migliorare l’esperienza utente.
  • Rispondere alle richieste di supporto: Utilizzare i feedback inviati per diagnosticare i problemi e migliorare l’App.

3. Come condividiamo le Sue informazioni

Non vendiamo le Sue informazioni personali. Condividiamo le informazioni solo nelle seguenti circostanze:

  • Fornitori di servizi: Utilizziamo i seguenti fornitori di servizi di terze parti per il funzionamento dell’App:
    • Sentry (Functional Software, Inc.) — segnalazione di crash e errori lato client, dati elaborati nell’UE
    • Microsoft Application Insights (Microsoft Corporation) — telemetria e monitoraggio degli errori lato server
    • Google Analytics (Google LLC) — analisi di utilizzo web (solo versione Web)
    • Google APIs (Google LLC) — autenticazione e archiviazione file su Google Drive
    • WayForPay — elaborazione dei pagamenti degli abbonamenti (solo versione Web)
    • Resend (Resend, Inc.) — invio di e-mail transazionali (solo versione Web)
    • Anthropic (Claude) — elaborazione dei prompt per la creazione di progetti basata sull’IA
    • Microsoft Azure — hosting del backend e archiviazione dei dati (regione East US)
  • Obblighi di legge: Possiamo divulgare informazioni se richiesto dalla legge o in risposta a procedimenti legali validi.

4. Archiviazione e sicurezza dei dati

  • Dati utente lato server (solo Web): Il Suo indirizzo e-mail, nome, lingua, data di registrazione e stato dell’abbonamento sono archiviati in Azure Table Storage sul nostro server backend.
  • I token OAuth (token di accesso, aggiornamento e ID di Google) sono archiviati in storage sicuro della piattaforma (iOS Keychain, Android Keystore, macOS Keychain o equivalente della piattaforma).
  • Lo stato dell’abbonamento (solo Web) viene memorizzato nella cache locale in storage sicuro per l’accesso offline.
  • Le richieste API al nostro backend sono firmate con HMAC-MD5 per la verifica dell’integrità.
  • L’App opera all’interno dei sandbox della piattaforma ove disponibili (macOS App Sandbox, iOS App Sandbox).
  • Utilizziamo HTTPS per tutte le comunicazioni di rete.

4a. Conservazione dei dati

  • Record utente lato server (solo Web): Il Suo record utente e la cronologia degli abbonamenti vengono conservati per tutto il tempo di esistenza del Suo account. Per richiedere la cancellazione dei Suoi dati, La preghiamo di contattarci all’indirizzo support@ingantt.com.
  • File di progetto: I file inviati per la conversione del formato vengono elaborati in memoria e non vengono conservati.
  • Prompt IA: I prompt testuali inviati per la creazione di progetti basata sull’IA non vengono archiviati sul nostro server.
  • Feedback: I messaggi di feedback vengono inoltrati al nostro team di supporto via e-mail e non vengono archiviati in un database.
  • Dati locali: Le preferenze, i token OAuth e lo stato dell’abbonamento memorizzato nella cache rimangono sul Suo dispositivo fino alla disconnessione o alla disinstallazione dell’App.

5. Le Sue scelte e i Suoi diritti

  • L’accesso è facoltativo sulle piattaforme native: Su iOS, Android, macOS e Windows, è possibile utilizzare l’App senza accedere con Google. L’accesso abilita l’archiviazione dei file su Google Drive. Nella versione Web, Google Sign-In è obbligatorio per il funzionamento dell’App, l’archiviazione dei file e la gestione degli abbonamenti.
  • Accesso a Google Drive: L’App utilizza l’ambito limitato drive.file, che consente l’accesso solo ai file creati dall’App o esplicitamente aperti da Lei nell’App — non all’intero Google Drive.
  • Allegati al feedback: Allegare il file di progetto quando si invia un feedback è facoltativo e richiede il Suo consenso esplicito.
  • Analisi web: Lei può utilizzare strumenti di blocco dei cookie o di opt-out basati sul browser per limitare la raccolta di dati di Google Analytics nella versione Web.
  • E-mail transazionali: Nella versione Web, riceverà e-mail relative all’account (benvenuto, promemoria del periodo di prova, notifiche sull’abbonamento). Si tratta di comunicazioni di servizio necessarie per la gestione del Suo abbonamento.
  • Cancellazione dei dati: Lei può disconnettersi dal Suo account Google all’interno dell’App in qualsiasi momento, il che rimuove i token OAuth archiviati dal dispositivo. Per richiedere la cancellazione del Suo record utente e di eventuali dati associati sui nostri server, La preghiamo di contattarci all’indirizzo support@ingantt.com.

6. Privacy dei minori

L’App non è destinata a minori di 13 anni (o l’età applicabile nella Sua giurisdizione). Non raccogliamo consapevolmente informazioni personali da minori. Se ritiene che un minore ci abbia fornito informazioni personali, La preghiamo di contattarci all’indirizzo support@ingantt.com affinché possiamo adottare le misure appropriate.

7. Trasferimenti internazionali di dati

Il nostro server backend è ospitato su Microsoft Azure nella regione East US. Le segnalazioni di crash vengono elaborate da Sentry nell’UE. Se Lei si trova al di fuori degli Stati Uniti, le Sue informazioni possono essere trasferite e trattate negli Stati Uniti o in altre giurisdizioni in cui operano i nostri fornitori di servizi.

8. Modifiche alla presente Informativa sulla Privacy

Possiamo aggiornare periodicamente la presente Informativa sulla Privacy. La informeremo di eventuali modifiche sostanziali aggiornando la data “Ultimo aggiornamento” in cima alla presente informativa. L’uso continuato dell’App dopo eventuali modifiche costituisce la Sua accettazione dell’informativa aggiornata.

9. Informative specifiche per piattaforma

iOS e macOS (Apple)

  • L’App è un’applicazione a pagamento distribuita tramite Apple App Store. Le pratiche sulla privacy di Apple si applicano alla transazione di acquisto.
  • L’App richiede solo i permessi di accesso ai file concessi esplicitamente tramite il selettore di file di sistema.

Android

  • L’App è un’applicazione a pagamento distribuita tramite Google Play. Le pratiche sulla privacy di Google si applicano alla transazione di acquisto.
  • L’App richiede permessi di lettura/scrittura dell’archiviazione per le operazioni locali sui file e il permesso di accesso a Internet per la connessione di rete.

Windows

  • L’App è un’applicazione a pagamento distribuita tramite Microsoft Store. Le pratiche sulla privacy di Microsoft si applicano alla transazione di acquisto.

Web

  • La versione Web utilizza un modello in abbonamento. I pagamenti degli abbonamenti vengono elaborati tramite WayForPay.
  • L’App è disponibile anche tramite Google Workspace Marketplace.
  • I cookie di Google Analytics vengono impostati sul dominio web.ingantt.com.
  • L’autenticazione Google OAuth utilizza una voce temporanea nel localStorage per il callback di accesso, che viene immediatamente cancellata dopo l’uso.
