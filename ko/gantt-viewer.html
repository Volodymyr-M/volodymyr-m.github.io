간트 뷰어 — 자주 묻는 질문

간트 뷰어란 무엇인가요?

간트 뷰어는 간트 차트 파일(일반적으로 Microsoft Project .MPP 또는 .XML)을 열어 프로젝트를 인터랙티브 타임라인으로 표시하는 도구입니다. 진짜 뷰어는 일정 스크롤, 확대/축소, 단계 펼치기/접기, 작업에 호버하여 종속성, 시작/종료 날짜, 리소스를 볼 수 있습니다. Ingantt는 Mac, iOS, iPad, Windows, Android, 브라우저에서 작동하는 간트 뷰어입니다.

Microsoft Project 없이 간트 차트를 어떻게 봅니까?

Ingantt에서 파일을 여세요. Microsoft Project의 네이티브 .MPP 형식과 MS Project XML을 직접 가져오므로 MS Project를 사용하는 동료에게 받은 파일이 완전히 인터랙티브한 간트 차트로 열립니다. MS Project 라이선스나 Windows 설치가 필요하지 않습니다. 작업, 종속성, 크리티컬 패스가 모두 그대로 유지됩니다.

무료 간트 차트 뷰어가 있나요?

Ingantt는 신용카드 없이 7일 무료 체험을 제공합니다. 체험판에는 모든 플랫폼에서 .MPP 및 MS Project XML 파일의 전체 보기 및 편집 기능이 포함되어 있습니다. "뷰어 에디션"과 "에디터 에디션"의 구분이 없습니다. 체험 후에는 유료 플랜이 사용자당 월 $5.99부터 시작합니다.

보고 있는 간트 차트를 인쇄하거나 공유할 수 있나요?

예. Ingantt는 간트 차트를 PDF 또는 PNG로 내보내어 인쇄하거나 이메일에 첨부하거나 슬라이드 덱에 붙여넣을 수 있습니다. PDF 내보내기는 어떤 확대 수준에서도 타임라인 레이아웃을 유지합니다. 동일한 .MPP를 나중에 다른 곳에서 열어야 한다면 Microsoft Project 없이 .MPP 파일 열기 가이드를 참조하세요.
