어디서든 인쇄 및 내보내기

04.MPP 파일 열기와 보기 — 사람들이 실제로 묻는 질문

Microsoft Project 없이 .MPP 파일을 어떻게 여나요?

브라우저에서 Ingantt를 사용하세요. .MPP 바이너리 형식을 직접 읽으므로 — Microsoft Project 라이선스, Windows 가상 머신, PDF를 이메일로 보내주는 변환 서비스가 필요하지 않습니다. Ingantt는 모든 작업, 하위 작업, 종속성, 리소스 할당, 마일스톤과 크리티컬 패스를 가져와 프로젝트를 완전한 대화형 간트 차트로 렌더링합니다. 7일 무료 체험은 대부분의 일회성 파일 열람 상황에 충분하며, 시작하는 데 신용카드도 필요 없습니다.

간트 뷰어란 무엇이며, Ingantt가 그에 해당하나요?

간트 뷰어는 간트 차트 파일 — 보통 Microsoft Project .MPP 또는 .XML 파일 — 을 열어 프로젝트를 대화형 타임라인으로 표시하는 도구입니다. 진짜 뷰어라면 일정을 가로로 스크롤하고, 확대·축소하며, 단계를 펼치거나 접고, 작업 위에 마우스를 올려 종속성, 시작일과 종료일, 리소스를 볼 수 있어야 합니다. Ingantt는 브라우저 기반의 간트 뷰어이자 에디터이므로, 손에 있는 어떤 기기에서든 동일한 파일을 열 수 있고 — 필요하면 변경 내역을 MS Project XML로 다시 내보낼 수도 있습니다.

브라우저에서 .MPP 파일을 열거나 간트 차트를 볼 수 있나요?

네. Ingantt는 Chrome, Safari, Firefox 또는 Edge 등 브라우저에서 Mac, Windows, Linux, Chromebook, iPhone, iPad 또는 Android 전반에 걸쳐 완전히 동작합니다. Google 계정으로 로그인하고 .MPP 파일을 업로드하거나 Google Drive에서 선택하면, 전체 프로젝트가 브라우저에서 확대·축소, 스크롤, 마우스오버 세부 정보 보기 기능과 함께 열립니다. 로컬에 설치되는 것도, 다운로드되는 것도 없습니다. 새 소프트웨어를 설치할 수 없는 업무용 기기이거나, Microsoft Project가 애초에 지원된 적 없는 Linux 또는 Chromebook이라면 이 경로가 올바른 선택입니다.

간트 차트를 인쇄하거나 PDF 또는 PNG로 내보낼 수 있나요?

네, 세 가지 모두 가능합니다. Ingantt는 브라우저에서 바로 간트 차트를 인쇄하고, 원하는 줌 레벨에서 PDF와 PNG로 내보냅니다 — 단일 페이지 개요가 필요하든 여러 페이지에 걸친 벽 크기 타임라인이 필요하든, 레이아웃은 깔끔하게 유지됩니다. PDF는 원본 파일에 손댈 필요가 없는 이해관계자와의 읽기 전용 공유에 보통 적합하고, PNG는 슬라이드, Notion 문서, Slack 메시지에 넣기에 편리하며, 하드카피가 필요할 때는 네이티브 인쇄가 가장 빠릅니다.

Ingantt에서 연 .MPP 파일을 다시 저장할 수 있나요?

.MPP 형식으로는 저장할 수 없습니다 — Ingantt는 MS Project XML로 내보내며, Microsoft Project는 이 형식을 바로 엽니다. .MPP 바이너리 형식은 독점적이고 문서화되지 않았으며 사실상 Microsoft만 쓰기 대상으로 삼을 수 있는 반면, MS Project XML은 Microsoft 자체 교환 형식이므로 양방향으로 깔끔하게 왕복됩니다. 따라서 Ingantt에서 .MPP를 열고, 편집을 수행한 뒤, MS Project XML로 내보내 .xml 파일을 여전히 Microsoft Project를 쓰는 동료에게 전달할 수 있습니다 — 그들은 여느 프로젝트 파일을 여는 방식 그대로 열면 됩니다. Project를 전혀 사용하지 않는 사람들과 읽기 전용으로 공유할 때는 PDF나 PNG가 올바른 내보내기입니다.

Microsoft Project 밖에서 .MPP를 열면 종속성, 리소스, 크리티컬 패스를 잃게 되나요?

Ingantt에서는 아닙니다 — 중요한 필드에 대해 가져오기는 무손실입니다. 작업, 하위 작업, finish-to-start 종속성, 리소스 할당, 작업량과 기간, 마일스톤, 크리티컬 패스 계산이 모두 원본 .MPP 파일에서 그대로 들어옵니다. Ingantt는 Microsoft Project가 내부적으로 사용하는 것과 동일한 프로젝트 구조를 사용하므로, Project에서 한나절 걸려 만든 간트 차트는 Ingantt에서 열어도 같은 간트 차트로 보입니다. 작업 목록만 보여주거나 프로젝트를 이미지로 평면화하는 프리미엄 뷰어들은 보통 계획에서 가장 중요한 부분인 종속성 그래프를 잃게 됩니다.

무료 .MPP 오프너나 간트 차트 뷰어가 있나요?

Ingantt는 시작 시 신용카드가 필요 없는 7일 무료 체험을 제공하며, 체험에는 .MPP 전체 가져오기, 보기, 편집, 인쇄, PDF/PNG/XML 내보내기가 모두 포함됩니다 — "뷰어 에디션"과 "에디터 에디션"으로 갈라 기능을 잠그는 방식이 아닙니다. .MPP 파일을 한 번 또는 몇 번만 열거나 보려는 대부분의 사용자에게는 무료 체험만으로 상황이 완전히 해결됩니다. 체험 이후 유료 플랜은 사용자당 월 $5.99부터 시작합니다. 여전히 Microsoft Project를 사용하는 외주 업체나 고객과 일한다면, 유료 플랜은 Microsoft 365 Project 한 달 비용보다도 저렴합니다.

간트 뷰어를 사용하려면 프로젝트 관리를 알아야 하나요?

아니요. Ingantt는 PM이 아닌 이해관계자 — 납기일을 검토하는 임원, 마일스톤을 확인하는 고객, 상황을 파악하려는 신입 팀원 — 가 새로운 것을 배우지 않고도 간트 차트를 열고 읽을 수 있도록 설계되어 있습니다. 타임라인을 스크롤해 언제 무엇이 일어나는지 보고, 작업 위에 마우스를 올려 담당자를 확인하고, 단계를 펼쳐 그 안의 작업들을 살펴보세요. 해독해야 할 전문 용어도 없고, 파일을 여는 것 외의 설정 단계도 없습니다.
