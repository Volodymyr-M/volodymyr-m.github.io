.MPP 파일 열기 — 막혔을 때의 질문

.MPP 파일을 어떻게 여나요?

가장 빠른 방법은 Ingantt에서 파일을 여는 것입니다. .MPP 파일은 Microsoft Project의 네이티브 바이너리 형식이며 기본적으로 Microsoft Project만이 이를 깔끔하게 열 수 있지만, MS Project는 Windows에서만 실행되고 수백 달러가 들며 단일 파일을 위해 구매하기에는 부적합합니다. Ingantt는 .MPP를 직접 가져오며 Mac, iPhone, iPad, Windows, Android, 브라우저에서 작동하고 신용카드 없이 7일 무료 체험을 제공합니다.

Mac에서 .MPP 파일을 어떻게 여나요?

Microsoft Project는 macOS에서 실행되지 않습니다. 따라서 답은 "Mac에 Project 설치"가 아니라 "Mac에서 네이티브로 .MPP를 읽는 도구를 사용"입니다. Ingantt는 Mac App Store에 네이티브 macOS 앱을 제공하며 Apple Silicon 및 Intel Mac에서 .MPP 파일을 직접 엽니다. 다운로드 후 .MPP를 열면 작업, 종속성, 크리티컬 패스가 포함된 전체 간트 차트가 표시됩니다.

iPhone이나 iPad에서 .MPP 파일을 열 수 있나요?

예. Ingantt는 App Store에 iPhone 및 iPad용 네이티브 iOS 앱을 제공하며 .MPP 및 MS Project XML 파일을 직접 엽니다. iPad에서는 핀치 줌과 터치 스크롤이 있는 넓은 타임라인을 얻을 수 있으며, 이는 iOS 기기에서 Microsoft Project 데스크톱 경험에 가장 가까운 것입니다.

.MPP 파일을 연 후 다시 저장할 수 있나요?

예. Ingantt는 프로젝트를 Microsoft Project의 네이티브 .MPP 형식으로 다시 저장하며 .XML, PDF, PNG도 지원합니다. Mac, iPhone 또는 브라우저에서 .MPP를 열고 편집한 후 .MPP로 저장하여 여전히 Microsoft Project를 사용하는 동료에게 전달할 수 있습니다. 차트를 보기만 하고 편집하거나 다시 저장할 계획이 없다면 간트 뷰어 페이지를 참조하세요.
