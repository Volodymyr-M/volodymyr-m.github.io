새 버전: MPP 파일 열기

중요한 기능을 추가한 Ingantt 업데이트를 발표하게 되어 기쁩니다!

아시다시피, Ingantt는 Microsoft Project로 만든 XML 파일을 열 수 있습니다. 하지만 이 포맷은 이 인기 있는 앱의 기본 포맷이 아니며, 대개 MPP(“.mpp” 확장자)라는 바이너리 포맷으로 파일을 저장합니다.

새 업데이트에서는 XML 파일을 열듯이 MPP 파일도 열 수 있게 되었습니다. 파일 선택기에서 MPP 파일을 고르면, 그대로 열립니다.

이 기능에는 한 가지 주의해야 할 점이 있습니다. Ingantt의 대부분 기능은 인터넷 연결이 필요 없지만, MPP 파일을 열기 위해서는 인터넷 연결이 필요합니다. 파일을 XML로 변환하기 위해 서버로 전송하는 과정이 필요하기 때문입니다. 데이터 교환은 안전한 연결을 통해 이뤄지며, 파일은 서버에 저장되지 않으므로 개인정보와 보안 측면에서 걱정하지 않으셔도 됩니다.

이번에 추가된 Ingantt의 새로운 기능을 마음껏 활용해 보시길 바랍니다. 더 많은 새로운 기능도 곧 출시될 예정입니다!
