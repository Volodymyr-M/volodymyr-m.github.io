07자주 묻는 질문

Ingantt for Windows란 무엇인가요?

Ingantt for Windows는 PC에서 네이티브로 실행되는 전문 프로젝트 관리 도구입니다. AI 기반 계획 및 MS Project 호환 기능을 통해 프로젝트 계획과 관리를 손쉽게 수행할 수 있습니다.

기존 Microsoft Project 파일을 가져올 수 있나요?

네! Ingantt는 Microsoft Project의 .MPP와 .XML 파일 가져오기를 완벽히 지원합니다. 작업, 리소스, 종속성 등 모든 프로젝트 데이터가 가져오기 과정에서 그대로 유지됩니다.

AI를 활용한 프로젝트 생성은 어떻게 작동하나요?

저희 AI 어시스턴트는 간단한 텍스트 설명만으로 완전한 프로젝트 계획을 생성할 수 있습니다. 프로젝트 목표와 요구 사항을 설명하면, AI가 모범 사례에 기반해 작업, 일정, 종속성을 포함하는 체계적인 계획을 만들어줍니다.

Windows 버전에 대한 무료 체험 기간이 있나요?

네. Microsoft Store에서 앱을 설치한 후 7일 동안 무료 체험을 이용할 수 있습니다.

Windows용 Ingantt는 어떻게 구매하나요?

Microsoft Store에서 구매할 수 있습니다. 한 번 구매하면 반복 과금 없이 계속 소유하게 됩니다.
