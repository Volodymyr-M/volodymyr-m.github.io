개인정보처리방침

시행일: 2026년 2월 22일 최종 업데이트: 2026년 2월 22일

Ingantt Development("당사", "저희", "우리의")는 Ingantt 애플리케이션("앱")과 https://ingantt.com 웹사이트("웹사이트")를 운영합니다. 본 개인정보처리방침은 귀하가 iOS, Android, macOS, Windows 및 Web(Google Workspace Marketplace를 통한 사용 포함)을 포함한 모든 지원 플랫폼에서 앱을 사용할 때 당사가 정보를 수집, 사용 및 공유하는 방법을 설명합니다.

1. 당사가 수집하는 정보

1.1 귀하가 제공하는 정보

  • Google 계정 정보: Google로 로그인하면 당사는 Google로부터 귀하의 이메일 주소, 표시 이름 및 프로필 사진 URL을 수신합니다. 네이티브 플랫폼(iOS, Android, macOS, Windows)에서는 로그인이 선택 사항이며 Google Drive 파일 저장을 활성화합니다. 웹 버전에서는 Google 로그인이 필수이며 Google Drive 파일 저장 및 구독 관리에 사용됩니다. 웹 버전을 처음 사용할 때 귀하의 이메일 주소, 이름, 로캘 및 등록 날짜를 포함하는 사용자 기록이 당사 서버에 생성됩니다.

  • 구독 및 결제 정보 (웹 전용): 웹을 통해 구독을 구매할 때 당사는 귀하의 이름, 이메일 주소, 우편 주소, 전화번호 및 결제 관련 세부 정보를 수집합니다. 이 정보는 거래 처리를 위해 당사의 결제 처리업체인 WayForPay에 전송됩니다.

  • 피드백: 앱을 통해 피드백을 제출할 때 당사는 귀하의 메시지 텍스트를 수집합니다. 문제 진단을 돕기 위해 현재 열려 있는 프로젝트 파일을 선택적으로 첨부할 수 있습니다.

  • AI 기능 입력: "Create with AI" 기능을 사용할 때, 귀하가 제공한 텍스트 프롬프트는 Anthropic (Claude)에 의한 처리를 위해 당사의 백엔드 서버로 전송됩니다.

  • 프로젝트 파일: 귀하가 생성하고 편집하는 프로젝트 파일은 귀하의 관리 하에 있습니다. 파일은 귀하의 기기에 로컬로 또는 귀하의 Google Drive 계정("Ingantt Schedules"라는 폴더)에 저장됩니다. 네이티브가 아닌 형식(MPP, GAN, XER 또는 기타 지원되는 프로젝트 파일 형식 등)의 파일을 열 때, 해당 파일은 형식 변환을 위해 당사 서버로 전송됩니다. 파일은 메모리에서만 처리되며 당사 서버에 저장되지 않습니다.

1.2 자동으로 수집되는 정보

  • 앱 메타데이터: 당사 백엔드에 대한 각 요청에는 앱 버전, 플랫폼 이름(iOS, Android, macOS, Windows 또는 Web), 시스템 로캘 및 로그인된 경우 귀하의 이메일 주소가 포함됩니다.

  • 충돌 및 오류 보고서: 당사는 프로덕션 환경에서 충돌 보고서 및 오류 로그를 수집하기 위해 Sentry(EU에서 호스팅)를 사용합니다. 여기에는 스택 트레이스, 오류 메시지, 앱 버전 및 플랫폼 정보가 포함됩니다. 당사는 충돌 보고서를 통해 의도적으로 개인 데이터를 수집하지 않습니다.

  • 웹 분석 (웹 전용): 앱의 웹 버전은 Google Analytics 4를 사용하여 페이지 조회수 및 세션 데이터를 포함한 표준 사용 분석 데이터를 수집합니다. Google Analytics는 귀하의 브라우저에 쿠키를 설정할 수 있습니다. 네이티브 모바일 및 데스크톱 버전의 앱에는 분석 추적이 포함되어 있지 않습니다.

  • 사용자 환경 설정: 앱은 표준 플랫폼 저장 메커니즘을 사용하여 귀하의 로컬 환경 설정(테마, 표시 옵션, 열 너비 및 최근 파일 목록 등)을 기기에 저장합니다. 이러한 환경 설정은 당사 서버로 전송되지 않습니다.

1.3 당사가 수집하지 않는 정보

  • 당사는 귀하의 데이터를 인공지능 또는 머신러닝 모델 학습에 사용하지 않습니다.
  • 당사는 광고를 제공하지 않으며 어떠한 광고 SDK도 통합하지 않습니다.
  • 당사는 위치 데이터, 연락처, 사진, 카메라 입력 또는 마이크 입력을 수집하지 않습니다.
  • 당사는 추적 목적으로 기기 식별자를 사용하지 않습니다.
  • Android 클라우드 백업은 앱에 대해 명시적으로 비활성화되어 있습니다.

2. 귀하의 정보 사용 방법

당사는 수집한 정보를 다음 목적으로 사용합니다:

  • 앱의 제공 및 운영: 귀하의 요청을 처리하고, Google Drive를 통한 파일 저장 및 동기화를 활성화하며, 파일 형식을 변환하고, AI 기반 프로젝트 생성을 제공합니다.
  • 구독 처리 (웹 전용): 웹 구독 결제를 처리하고, 구독 상태를 확인하며, 귀하의 계정을 관리합니다.
  • 트랜잭션 이메일 발송 (웹 전용): 환영 메시지, 체험 기간 알림, 구독 확인 및 취소 공지 등 계정 관련 알림을 발송합니다.
  • 안정성 및 성능 향상: 충돌 보고서, 오류 데이터 및 서버 측 원격 측정 데이터를 분석하여 버그를 식별하고 수정합니다.
  • 사용 패턴 파악 (웹 전용): 집계된 분석 데이터를 사용하여 웹 앱의 사용 방식을 이해하고 사용자 경험을 개선합니다.
  • 지원 요청 대응: 피드백 제출물을 사용하여 문제를 진단하고 앱을 개선합니다.

3. 귀하의 정보 공유 방법

당사는 귀하의 개인 정보를 판매하지 않습니다. 당사는 다음 상황에서만 정보를 공유합니다:

  • 서비스 제공업체: 당사는 앱을 운영하기 위해 다음의 제3자 서비스 제공업체를 사용합니다:

    • Sentry (Functional Software, Inc.) — 클라이언트 측 충돌 및 오류 보고, EU에서 데이터 처리
    • Microsoft Application Insights (Microsoft Corporation) — 서버 측 원격 측정 및 오류 모니터링
    • Google Analytics (Google LLC) — 웹 사용 분석 (웹 버전 전용)
    • Google APIs (Google LLC) — 인증 및 Google Drive 파일 저장
    • WayForPay — 구독 결제 처리 (웹 버전 전용)
    • Resend (Resend, Inc.) — 트랜잭션 이메일 발송 (웹 버전 전용)
    • Anthropic (Claude) — AI 기반 프로젝트 생성 프롬프트 처리
    • Microsoft Azure — 백엔드 호스팅 및 데이터 저장 (East US 리전)

  • 법적 요건: 당사는 법률에 의해 요구되거나 유효한 법적 절차에 응하여 정보를 공개할 수 있습니다.

4. 데이터 저장 및 보안

  • 서버 측 사용자 데이터 (웹 전용): 귀하의 이메일 주소, 이름, 로캘, 등록 날짜 및 구독 상태는 당사 백엔드 서버의 Azure Table Storage에 저장됩니다.
  • OAuth 토큰 (Google 액세스, 갱신 및 ID 토큰)은 플랫폼 보안 저장소(iOS Keychain, Android Keystore, macOS Keychain 또는 플랫폼 동등 수준)에 저장됩니다.
  • **구독 상태 (웹 전용)**는 오프라인 접근을 위해 보안 저장소에 로컬로 캐시됩니다.
  • API 요청은 무결성 검증을 위해 HMAC-MD5로 서명됩니다.
  • 앱은 사용 가능한 플랫폼 샌드박스(macOS App Sandbox, iOS App Sandbox) 내에서 작동합니다.
  • 당사는 모든 네트워크 통신에 HTTPS를 사용합니다.

4a. 데이터 보유

  • 서버 측 사용자 기록 (웹 전용): 귀하의 사용자 기록 및 구독 이력은 계정이 존재하는 한 보유됩니다. 데이터 삭제를 요청하려면 support@ingantt.com으로 문의하십시오.
  • 프로젝트 파일: 형식 변환을 위해 제출된 파일은 메모리에서 처리되며 보유되지 않습니다.
  • AI 프롬프트: AI 기반 프로젝트 생성을 위해 전송된 텍스트 프롬프트는 당사 서버에 저장되지 않습니다.
  • 피드백: 피드백 메시지는 이메일을 통해 당사 지원팀에 전달되며 데이터베이스에 저장되지 않습니다.
  • 로컬 데이터: 환경 설정, OAuth 토큰 및 캐시된 구독 상태는 귀하가 로그아웃하거나 앱을 삭제할 때까지 기기에 남아 있습니다.

5. 귀하의 선택 및 권리

  • 네이티브 플랫폼에서 로그인은 선택 사항입니다: iOS, Android, macOS, Windows에서는 Google에 로그인하지 않고도 앱을 사용할 수 있습니다. 로그인하면 Google Drive 파일 저장이 활성화됩니다. 웹 버전에서는 앱 기능, 파일 저장 및 구독 관리를 위해 Google 로그인이 필수입니다.
  • Google Drive 접근: 앱은 제한된 drive.file 범위를 사용하며, 앱이 생성한 파일이나 귀하가 앱에서 명시적으로 연 파일에만 접근을 허용합니다 — 귀하의 전체 Google Drive에는 접근하지 않습니다.
  • 피드백 첨부: 피드백 제출 시 프로젝트 파일을 첨부하는 것은 선택 사항이며 귀하의 명시적 동의가 필요합니다.
  • 웹 분석: 브라우저 기반 쿠키 차단 또는 옵트아웃 도구를 사용하여 웹 버전에서의 Google Analytics 데이터 수집을 제한할 수 있습니다.
  • 트랜잭션 이메일: 웹 버전에서 귀하는 계정 관련 이메일(환영, 체험 기간 알림, 구독 알림)을 수신합니다. 이는 구독 관리에 필요한 서비스 통신입니다.
  • 데이터 삭제: 앱 내에서 언제든지 Google 계정에서 로그아웃할 수 있으며, 이 경우 기기에 저장된 OAuth 토큰이 제거됩니다. 당사 서버에 있는 귀하의 사용자 기록 및 관련 데이터의 삭제를 요청하려면 support@ingantt.com으로 문의하십시오.

6. 아동 개인정보 보호

앱은 만 13세 미만(또는 해당 관할 구역의 적용 연령 미만)의 아동을 대상으로 하지 않습니다. 당사는 아동으로부터 의도적으로 개인 정보를 수집하지 않습니다. 아동이 당사에 개인 정보를 제공한 것으로 판단되는 경우 support@ingantt.com으로 문의하여 당사가 적절한 조치를 취할 수 있도록 해 주십시오.

7. 국제 데이터 이전

당사의 백엔드 서버는 East US 리전의 Microsoft Azure에서 호스팅됩니다. 충돌 보고서는 EU의 Sentry에 의해 처리됩니다. 귀하가 미국 이외의 지역에 위치한 경우, 귀하의 정보는 당사의 서비스 제공업체가 운영하는 미국 또는 기타 관할 구역으로 전송 및 처리될 수 있습니다.

8. 본 개인정보처리방침의 변경

당사는 수시로 본 개인정보처리방침을 업데이트할 수 있습니다. 본 방침 상단의 "최종 업데이트" 날짜를 업데이트하여 중대한 변경 사항을 알려 드리겠습니다. 변경 후에도 앱을 계속 사용하는 것은 업데이트된 방침에 대한 귀하의 동의를 구성합니다.

9. 플랫폼별 공개 사항

iOS 및 macOS (Apple)

  • 앱은 Apple App Store를 통해 배포되는 유료 애플리케이션입니다. 구매 거래에는 Apple의 개인정보 관행이 적용됩니다.
  • 앱은 시스템 파일 선택기를 통해 귀하가 명시적으로 부여한 파일 접근 권한만 요청합니다.

Android

  • 앱은 Google Play를 통해 배포되는 유료 애플리케이션입니다. 구매 거래에는 Google의 개인정보 관행이 적용됩니다.
  • 앱은 로컬 파일 작업을 위한 저장소 읽기/쓰기 권한과 네트워크 접근을 위한 인터넷 권한을 요청합니다.

Windows

  • 앱은 Microsoft Store를 통해 배포되는 유료 애플리케이션입니다. 구매 거래에는 Microsoft의 개인정보 관행이 적용됩니다.

Web

  • 웹 버전은 구독 모델을 사용합니다. 구독 결제는 WayForPay를 통해 처리됩니다.
  • 앱은 Google Workspace Marketplace를 통해서도 이용할 수 있습니다.
  • Google Analytics 쿠키는 web.ingantt.com 도메인에 설정됩니다.
  • Google OAuth 인증은 로그인 콜백을 위한 임시 localStorage 항목을 사용하며, 사용 후 즉시 삭제됩니다.
