เปิดไฟล์ .MPP โดยไม่ต้องใช้ Microsoft Project — บนทุกอุปกรณ์

คุณไม่จำเป็นต้องใช้ Microsoft Project เพื่อเปิดไฟล์ .MPP อีกต่อไป Ingantt อ่านรูปแบบไบนารีดั้งเดิม .MPP บน Mac, iPhone, iPad, Windows, Android และในเบราว์เซอร์ และนำทุกงาน งานย่อย ความสัมพันธ์ ทรัพยากร เหตุการณ์สำคัญ และเส้นทางวิกฤตเข้ามา — ในประมาณ 30 วินาที

02ทำไม Ingantt จึงเปิดไฟล์ .MPP ที่เครื่องมืออื่นเปิดไม่ได้

อ่านไบนารี .MPP ของจริง

แยกวิเคราะห์รูปแบบ .MPP ดั้งเดิมของ Microsoft Project โดยตรง — ไม่ใช่สรุปรายการงาน ไม่ใช่ภาพหน้าจอ โครงการทั้งหมดเข้ามา

ทำงานที่ MS Project ไม่ทำงาน

แอปเนทีฟบน Mac, iPhone, iPad และ Android พร้อมเวอร์ชันเบราว์เซอร์ — บนทุกแพลตฟอร์มที่ Microsoft Project ไม่ทำงาน

ไม่มีอะไรสูญหาย

งาน งานย่อย ความสัมพันธ์แบบ finish-to-start ทรัพยากร เหตุการณ์สำคัญ และเส้นทางวิกฤตนำเข้าด้วยโครงสร้างเดิม

บันทึกกลับเป็น .MPP

หลังจากเปิด บันทึกไฟล์เป็น .MPP หรือ MS Project XML เพื่อให้เพื่อนร่วมงานที่ใช้ Microsoft Project ยังสามารถเปิดได้ตามปกติ

03ดู Ingantt ในการใช้งานจริง

04การเปิดไฟล์ .MPP — คำถามเมื่อคุณติดขัด

ฉันจะเปิดไฟล์ .MPP โดยไม่ใช้ Microsoft Project ได้อย่างไร?

ใช้ Ingantt มันอ่านรูปแบบไบนารี .MPP โดยตรง — ไม่ต้องมีใบอนุญาต Microsoft Project ไม่ต้องมีเครื่องเสมือน Windows ไม่ต้องมีบริการแปลงที่ส่ง PDF ทางอีเมล Ingantt นำเข้าทุกงาน งานย่อย ความสัมพันธ์ การจัดสรรทรัพยากร เหตุการณ์สำคัญ และเส้นทางวิกฤต ทดลองฟรี 7 วันครอบคลุมสถานการณ์ครั้งเดียวส่วนใหญ่

ฉันจะเปิดไฟล์ .MPP บน Mac ได้อย่างไร?

Microsoft Project ไม่ทำงานบน macOS — ไม่มีเวอร์ชัน Mac ดังนั้นคำตอบจึงไม่ใช่ "ติดตั้ง Project บน Mac" แต่ "ใช้สิ่งที่อ่าน .MPP ได้อย่างเป็นธรรมชาติบน Mac" Ingantt มีแอป macOS เนทีฟใน Mac App Store ที่เปิดไฟล์ .MPP โดยตรงบน Mac Apple Silicon และ Intel ดาวน์โหลด เปิด .MPP แล้วคุณจะเห็นแกนต์ชาร์ตเต็มรูปแบบพร้อมงาน ความสัมพันธ์ และเส้นทางวิกฤต

ฉันสามารถบันทึกไฟล์ .MPP หลังจากเปิดได้หรือไม่?

ได้ Ingantt บันทึกโครงการกลับเป็นรูปแบบ .MPP ดั้งเดิมของ Microsoft Project พร้อมด้วย .XML, PDF และ PNG คุณสามารถเปิด .MPP บน Mac, iPhone หรือในเบราว์เซอร์ แก้ไข บันทึกกลับเป็น .MPP และส่งไฟล์ให้เพื่อนร่วมงานที่ยังใช้ Microsoft Project หากคุณเพียงต้องการดูแผนภูมิโดยไม่แก้ไข หน้า โปรแกรมดูแกนต์ ของเราครอบคลุมกรณีอ่านอย่างเดียว
