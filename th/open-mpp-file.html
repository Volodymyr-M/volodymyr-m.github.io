เปิดไฟล์ .MPP และดู พิมพ์ หรือส่งออก Gantt Chart — ไม่ต้องใช้ Microsoft Project

ลากวางไฟล์ .MPP หรือไฟล์ Microsoft Project XML ลงใน Ingantt ในเบราว์เซอร์ของคุณ แล้วมันจะกลายเป็น Gantt chart แบบโต้ตอบที่คุณซูม เลื่อน กรอง พิมพ์ และส่งออกเป็น PDF, PNG หรือ MS Project XML ได้ — ไม่ต้องติดตั้งอะไรเลย ทุกงาน งานย่อย ความสัมพันธ์ ทรัพยากร เหตุการณ์สำคัญ และเส้นทางวิกฤตจะถ่ายทอดเข้ามาครบถ้วนในเวลาประมาณ 30 วินาที

02สิ่งที่โปรแกรมเปิด .MPP และดู Gantt ของจริงควรทำได้

อ่านไบนารี .MPP ของจริง

แยกวิเคราะห์รูปแบบ .MPP ดั้งเดิมของ Microsoft Project และ MS Project XML โดยตรง — ไม่ใช่สรุปรายการงาน ไม่ใช่ภาพหน้าจอ งาน งานย่อย ความสัมพันธ์ เหตุการณ์สำคัญ และทรัพยากรทั้งหมดเข้ามาครบถ้วน

โต้ตอบได้ ไม่ใช่ภาพ

ซูมไทม์ไลน์ เลื่อนดูเฟส ขยายหรือย่อกลุ่ม วางเมาส์บนงานใดก็ได้เพื่อดูรายละเอียด นี่คือ Gantt chart ของจริง — ไม่ใช่ PDF แบนที่ได้แต่เลื่อนดู

พิมพ์และส่งออกได้ทุกที่

พิมพ์ Gantt chart ตรงจากเบราว์เซอร์ หรือส่งออกเป็น PDF และ PNG ที่ระดับซูมใด ๆ — พร้อมแนบอีเมล วางในสไลด์ หรือติดบนผนัง

บันทึกกลับเป็น MS Project XML

หลังจากดูหรือแก้ไข ส่งออกไฟล์เป็น MS Project XML เพื่อให้เพื่อนร่วมงานที่ยังใช้ Microsoft Project เปิดได้ที่ปลายทาง — ไม่ผูกมัดกับรูปแบบใด

03ดู Ingantt ในการใช้งานจริง

04การเปิดและดูไฟล์ .MPP — คำถามที่ผู้คนถามกันจริง ๆ

ฉันจะเปิดไฟล์ .MPP โดยไม่ใช้ Microsoft Project ได้อย่างไร?

ใช้ Ingantt ในเบราว์เซอร์ของคุณ มันอ่านรูปแบบไบนารี .MPP โดยตรง — ไม่ต้องมีใบอนุญาต Microsoft Project เครื่องเสมือน Windows หรือบริการแปลงที่ส่ง PDF ทางอีเมล Ingantt นำเข้าทุกงาน งานย่อย ความสัมพันธ์ การมอบหมายทรัพยากร เหตุการณ์สำคัญ และเส้นทางวิกฤต และแสดงโครงการเป็น Gantt chart แบบโต้ตอบเต็มรูปแบบ การทดลองใช้ฟรี 7 วันเพียงพอสำหรับสถานการณ์การเปิดไฟล์ครั้งเดียวส่วนใหญ่ และไม่ต้องใช้บัตรเครดิตในการเริ่มต้น

โปรแกรมดู Gantt คืออะไร และ Ingantt เป็นหรือไม่?

โปรแกรมดู Gantt คือเครื่องมือที่เปิดไฟล์ Gantt chart — โดยทั่วไปเป็นไฟล์ .MPP หรือ .XML ของ Microsoft Project — และแสดงโครงการเป็นไทม์ไลน์แบบโต้ตอบ โปรแกรมดูของจริงให้คุณเลื่อนข้ามตาราง ซูมเข้าออก ขยายหรือย่อเฟส และวางเมาส์บนงานเพื่อดูความสัมพันธ์ วันที่เริ่มและสิ้นสุด และทรัพยากร Ingantt เป็นโปรแกรมดูและแก้ไข Gantt บนเบราว์เซอร์ ดังนั้นคุณสามารถเปิดไฟล์เดียวกันบนอุปกรณ์ใดก็ได้ที่มี — และส่งออกการเปลี่ยนแปลงกลับเป็น MS Project XML เมื่อจำเป็น

ฉันสามารถเปิดไฟล์ .MPP หรือดู Gantt chart ในเบราว์เซอร์ได้หรือไม่?

ได้ Ingantt ทำงานในเบราว์เซอร์ทั้งหมด — Chrome, Safari, Firefox หรือ Edge — บน Mac, Windows, Linux, Chromebook, iPhone, iPad หรือ Android ลงชื่อเข้าใช้ด้วยบัญชี Google อัปโหลดไฟล์ .MPP หรือเลือกจาก Google Drive แล้วโครงการทั้งหมดจะเปิดในเบราว์เซอร์พร้อมซูม เลื่อน และวางเมาส์ดูรายละเอียด ไม่มีการติดตั้งในเครื่องและไม่มีการดาวน์โหลด นี่คือเส้นทางที่ใช่หากคุณอยู่บนเครื่องทำงานที่ติดตั้งซอฟต์แวร์ใหม่ไม่ได้ หรือบน Linux หรือ Chromebook ที่ Microsoft Project ไม่เคยเป็นตัวเลือก

ฉันสามารถพิมพ์ Gantt chart หรือส่งออกเป็น PDF หรือ PNG ได้หรือไม่?

ได้ทั้งสามอย่าง Ingantt พิมพ์ Gantt chart ตรงจากเบราว์เซอร์ และส่งออกเป็น PDF และ PNG ที่ระดับซูมใดก็ได้ — ดังนั้นไม่ว่าคุณต้องการภาพรวมหน้าเดียวหรือไทม์ไลน์ขนาดผนังหลายหน้า เลย์เอาต์ยังคงสะอาดตา PDF เป็นตัวเลือกปกติสำหรับการแบ่งปันแบบอ่านอย่างเดียวกับผู้มีส่วนได้เสียที่ไม่ต้องแตะไฟล์ต้นฉบับ PNG เหมาะสำหรับวางในสไลด์ เอกสาร Notion หรือข้อความ Slack การพิมพ์เนทีฟเร็วที่สุดเมื่อคุณแค่ต้องการสำเนากระดาษ

ฉันสามารถบันทึกไฟล์ .MPP กลับได้หรือไม่หลังจากเปิดใน Ingantt?

ไม่ใช่ในรูป .MPP — Ingantt ส่งออกเป็น MS Project XML ซึ่ง Microsoft Project เปิดได้โดยตรง รูปแบบไบนารี .MPP เป็นของเฉพาะ ไม่มีเอกสารประกอบ และโดยพื้นฐานแล้วเป็นเป้าหมายการเขียนของ Microsoft เท่านั้น ส่วน MS Project XML เป็นรูปแบบแลกเปลี่ยนของ Microsoft เองและแลกเปลี่ยนสองทางได้สะอาดตา ดังนั้นคุณสามารถเปิด .MPP ใน Ingantt แก้ไข ส่งออกเป็น MS Project XML และส่งไฟล์ .xml ให้เพื่อนร่วมงานที่ยังใช้ Microsoft Project — พวกเขาจะเปิดได้เหมือนที่เปิดไฟล์โครงการใด ๆ สำหรับการแบ่งปันแบบอ่านอย่างเดียวกับคนที่ไม่ใช้ Project เลย PDF หรือ PNG คือรูปแบบส่งออกที่ใช่

ฉันจะสูญเสียความสัมพันธ์ ทรัพยากร หรือเส้นทางวิกฤตเมื่อเปิด .MPP นอก Microsoft Project หรือไม่?

ใน Ingantt ไม่ — การนำเข้าไม่มีการสูญเสียสำหรับฟิลด์ที่สำคัญ งาน งานย่อย ความสัมพันธ์แบบ finish-to-start การมอบหมายทรัพยากร งานและระยะเวลา เหตุการณ์สำคัญ และการคำนวณเส้นทางวิกฤตทั้งหมดถ่ายทอดมาจากไฟล์ .MPP ต้นฉบับ Ingantt ใช้โครงสร้างโครงการแบบเดียวกับที่ Microsoft Project ใช้ภายใน ดังนั้น Gantt chart ที่ใช้เวลาทั้งบ่ายสร้างใน Project จะยังดูเหมือน Gantt chart เดิมหลังจากคุณเปิดใน Ingantt โปรแกรมดูฟรีเมียมที่แสดงแต่รายการงานหรือทำให้โครงการแบนเป็นภาพจะสูญเสียกราฟความสัมพันธ์ ซึ่งมักเป็นส่วนที่สำคัญที่สุดของแผน

มีโปรแกรมเปิด .MPP หรือโปรแกรมดู Gantt chart ฟรีหรือไม่?

Ingantt มีทดลองใช้ฟรี 7 วันโดยไม่ต้องใช้บัตรเครดิตในการเริ่มต้น และการทดลองรวมถึงการนำเข้า .MPP เต็มรูปแบบ การดู การแก้ไข การพิมพ์ และการส่งออก PDF/PNG/XML — ไม่มีอะไรถูกล็อกไว้หลัง "รุ่นโปรแกรมดู" กับ "รุ่นโปรแกรมแก้ไข" สำหรับคนส่วนใหญ่ที่ต้องการเปิดหรือดูไฟล์ .MPP เพียงครั้งเดียวหรือไม่กี่ครั้ง การทดลองใช้ฟรีครอบคลุมสถานการณ์ทั้งหมด หลังทดลอง แพ็กเกจแบบเสียเงินเริ่มต้นที่ $5.99 ต่อผู้ใช้ต่อเดือน หากคุณทำงานกับผู้รับเหมาหรือลูกค้าที่ยังใช้ Microsoft Project แพ็กเกจเสียเงินจะน้อยกว่าค่าใช้จ่าย Microsoft 365 Project เพียงหนึ่งเดือน

ฉันต้องรู้การจัดการโครงการเพื่อใช้โปรแกรมดู Gantt หรือไม่?

ไม่ต้อง Ingantt ถูกออกแบบเพื่อให้ผู้มีส่วนได้เสียที่ไม่ใช่ PM — ผู้บริหารที่ทบทวนวันที่ส่งมอบ ลูกค้าที่ตรวจสอบเหตุการณ์สำคัญ สมาชิกทีมใหม่ที่กำลังทำความเข้าใจ — สามารถเปิด Gantt chart และอ่านได้โดยไม่ต้องเรียนรู้อะไรใหม่ คุณเลื่อนไทม์ไลน์เพื่อดูว่าสิ่งต่าง ๆ เกิดขึ้นเมื่อไร วางเมาส์บนงานเพื่อดูว่าใครรับผิดชอบ และขยายเฟสเพื่อดูงานข้างใน ไม่มีศัพท์เฉพาะให้ถอดรหัสและไม่มีขั้นตอนการตั้งค่านอกเหนือจากการเปิดไฟล์
