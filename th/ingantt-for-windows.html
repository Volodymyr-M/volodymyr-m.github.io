คำถามที่พบบ่อย

Ingantt for Windows คืออะไร

Ingantt for Windows เป็นเครื่องมือจัดการโครงการระดับมืออาชีพที่ทำงานแบบเนทีฟบน PC ของคุณ ช่วยให้คุณสร้างและจัดการแผนโครงการด้วยการวางแผนที่มี AI ช่วยและรองรับ MS Project

ฉันสามารถนำเข้าไฟล์ Microsoft Project ที่มีอยู่แล้วได้หรือไม่

ได้! Ingantt รองรับการนำเข้าไฟล์ .MPP และ .XML จาก Microsoft Project อย่างสมบูรณ์ ข้อมูลโครงการของคุณ ทั้งงาน ทรัพยากร และความสัมพันธ์ระหว่างงาน จะถูกเก็บรักษาไว้ในกระบวนการนำเข้า

การสร้างโครงการด้วย AI ทำงานอย่างไร

ผู้ช่วย AI ของเราสามารถสร้างแผนโครงการที่สมบูรณ์ได้จากข้อความสั้น ๆ เพียงคุณระบุเป้าหมายและความต้องการของโครงการ AI ก็จะสร้างแผนงานที่มีการแบ่งงาน กำหนดเวลา และความสัมพันธ์ตามหลักปฏิบัติที่ดีที่สุด

มีรุ่นทดลองใช้ฟรีสำหรับเวอร์ชัน Windows หรือไม่

มี หลังจากติดตั้งแอปจาก Microsoft Store แล้ว คุณสามารถทดลองใช้ฟรีได้ 7 วัน

ฉันจะซื้อ Ingantt สำหรับ Windows ได้อย่างไร
