  • สร้างโปรเจกต์ด้วย AI

  • จัดตารางได้ง่าย

    อัปเดตส่วนใด ๆ ของแผน แล้วเห็นผลโดยรวมทันที

  • ความคุ้นเคย

    หากคุ้นเคยกับ MS Project หรือซอฟต์แวร์คล้ายกัน ก็ใช้งานได้เลย!

การติดตามขั้นสูง

  • ติดตามความคืบหน้า

  • การใช้ทรัพยากร

    ตรวจสอบการจัดสรรทรัพยากรและระบุจุดติดขัดที่อาจเกิดขึ้น

  • วิเคราะห์ไทม์ไลน์

    วิเคราะห์ Critical Path และติดตามการเปลี่ยนแปลงของตารางเวลา

การเชื่อมต่อที่ทรงพลัง

  • ความเข้ากันได้กับ MS Project

    นำเข้าไฟล์ MPP จาก MS Project และบันทึกในรูปแบบ XML ที่ยังคงเข้ากันได้เสมอ

  • ตัวเลือกการส่งออก

    ส่งออกเป็น PDF และ PNG ได้ทุกเมื่อ

  • จัดเก็บและเปิดไฟล์ใน Google Drive

    ใช้ Google Drive เพื่อจัดเก็บไฟล์โปรเจกต์ของคุณ

Ingantt ทำอะไรได้บ้าง

การวางแผนขั้นสูง

  • สร้างแผนโปรเจกต์จากข้อความสั้น ๆ ด้วย AI
  • เปลี่ยนแปลงอะไรก็ได้แล้วเห็นผลทันที
  • คาดการณ์วันที่สิ้นสุดและระบุอุปสรรคได้ตั้งแต่เนิ่น ๆ
  • วางแผนต้นทุน

การจัดการงาน

  • แบ่งโครงการออกเป็นงานย่อย
  • ตั้งค่าความเชื่อมโยงและกำหนดเส้นตาย
  • ติ๊กงานเป็นหมุดหมาย (milestone)
  • ติดตาม Critical Path
  • ตรวจสอบสถานะความสำเร็จของงาน

การทำงานร่วมกัน

  • ทำงานร่วมกับ Google Drive
  • แลกเปลี่ยนไฟล์ในรูปแบบ XML, PDF หรือ PNG

การติดตามขั้นสูง

  • ตรวจสอบการจัดสรรทรัพยากร
  • วิเคราะห์ Critical Path
  • ติดตามการใช้งานทรัพยากร
  • วิเคราะห์ไทม์ไลน์

การบริหารทรัพยากร

  • จัดการทรัพยากรแรงงาน วัสดุ และต้นทุน
  • กำหนดตารางและเวลาทำงานของทรัพยากร
  • ปรับสมดุลอัตโนมัติหากมีทรัพยากรที่ถูกใช้งานมากเกินไป
  • ทบทวนการใช้ทรัพยากรในช่วงเวลา

การเชื่อมต่อ

  • เข้ากันได้กับไฟล์ MS Project (MPP และ XML)
  • ส่งออกเป็น PDF และ PNG
  • ทำงานร่วมกับ Google Drive

Ingantt for macOS คืออะไร

Ingantt for macOS คือเครื่องมือการบริหารโครงการระดับมืออาชีพที่ทำงานแบบเนทีฟบน Mac มาพร้อมความสามารถในการวางแผนด้วย AI และเข้ากันได้กับ MS Project ช่วยให้คุณสร้างและจัดการแผนโปรเจกต์ได้ง่ายดาย

ฉันสามารถนำเข้าไฟล์ Microsoft Project ที่มีอยู่แล้วได้ไหม

ได้แน่นอน! Ingantt รองรับการนำเข้าไฟล์ .MPP และ .XML จาก Microsoft Project ข้อมูลโครงการของคุณ ทั้งงาน ทรัพยากร และความเชื่อมโยง จะยังคงอยู่ครบถ้วน

AI ช่วยสร้างโปรเจกต์อย่างไร

ผู้ช่วย AI ของเราสามารถสร้างแผนโปรเจกต์ครบวงจรจากคำอธิบายข้อความสั้น ๆ เพียงอธิบายเป้าหมายและข้อกำหนดของคุณ AI จะสร้างแผนที่มีงาน ไทม์ไลน์ และความเชื่อมโยง โดยอิงจากแนวทางปฏิบัติที่ดีที่สุด

มีรุ่นทดลองใช้ฟรีสำหรับเวอร์ชัน macOS ไหม

ไม่มี รุ่น macOS เป็นการซื้อขาดครั้งเดียว ไม่มีค่าสมาชิกหรือรุ่นทดลองฟรี

ฉันจะซื้อ Ingantt for macOS ได้อย่างไร

ไปที่ Mac App Store เพื่อทำการซื้อ คุณจะครอบครองแอปตลอดไปโดยไม่ต้องเสียค่าบริการรายเดือน
