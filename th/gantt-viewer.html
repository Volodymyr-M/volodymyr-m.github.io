02สิ่งที่โปรแกรมดูแกนต์ที่แท้จริงควรทำ

เปิดไฟล์ .MPP ของจริง

อ่านไฟล์ Microsoft Project .MPP และ .XML อย่างเป็นธรรมชาติ — งาน งานย่อย ความสัมพันธ์ เหตุการณ์สำคัญ และทรัพยากรยังคงอยู่ครบ

โต้ตอบได้ ไม่ใช่รูปภาพ

ซูมเส้นเวลา เลื่อนผ่านเฟส ขยายกลุ่ม วางเมาส์เหนืองาน แกนต์ชาร์ตของจริง ไม่ใช่ภาพนิ่ง

บนทุกอุปกรณ์

แอปเนทีฟสำหรับ Mac, iOS, iPad, Windows และ Android พร้อมเวอร์ชันเบราว์เซอร์ — เปิดไฟล์เดียวกันได้ทุกที่

ไม่ต้องมีความรู้การจัดการโครงการ

หากคุณเพียงต้องการเห็นสิ่งที่แกนต์แสดง — วันที่ ความสัมพันธ์ ใครทำอะไร — คุณทำได้โดยไม่ต้องเรียนรู้สิ่งใหม่

04โปรแกรมดูแกนต์ — คำถามที่พบบ่อย

ฉันจะดูแกนต์ชาร์ตโดยไม่ใช้ Microsoft Project ได้อย่างไร?

เปิดไฟล์ใน Ingantt มันนำเข้ารูปแบบ .MPP ดั้งเดิมของ Microsoft Project และ MS Project XML โดยตรง ดังนั้นไฟล์จากเพื่อนร่วมงานที่ใช้ MS Project จะเปิดเป็นแกนต์ชาร์ตแบบโต้ตอบเต็มรูปแบบ — โดยไม่ต้องมีใบอนุญาต MS Project หรือการติดตั้ง Windows งาน ความสัมพันธ์ และเส้นทางวิกฤตยังคงอยู่ครบ

มีโปรแกรมดูแกนต์ฟรีหรือไม่?

Ingantt มีทดลองใช้ฟรี 7 วันโดยไม่ต้องใช้บัตรเครดิต การทดลองรวมการดูและแก้ไขไฟล์ .MPP และ MS Project XML อย่างเต็มรูปแบบบนทุกแพลตฟอร์ม หลังการทดลอง แผนแบบชำระเงินเริ่มต้นที่ $5.99 ต่อผู้ใช้ต่อเดือน

ฉันสามารถพิมพ์หรือแชร์แกนต์ชาร์ตที่ฉันกำลังดูได้หรือไม่?

ได้ Ingantt ส่งออกแกนต์ชาร์ตเป็น PDF หรือ PNG เพื่อพิมพ์ แนบไปกับอีเมล หรือวางลงในสไลด์ การส่งออก PDF รักษาเค้าโครงเส้นเวลาที่ระดับซูมใดก็ได้ หากคุณต้องการเปิด .MPP เดียวกันในที่อื่นในภายหลัง ดูคู่มือของเราเกี่ยวกับ การเปิดไฟล์ .MPP โดยไม่ใช้ Microsoft Project
