Ingantt Development ("เรา" หรือ "ของเรา") ดำเนินงานแอปพลิเคชัน Ingantt ("แอป") และเว็บไซต์ที่ https://ingantt.com ("เว็บไซต์") นโยบายความเป็นส่วนตัวนี้อธิบายวิธีที่เราเก็บรวบรวม ใช้ และแชร์ข้อมูลเมื่อคุณใช้งานแอปบนทุกแพลตฟอร์มที่รองรับ (iOS, Android, macOS, Windows และ Web รวมถึงผ่าน Google Workspace Marketplace)

1. ข้อมูลที่เราเก็บรวบรวม

1.1 ข้อมูลที่คุณให้

  • ข้อมูล Google Account: หากคุณลงชื่อเข้าใช้ด้วย Google เราจะได้รับที่อยู่อีเมล ชื่อที่แสดง และ URL รูปโปรไฟล์ของคุณจาก Google บนแพลตฟอร์มเนทีฟ (iOS, Android, macOS, Windows) การลงชื่อเข้าใช้เป็นทางเลือกและเปิดใช้งานการจัดเก็บไฟล์บน Google Drive บนเวอร์ชัน Web จำเป็นต้องใช้ Google Sign-In และใช้สำหรับการจัดเก็บไฟล์บน Google Drive และการจัดการสมัครสมาชิก เมื่อคุณใช้งานเวอร์ชัน Web เป็นครั้งแรก ระบบจะสร้างบันทึกผู้ใช้บนเซิร์ฟเวอร์ของเราที่มีที่อยู่อีเมล ชื่อ ภาษา และวันที่ลงทะเบียนของคุณ

  • ข้อมูลการสมัครสมาชิกและการเรียกเก็บเงิน (Web เท่านั้น): เมื่อซื้อสมัครสมาชิกผ่าน Web เราจะเก็บรวบรวมชื่อ ที่อยู่อีเมล ที่อยู่ทางไปรษณีย์ หมายเลขโทรศัพท์ และรายละเอียดที่เกี่ยวข้องกับการชำระเงินของคุณ ข้อมูลนี้จะถูกส่งไปยัง WayForPay ผู้ให้บริการประมวลผลการชำระเงินของเรา เพื่อประมวลผลธุรกรรม

  • ข้อเสนอแนะ: เมื่อคุณส่งข้อเสนอแนะผ่านแอป เราจะเก็บรวบรวมข้อความของข้อความคุณ คุณอาจเลือกแนบไฟล์โครงการที่เปิดอยู่ในปัจจุบันเพื่อช่วยเราวินิจฉัยปัญหา

  • ข้อมูลนำเข้าฟีเจอร์ AI: เมื่อคุณใช้ฟีเจอร์ "Create with AI" ข้อความแจ้งที่คุณป้อนจะถูกส่งไปยังเซิร์ฟเวอร์แบ็กเอนด์ของเราเพื่อประมวลผลโดย Anthropic (Claude)

  • ไฟล์โครงการ: ไฟล์โครงการที่คุณสร้างและแก้ไขยังคงอยู่ภายใต้การควบคุมของคุณ ไฟล์จะถูกจัดเก็บในเครื่องบนอุปกรณ์ของคุณหรือในบัญชี Google Drive ของคุณ (ในโฟลเดอร์ชื่อ "Ingantt Schedules") เมื่อคุณเปิดไฟล์ในรูปแบบที่ไม่ใช่เนทีฟ (เช่น MPP, GAN, XER หรือรูปแบบไฟล์โครงการที่รองรับอื่น ๆ) ไฟล์จะถูกส่งไปยังเซิร์ฟเวอร์ของเราเพื่อแปลงรูปแบบ ไฟล์จะถูกประมวลผลในหน่วยความจำเท่านั้นและไม่ถูกจัดเก็บบนเซิร์ฟเวอร์ของเรา

1.2 ข้อมูลที่เก็บรวบรวมโดยอัตโนมัติ

  • ข้อมูลเมตาของแอป: คำขอแต่ละรายการไปยังแบ็กเอนด์ของเราประกอบด้วยเวอร์ชันแอป ชื่อแพลตฟอร์ม (iOS, Android, macOS, Windows หรือ Web) ภาษาของระบบ และหากลงชื่อเข้าใช้แล้ว ที่อยู่อีเมลของคุณ

  • รายงานข้อขัดข้องและข้อผิดพลาด: เราใช้ Sentry (โฮสต์ในสหภาพยุโรป) เพื่อเก็บรวบรวมรายงานข้อขัดข้องและบันทึกข้อผิดพลาดในการผลิต ซึ่งรวมถึง stack traces ข้อความข้อผิดพลาด เวอร์ชันแอป และข้อมูลแพลตฟอร์ม เราไม่ได้ตั้งใจเก็บรวบรวมข้อมูลส่วนบุคคลผ่านรายงานข้อขัดข้อง

  • การวิเคราะห์เว็บ (Web เท่านั้น): แอปเวอร์ชัน Web ใช้ Google Analytics 4 เพื่อเก็บรวบรวมข้อมูลวิเคราะห์การใช้งานมาตรฐาน รวมถึงการดูหน้าเว็บและข้อมูลเซสชัน Google Analytics อาจตั้งค่าคุกกี้บนเบราว์เซอร์ของคุณ แอปเวอร์ชันมือถือและเดสก์ท็อปไม่มีการติดตามการวิเคราะห์

  • การตั้งค่าผู้ใช้: แอปจัดเก็บการตั้งค่าภายในเครื่องของคุณ (เช่น ธีม ตัวเลือกการแสดงผล ความกว้างคอลัมน์ และรายการไฟล์ล่าสุด) บนอุปกรณ์ของคุณโดยใช้กลไกจัดเก็บข้อมูลมาตรฐานของแพลตฟอร์ม การตั้งค่าเหล่านี้ไม่ถูกส่งไปยังเซิร์ฟเวอร์ของเรา

1.3 ข้อมูลที่เราไม่เก็บรวบรวม

  • เราไม่ใช้ข้อมูลของคุณเพื่อฝึกฝนโมเดลปัญญาประดิษฐ์หรือการเรียนรู้ของเครื่อง
  • เราไม่แสดงโฆษณาและไม่รวม SDK โฆษณาใด ๆ
  • เราไม่เก็บรวบรวมข้อมูลตำแหน่ง รายชื่อผู้ติดต่อ รูปถ่าย การป้อนข้อมูลกล้อง หรือการป้อนข้อมูลไมโครโฟน
  • เราไม่ใช้ตัวระบุอุปกรณ์เพื่อวัตถุประสงค์ในการติดตาม
  • การสำรองข้อมูลบนคลาวด์ของ Android ถูกปิดใช้งานอย่างชัดเจนสำหรับแอป

2. วิธีที่เราใช้ข้อมูลของคุณ

เราใช้ข้อมูลที่เก็บรวบรวมเพื่อวัตถุประสงค์ดังต่อไปนี้:

  • ให้บริการและดำเนินงานแอป: ประมวลผลคำขอของคุณ เปิดใช้งานการจัดเก็บและการซิงโครไนซ์ไฟล์ผ่าน Google Drive แปลงรูปแบบไฟล์ และให้บริการการสร้างโครงการด้วย AI
  • ประมวลผลการสมัครสมาชิก (Web เท่านั้น): ประมวลผลการชำระเงินการสมัครสมาชิก Web ตรวจสอบสถานะการสมัครสมาชิก และจัดการบัญชีของคุณ
  • ส่งอีเมลธุรกรรม (Web เท่านั้น): ส่งการแจ้งเตือนที่เกี่ยวข้องกับบัญชี เช่น ข้อความต้อนรับ การเตือนช่วงทดลองใช้ การยืนยันการสมัครสมาชิก และการแจ้งเตือนการยกเลิก
  • ปรับปรุงความเสถียรและประสิทธิภาพ: วิเคราะห์รายงานข้อขัดข้อง ข้อมูลข้อผิดพลาด และการวัดระยะไกลฝั่งเซิร์ฟเวอร์เพื่อระบุและแก้ไขข้อบกพร่อง
  • เข้าใจรูปแบบการใช้งาน (Web เท่านั้น): ใช้ข้อมูลวิเคราะห์แบบรวมเพื่อเข้าใจวิธีการใช้งานแอป Web และปรับปรุงประสบการณ์ผู้ใช้
  • ตอบสนองคำขอการสนับสนุน: ใช้ข้อเสนอแนะที่ส่งมาเพื่อวินิจฉัยปัญหาและปรับปรุงแอป

3. วิธีที่เราแชร์ข้อมูลของคุณ

เราไม่ขายข้อมูลส่วนบุคคลของคุณ เราแชร์ข้อมูลเฉพาะในสถานการณ์ดังต่อไปนี้:

  • ผู้ให้บริการ: เราใช้ผู้ให้บริการบุคคลที่สามดังต่อไปนี้เพื่อดำเนินงานแอป:

    • Sentry (Functional Software, Inc.) -- การรายงานข้อขัดข้องและข้อผิดพลาดฝั่งไคลเอนต์ ข้อมูลประมวลผลในสหภาพยุโรป
    • Microsoft Application Insights (Microsoft Corporation) -- การวัดระยะไกลและการตรวจสอบข้อผิดพลาดฝั่งเซิร์ฟเวอร์
    • Google Analytics (Google LLC) -- การวิเคราะห์การใช้งานเว็บ (เวอร์ชัน Web เท่านั้น)
    • Google APIs (Google LLC) -- การยืนยันตัวตนและการจัดเก็บไฟล์บน Google Drive
    • WayForPay -- การประมวลผลการชำระเงินการสมัครสมาชิก (เวอร์ชัน Web เท่านั้น)
    • Resend (Resend, Inc.) -- การส่งอีเมลธุรกรรม (เวอร์ชัน Web เท่านั้น)
    • Anthropic (Claude) -- การประมวลผลข้อความแจ้งการสร้างโครงการด้วย AI
    • Microsoft Azure -- โฮสติ้งแบ็กเอนด์และการจัดเก็บข้อมูล (East US region)

  • ข้อกำหนดทางกฎหมาย: เราอาจเปิดเผยข้อมูลหากกฎหมายกำหนดหรือเพื่อตอบสนองต่อกระบวนการทางกฎหมายที่ถูกต้อง

4. การจัดเก็บข้อมูลและความปลอดภัย

  • ข้อมูลผู้ใช้ฝั่งเซิร์ฟเวอร์ (Web เท่านั้น): ที่อยู่อีเมล ชื่อ ภาษา วันที่ลงทะเบียน และสถานะการสมัครสมาชิกของคุณถูกจัดเก็บใน Azure Table Storage บนเซิร์ฟเวอร์แบ็กเอนด์ของเรา
  • OAuth tokens (Google access, refresh และ ID tokens) ถูกจัดเก็บในพื้นที่จัดเก็บที่ปลอดภัยของแพลตฟอร์ม (iOS Keychain, Android Keystore, macOS Keychain หรือเทียบเท่าบนแพลตฟอร์ม)
  • สถานะการสมัครสมาชิก (Web เท่านั้น) ถูกแคชในเครื่องในพื้นที่จัดเก็บที่ปลอดภัยเพื่อการเข้าถึงแบบออฟไลน์
  • คำขอ API ไปยังแบ็กเอนด์ของเราถูกลงนามด้วย HMAC-MD5 เพื่อตรวจสอบความสมบูรณ์
  • แอปทำงานภายใน sandbox ของแพลตฟอร์มเมื่อพร้อมใช้งาน (macOS App Sandbox, iOS App Sandbox)
  • เราใช้ HTTPS สำหรับการสื่อสารเครือข่ายทั้งหมด

4a. การเก็บรักษาข้อมูล

  • บันทึกผู้ใช้ฝั่งเซิร์ฟเวอร์ (Web เท่านั้น): บันทึกผู้ใช้และประวัติการสมัครสมาชิกของคุณถูกเก็บรักษาไว้ตราบเท่าที่บัญชีของคุณยังคงอยู่ หากต้องการขอลบข้อมูลของคุณ ติดต่อเราที่ support@ingantt.com
  • ไฟล์โครงการ: ไฟล์ที่ส่งมาเพื่อแปลงรูปแบบจะถูกประมวลผลในหน่วยความจำและไม่ถูกเก็บรักษาไว้
  • ข้อความแจ้ง AI: ข้อความแจ้งที่ส่งสำหรับการสร้างโครงการด้วย AI ไม่ถูกจัดเก็บบนเซิร์ฟเวอร์ของเรา
  • ข้อเสนอแนะ: ข้อความข้อเสนอแนะถูกส่งต่อไปยังทีมสนับสนุนของเราทางอีเมลและไม่ถูกจัดเก็บในฐานข้อมูล
  • ข้อมูลในเครื่อง: การตั้งค่า, OAuth tokens และสถานะการสมัครสมาชิกที่แคชไว้ยังคงอยู่บนอุปกรณ์ของคุณจนกว่าคุณจะออกจากระบบหรือถอนการติดตั้งแอป

5. ทางเลือกและสิทธิ์ของคุณ

  • การลงชื่อเข้าใช้เป็นทางเลือกบนแพลตฟอร์มเนทีฟ: บน iOS, Android, macOS และ Windows คุณสามารถใช้งานแอปโดยไม่ต้องลงชื่อเข้าใช้ Google การลงชื่อเข้าใช้เปิดใช้งานการจัดเก็บไฟล์บน Google Drive บนเวอร์ชัน Web จำเป็นต้องใช้ Google Sign-In สำหรับฟังก์ชันการทำงานของแอป การจัดเก็บไฟล์ และการจัดการสมัครสมาชิก
  • การเข้าถึง Google Drive: แอปใช้ขอบเขต drive.file แบบจำกัด ซึ่งอนุญาตให้เข้าถึงได้เฉพาะไฟล์ที่แอปสร้างขึ้นหรือที่คุณเปิดอย่างชัดเจนในแอปเท่านั้น -- ไม่ใช่ Google Drive ทั้งหมดของคุณ
  • ไฟล์แนบข้อเสนอแนะ: การแนบไฟล์โครงการของคุณเมื่อส่งข้อเสนอแนะเป็นทางเลือกและต้องได้รับความยินยอมอย่างชัดเจนจากคุณ
  • การวิเคราะห์เว็บ: คุณสามารถใช้เครื่องมือบล็อกคุกกี้ของเบราว์เซอร์หรือเครื่องมือปฏิเสธเพื่อจำกัดการเก็บรวบรวมข้อมูล Google Analytics บนเวอร์ชัน Web
  • อีเมลธุรกรรม: บนเวอร์ชัน Web คุณจะได้รับอีเมลที่เกี่ยวข้องกับบัญชี (ต้อนรับ การเตือนช่วงทดลองใช้ การแจ้งเตือนการสมัครสมาชิก) สิ่งเหล่านี้เป็นการสื่อสารเกี่ยวกับบริการที่จำเป็นสำหรับการจัดการสมัครสมาชิกของคุณ
  • การลบข้อมูล: คุณสามารถออกจากระบบ Google account ภายในแอปได้ทุกเมื่อ ซึ่งจะลบ OAuth tokens ที่จัดเก็บไว้ออกจากอุปกรณ์ หากต้องการขอลบบันทึกผู้ใช้และข้อมูลที่เกี่ยวข้องบนเซิร์ฟเวอร์ของเรา ติดต่อเราที่ support@ingantt.com

6. ความเป็นส่วนตัวของเด็ก

แอปไม่ได้มุ่งเป้าไปที่เด็กอายุต่ำกว่า 13 ปี (หรืออายุที่บังคับใช้ในเขตอำนาจศาลของคุณ) เราไม่เก็บรวบรวมข้อมูลส่วนบุคคลจากเด็กโดยเจตนา หากคุณเชื่อว่าเด็กได้ให้ข้อมูลส่วนบุคคลแก่เรา กรุณาติดต่อเราที่ support@ingantt.com เพื่อให้เราดำเนินการที่เหมาะสม

7. การโอนข้อมูลระหว่างประเทศ

เซิร์ฟเวอร์แบ็กเอนด์ของเราโฮสต์บน Microsoft Azure ใน East US region รายงานข้อขัดข้องประมวลผลโดย Sentry ในสหภาพยุโรป หากคุณอยู่นอกสหรัฐอเมริกา ข้อมูลของคุณอาจถูกโอนไปยังและประมวลผลในสหรัฐอเมริกาหรือเขตอำนาจศาลอื่น ๆ ที่ผู้ให้บริการของเราดำเนินงาน

8. การเปลี่ยนแปลงนโยบายความเป็นส่วนตัวนี้

เราอาจปรับปรุงนโยบายความเป็นส่วนตัวนี้เป็นครั้งคราว เราจะแจ้งให้คุณทราบเกี่ยวกับการเปลี่ยนแปลงที่สำคัญโดยการอัปเดตวันที่ "อัปเดตล่าสุด" ที่ด้านบนของนโยบายนี้ การใช้งานแอปอย่างต่อเนื่องหลังจากการเปลี่ยนแปลงใด ๆ ถือว่าคุณยอมรับนโยบายที่ปรับปรุงแล้ว

  • แอปขอเฉพาะสิทธิ์การเข้าถึงไฟล์ที่คุณอนุญาตอย่างชัดเจนผ่านตัวเลือกไฟล์ของระบบ

Android

Windows

  • แอปเป็นแอปพลิเคชันแบบชำระเงินที่แจกจ่ายผ่าน Microsoft Store แนวปฏิบัติด้านความเป็นส่วนตัวของ Microsoft ใช้กับธุรกรรมการซื้อ

Web
