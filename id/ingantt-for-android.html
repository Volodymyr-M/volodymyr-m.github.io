07Pertanyaan yang Sering Diajukan

Apa itu Ingantt untuk Android?

Ingantt untuk Android adalah alat manajemen proyek profesional yang berjalan secara native di perangkat Android Anda. Anda dapat membuat dan mengelola rencana proyek saat bepergian, dengan perencanaan bertenaga AI dan kompatibilitas MS Project.

Dapatkah saya mengimpor file Microsoft Project saya yang sudah ada?

Bisa! Ingantt sepenuhnya mendukung impor file .MPP dan .XML dari Microsoft Project. Data proyek Anda, termasuk tugas, sumber daya, dan dependensi, akan dipertahankan selama proses impor.

Bagaimana cara kerja pembuatan proyek AI?

Asisten AI kami dapat menghasilkan rencana proyek lengkap dari deskripsi teks sederhana. Cukup jelaskan tujuan dan kebutuhan proyek Anda, dan AI akan membuat rencana terstruktur dengan tugas, garis waktu, dan dependensi berdasarkan praktik terbaik.

Apakah ada uji coba gratis untuk versi Android?

Tidak. Versi Android adalah pembelian satu kali, tanpa langganan atau uji coba gratis.

Bagaimana saya bisa membeli Ingantt untuk Android?

Cukup kunjungi Google Play Store di perangkat Android Anda dan beli di sana. Anda memiliki aplikasinya selamanya tanpa biaya berulang.
