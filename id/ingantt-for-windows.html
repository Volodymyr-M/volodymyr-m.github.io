Pertanyaan yang Sering Diajukan

Apa itu Ingantt untuk Windows?

Ingantt untuk Windows adalah alat manajemen proyek profesional yang berjalan secara native di PC Anda. Alat ini memungkinkan Anda membuat dan mengelola rencana proyek dengan perencanaan bertenaga AI dan kompatibilitas MS Project.

Bisakah saya mengimpor file Microsoft Project yang sudah ada?

Ya! Ingantt mendukung impor file .MPP dan .XML dari Microsoft Project sepenuhnya. Data proyek Anda, termasuk tugas, sumber daya, dan ketergantungan, akan tetap terjaga selama proses impor.

Bagaimana cara kerja pembuatan proyek dengan AI?

Asisten AI kami dapat menghasilkan rencana proyek lengkap dari deskripsi teks sederhana. Cukup jelaskan tujuan dan kebutuhan proyek Anda, dan AI akan membuat rencana terstruktur dengan tugas, jadwal, dan ketergantungan berdasarkan praktik terbaik.

Apakah ada uji coba gratis untuk versi Windows?

Ya. Setelah diinstal dari Microsoft Store, uji coba gratis tersedia selama 7 hari.

Bagaimana saya dapat membeli Ingantt untuk Windows?

Kunjungi Microsoft Store dan beli di sana. Anda akan memiliki aplikasi ini selamanya tanpa biaya berulang.
