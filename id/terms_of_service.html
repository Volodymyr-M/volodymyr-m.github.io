Syarat Layanan

Terakhir Diperbarui: 22 Februari 2026

Syarat Layanan ("Syarat") ini mengatur akses dan penggunaan Anda terhadap Ingantt ("Aplikasi," "Layanan"), sebuah aplikasi penjadwalan proyek yang disediakan oleh Ingantt Development ("kami," "milik kami"). Dengan mengakses atau menggunakan Aplikasi, Anda setuju untuk terikat oleh Syarat ini. Jika Anda tidak setuju, jangan gunakan Aplikasi.

1. Deskripsi Layanan

Ingantt adalah aplikasi penjadwalan proyek lintas platform yang menyediakan:

  • Penjadwalan proyek menggunakan algoritma Critical Path Method (CPM)
  • Manajemen tugas dengan dependensi, batasan, dan penugasan sumber daya
  • Visualisasi dan rendering Gantt chart
  • Manajemen sumber daya dan pemerataan sumber daya
  • Manajemen kalender dan jadwal kerja
  • Impor dan ekspor proyek dalam format XML (kompatibel dengan Microsoft Project), CSV, PDF, dan Markdown
  • Konversi file Microsoft Project (.mpp) melalui server kami
  • Integrasi Google Drive untuk penyimpanan dan berbagi file
  • Pembuatan proyek berbasis AI ("Create with AI")

Aplikasi tersedia di Web (web.ingantt.com dan melalui Google Workspace Marketplace), iOS, Android, macOS, dan Windows.

2. Kelayakan

Anda harus berusia minimal 13 tahun untuk menggunakan Aplikasi. Jika Anda berusia antara 13 dan 18 tahun, Anda harus memiliki persetujuan orang tua atau wali. Dengan menggunakan Aplikasi, Anda menyatakan dan menjamin bahwa Anda memenuhi persyaratan ini.

3. Akun dan Autentikasi

3.1 Google Sign-In

Versi web Aplikasi memerlukan Google Sign-In untuk autentikasi. Pada platform lain, Google Sign-In mungkin tersedia untuk fitur seperti integrasi Google Drive. Dengan masuk, Anda mengizinkan kami untuk mengakses informasi profil dasar Anda (nama dan alamat email) untuk identifikasi akun, manajemen lisensi, dan penyediaan layanan.

3.2 Keamanan Akun

Anda bertanggung jawab untuk menjaga keamanan kredensial akun Google Anda. Anda setuju untuk segera memberitahu kami di support@ingantt.com mengenai penggunaan akun Anda yang tidak sah.

4. Pembelian dan Pembayaran

4.1 Model Pembelian

Aplikasi menggunakan model pembelian yang berbeda tergantung pada platform:

  • Web (web.ingantt.com dan Google Workspace Marketplace): Aplikasi beroperasi berdasarkan langganan dengan paket bulanan dan tahunan. Uji coba gratis mungkin tersedia untuk pengguna baru. Fitur tertentu memerlukan langganan aktif untuk diakses.
  • iOS (App Store), Android (Google Play), macOS (App Store), dan Windows (Microsoft Store): Aplikasi tersedia sebagai pembelian satu kali. Semua fitur termasuk dalam pembelian Anda. Tidak diperlukan pembelian dalam aplikasi atau langganan.

4.2 Detail Langganan Web

Untuk versi web Aplikasi:

  • Uji Coba Gratis: Pengguna baru dapat menerima periode uji coba gratis dengan akses ke fitur-fitur Aplikasi. Periode uji coba dimulai saat penggunaan pertama.
  • Paket Langganan: Setelah periode uji coba, akses berkelanjutan memerlukan langganan aktif (bulanan atau tahunan). Harga ditampilkan di dalam Aplikasi.
  • Perpanjangan Otomatis: Langganan diperpanjang secara otomatis pada akhir setiap periode penagihan kecuali dibatalkan sebelum tanggal perpanjangan.
  • Pembatalan: Anda dapat membatalkan langganan Anda kapan saja melalui Aplikasi atau dengan menghubungi support@ingantt.com. Pembatalan berlaku pada akhir periode penagihan saat ini.

4.3 Pemrosesan Pembayaran

Pembayaran diproses melalui platform yang berlaku:

  • iOS dan macOS (App Store): Dikelola melalui Apple dan tunduk pada syarat Apple
  • Android (Google Play): Dikelola melalui Google Play dan tunduk pada syarat Google
  • Windows (Microsoft Store): Dikelola melalui Microsoft dan tunduk pada syarat Microsoft
  • Web: Dikelola melalui pemroses pembayaran pihak ketiga kami (WayForPay) dan tunduk pada syarat mereka

4.4 Pengembalian Dana

Kebijakan pengembalian dana diatur oleh platform tempat Anda melakukan pembelian:

  • Apple App Store: Tunduk pada kebijakan pengembalian dana Apple
  • Google Play Store: Tunduk pada kebijakan pengembalian dana Google
  • Microsoft Store: Tunduk pada kebijakan pengembalian dana Microsoft
  • Web: Hubungi support@ingantt.com untuk permintaan pengembalian dana dalam 7 hari setelah pembelian

5. Konten dan Data Pengguna

5.1 File Proyek Anda

Anda mempertahankan kepemilikan penuh atas semua file proyek, jadwal, dan data yang Anda buat atau impor menggunakan Aplikasi ("Konten Pengguna"). Kami tidak mengklaim hak kepemilikan apa pun atas Konten Pengguna Anda.

5.2 Penyimpanan Lokal

File proyek dapat disimpan secara lokal di perangkat Anda. Anda bertanggung jawab untuk memelihara cadangan file lokal Anda. Kami tidak bertanggung jawab atas kehilangan data yang diakibatkan oleh kegagalan perangkat, penghapusan instalasi aplikasi, atau kejadian lain di luar kendali kami.

5.3 Penyimpanan Google Drive

Jika Anda memilih untuk menggunakan integrasi Google Drive, file Anda disimpan dalam folder bernama "Ingantt Schedules" di akun Google Drive pribadi Anda dan tunduk pada Syarat Layanan dan Kebijakan Privasi Google. Aplikasi menggunakan cakupan terbatas drive.file, yang berarti hanya dapat mengakses file yang dibuatnya atau file yang secara eksplisit Anda buka di Aplikasi — Aplikasi tidak memiliki akses ke seluruh Google Drive Anda. Kami mengakses Google Drive Anda hanya untuk membaca, menulis, dan mengelola file proyek sesuai arahan Anda.

5.4 Simpan Otomatis

Aplikasi dapat secara otomatis menyimpan pekerjaan Anda ke penyimpanan internal sebagai langkah pengamanan terhadap kehilangan data. Data yang disimpan otomatis disimpan secara lokal di perangkat Anda.

5.5 Data yang Kami Kumpulkan

Dalam rangka menyediakan Layanan, kami dapat mengumpulkan dan memproses:

  • Alamat email dan nama Anda (untuk autentikasi dan manajemen lisensi)
  • Informasi pembayaran (diproses oleh penyedia pembayaran pihak ketiga; kami tidak menyimpan detail pembayaran lengkap)
  • Data penggunaan dan error aplikasi (untuk pelaporan kerusakan dan peningkatan stabilitas)
  • Prompt AI yang Anda kirimkan saat menggunakan fitur "Create with AI" (diteruskan ke penyedia AI pihak ketiga, tidak disimpan di server kami)
  • Umpan balik yang Anda kirimkan, yang mungkin menyertakan lampiran file proyek

Untuk detail lengkap tentang pengumpulan dan penggunaan data, silakan tinjau Kebijakan Privasi kami.

6. Fitur Bertenaga AI

6.1 Create with AI

Aplikasi menawarkan pembuatan proyek bertenaga AI. Saat Anda menggunakan fitur ini:

  • Prompt teks Anda dikirim ke server kami, yang meneruskannya ke Anthropic (Claude) untuk diproses. Kami tidak menyimpan prompt Anda di server kami. Pemrosesan prompt Anda oleh Anthropic tunduk pada syarat layanan dan kebijakan privasi mereka.
  • Jadwal proyek yang dihasilkan disediakan sebagai titik awal dan harus ditinjau keakuratannya
  • Kami tidak menjamin keakuratan, kelengkapan, atau kesesuaian konten yang dihasilkan AI
  • Anda sepenuhnya bertanggung jawab untuk meninjau dan memvalidasi jadwal yang dihasilkan AI sebelum mengandalkannya

6.2 Bukan Saran Profesional

Konten yang dihasilkan AI bukan merupakan saran manajemen proyek profesional. Anda harus menggunakan penilaian independen saat menggunakan rencana proyek yang dihasilkan AI.

7. Kompatibilitas Microsoft Project

7.1 Dukungan Format File

Aplikasi mendukung impor dan ekspor file XML yang kompatibel dengan Microsoft Project. Aplikasi juga dapat mengonversi berbagai format file proyek (termasuk .mpp, .mpt, .mpx, .gan, .xer, dan lainnya) ke XML melalui layanan konversi sisi server kami. File yang diunggah diproses di memori dan tidak disimpan di server kami.

7.2 Penyangkalan Kompatibilitas

Meskipun kami berusaha untuk kompatibilitas dengan format file Microsoft Project, kami tidak menjamin kesetiaan sempurna dalam semua kasus. Beberapa fitur, pemformatan, atau data yang ada di file Microsoft Project mungkin tidak sepenuhnya didukung. Anda harus memverifikasi file yang dikonversi untuk keakuratannya.

7.3 Tanpa Afiliasi

Ingantt tidak berafiliasi dengan, didukung oleh, atau disponsori oleh Microsoft Corporation. "Microsoft Project" adalah merek dagang dari Microsoft Corporation.
