Pertanyaan yang Sering Diajukan

Apa itu Ingantt untuk macOS?

Ingantt untuk macOS adalah alat manajemen proyek profesional yang berjalan secara native di Mac Anda. Alat ini memungkinkan Anda membuat dan mengelola rencana proyek dengan bantuan AI dan kompatibilitas MS Project.

Bisakah saya mengimpor file Microsoft Project yang sudah ada?

Tentu! Ingantt sepenuhnya mendukung impor file .MPP dan .XML dari Microsoft Project. Data proyek Anda, termasuk tugas, sumber daya, dan ketergantungan, akan dipertahankan selama proses impor.

Bagaimana cara kerja pembuatan proyek dengan AI?

Asisten AI kami dapat membuat rencana proyek lengkap dari deskripsi teks sederhana. Cukup jelaskan tujuan dan persyaratan proyek Anda, dan AI akan menghasilkan rencana terstruktur dengan tugas, jadwal, dan ketergantungan berdasarkan praktik terbaik.

Apakah ada uji coba gratis untuk versi macOS?

Tidak. Versi macOS adalah pembelian sekali bayar, tanpa langganan maupun uji coba gratis.

Bagaimana cara membeli Ingantt untuk macOS?

Kunjungi Mac App Store dan beli di sana. Anda akan memiliki aplikasi tersebut selamanya tanpa biaya berulang.
