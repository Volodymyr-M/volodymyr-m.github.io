04Membuka dan Melihat File .MPP — Pertanyaan yang Benar-Benar Ditanyakan Orang

Bagaimana cara membuka file .MPP tanpa Microsoft Project?

Gunakan Ingantt di browser Anda. Ia membaca format biner .MPP secara langsung — tidak perlu lisensi Microsoft Project, mesin virtual Windows, atau layanan konversi yang mengirim PDF melalui email. Ingantt mengimpor setiap tugas, sub-tugas, dependensi, penugasan sumber daya, tonggak, dan jalur kritis, dan merender proyek sebagai Gantt chart interaktif penuh. Uji coba gratis 7 hari cukup untuk sebagian besar situasi membuka file satu kali, dan tidak perlu kartu kredit untuk memulai.

Apa itu Gantt viewer, dan apakah Ingantt termasuk?

Gantt viewer adalah alat yang membuka file Gantt chart — biasanya file Microsoft Project .MPP atau .XML — dan menampilkan proyek sebagai timeline interaktif. Viewer yang sebenarnya memungkinkan Anda menggulir jadwal, memperbesar dan memperkecil, memperluas atau menciutkan fase, dan mengarahkan ke tugas untuk melihat dependensi, tanggal mulai dan akhir, serta sumber daya. Ingantt adalah Gantt viewer dan editor berbasis browser, jadi Anda dapat membuka file yang sama di perangkat apa pun yang Anda miliki — dan mengekspor perubahan kembali ke MS Project XML saat Anda perlu.

Bisakah saya membuka file .MPP atau melihat Gantt chart di browser?

Ya. Ingantt berjalan sepenuhnya di browser — Chrome, Safari, Firefox, atau Edge — di Mac, Windows, Linux, Chromebook, iPhone, iPad, atau Android. Masuk dengan akun Google, unggah file .MPP atau pilih dari Google Drive, dan seluruh proyek terbuka di browser dengan zoom, gulir, dan arahkan untuk detail. Tidak ada yang diinstal secara lokal dan tidak ada yang diunduh. Ini adalah jalur yang tepat jika Anda menggunakan mesin kerja yang tidak dapat menginstal perangkat lunak baru, atau di Linux atau Chromebook di mana Microsoft Project tidak pernah menjadi pilihan.

Bisakah saya mencetak Gantt chart saya, atau mengekspornya ke PDF atau PNG?

Ya, ketiganya. Ingantt mencetak Gantt chart langsung dari browser, dan mengekspor ke PDF dan PNG pada tingkat zoom mana pun — jadi baik Anda memerlukan gambaran satu halaman atau timeline seukuran dinding di beberapa halaman, tata letaknya tetap bersih. PDF adalah pilihan biasa untuk berbagi hanya-baca dengan pemangku kepentingan yang tidak perlu menyentuh file asli; PNG berguna untuk dimasukkan ke slide, dokumen Notion, atau pesan Slack; pencetakan asli paling cepat ketika Anda hanya ingin salinan cetak.

Bisakah saya menyimpan file .MPP kembali setelah dibuka di Ingantt?

Tidak sebagai .MPP — Ingantt mengekspor ke MS Project XML, yang langsung dibuka Microsoft Project. Format biner .MPP bersifat proprietary, tidak terdokumentasi, dan pada dasarnya merupakan target tulis khusus Microsoft; MS Project XML adalah format pertukaran milik Microsoft sendiri dan bolak-balik dengan bersih. Jadi Anda dapat membuka .MPP di Ingantt, melakukan pengeditan, mengekspor sebagai MS Project XML, dan menyerahkan file .xml ke rekan yang masih menggunakan Microsoft Project — mereka membukanya seperti mereka membuka file proyek apa pun. Untuk berbagi hanya-baca dengan orang yang tidak menggunakan Project sama sekali, PDF atau PNG adalah ekspor yang tepat.

Apakah saya akan kehilangan dependensi, sumber daya, atau jalur kritis saat membuka .MPP di luar Microsoft Project?

Di Ingantt, tidak — impornya lossless untuk bidang yang penting. Tugas, sub-tugas, dependensi finish-to-start, penugasan sumber daya, pekerjaan dan durasi, tonggak, dan perhitungan jalur kritis semuanya masuk dari file .MPP asli. Ingantt menggunakan struktur proyek yang sama dengan yang digunakan Microsoft Project secara internal, jadi Gantt chart yang membutuhkan waktu seharian untuk dibangun di Project masih terlihat seperti Gantt chart yang sama setelah Anda membukanya di Ingantt. Penampil gratis yang hanya menunjukkan daftar tugas atau meratakan proyek menjadi gambar kehilangan grafik dependensi, yang biasanya merupakan bagian paling penting dari rencana.

Apakah ada pembuka .MPP gratis atau penampil Gantt chart?

Ingantt memiliki uji coba gratis 7 hari tanpa perlu kartu kredit untuk memulai, dan uji coba mencakup impor .MPP penuh, melihat, mengedit, mencetak, dan ekspor PDF/PNG/XML — tidak ada yang dikunci di balik pemisahan "edisi penampil" vs. "edisi editor". Bagi sebagian besar orang yang hanya perlu membuka atau melihat file .MPP satu atau beberapa kali, uji coba gratis sepenuhnya mencakup situasinya. Setelah uji coba, paket berbayar mulai dari $5.99 per pengguna per bulan. Jika Anda bekerja dengan vendor atau klien yang masih menggunakan Microsoft Project, paket berbayar lebih murah daripada biaya satu bulan Microsoft 365 Project.

Apakah saya perlu tahu manajemen proyek untuk menggunakan Gantt viewer?

Tidak. Ingantt dirancang agar pemangku kepentingan non-PM — eksekutif yang meninjau tanggal pengiriman, klien yang memeriksa tonggak, anggota tim baru yang sedang berorientasi — dapat membuka Gantt chart dan membacanya tanpa mempelajari hal baru. Anda menggulir timeline untuk melihat kapan hal-hal terjadi, mengarahkan ke tugas untuk melihat siapa yang mengerjakannya, dan memperluas fase untuk melihat tugas di dalamnya. Tidak ada jargon yang harus diuraikan dan tidak ada langkah pengaturan selain membuka file.
