Ingantt

Kebijakan Privasi

Tanggal Berlaku: 22 Februari 2026 Terakhir Diperbarui: 22 Februari 2026

Ingantt Development ("kami" atau "milik kami") mengoperasikan aplikasi Ingantt ("Aplikasi") dan situs web di https://ingantt.com ("Situs Web"). Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan membagikan informasi saat Anda menggunakan Aplikasi di semua platform yang didukung (iOS, Android, macOS, Windows, dan Web, termasuk melalui Google Workspace Marketplace).

1. Informasi yang Kami Kumpulkan

1.1 Informasi yang Anda Berikan

  • Informasi Akun Google: Jika Anda masuk dengan Google, kami menerima alamat email, nama tampilan, dan URL foto profil Anda dari Google. Pada platform native (iOS, Android, macOS, Windows), masuk bersifat opsional dan mengaktifkan penyimpanan file Google Drive. Pada versi Web, Google Sign-In diperlukan dan digunakan untuk penyimpanan file Google Drive dan manajemen langganan. Saat Anda pertama kali menggunakan versi Web, catatan pengguna dibuat di server kami yang berisi alamat email, nama, lokale, dan tanggal pendaftaran Anda.

  • Informasi Langganan dan Penagihan (khusus Web): Saat membeli langganan melalui Web, kami mengumpulkan nama, alamat email, alamat surat, nomor telepon, dan detail terkait pembayaran Anda. Informasi ini dikirimkan ke WayForPay, pemroses pembayaran kami, untuk pemrosesan transaksi.

  • Umpan Balik: Saat Anda mengirimkan umpan balik melalui Aplikasi, kami mengumpulkan teks pesan Anda. Anda dapat secara opsional memilih untuk melampirkan file proyek yang sedang dibuka untuk membantu kami mendiagnosis masalah.

  • Input Fitur AI: Saat Anda menggunakan fitur "Create with AI", prompt teks yang Anda berikan dikirim ke server backend kami untuk diproses oleh Anthropic (Claude).

  • File Proyek: File proyek yang Anda buat dan edit tetap di bawah kendali Anda. File disimpan secara lokal di perangkat Anda atau di akun Google Drive Anda (dalam folder bernama "Ingantt Schedules"). Saat Anda membuka file dalam format non-native (seperti MPP, GAN, XER, atau format file proyek lain yang didukung), file dikirimkan ke server kami untuk konversi format. File diproses di memori saja dan tidak disimpan di server kami.

1.2 Informasi yang Dikumpulkan Secara Otomatis

  • Metadata Aplikasi: Setiap permintaan ke backend kami mencakup versi aplikasi Anda, nama platform (iOS, Android, macOS, Windows, atau Web), lokale sistem, dan, jika masuk, alamat email Anda.

  • Laporan Kerusakan dan Error: Kami menggunakan Sentry (dihosting di UE) untuk mengumpulkan laporan kerusakan dan log error di produksi. Ini termasuk stack trace, pesan error, versi aplikasi, dan informasi platform. Kami tidak dengan sengaja mengumpulkan data pribadi melalui laporan kerusakan.

  • Analitik Web (khusus Web): Versi Web Aplikasi menggunakan Google Analytics 4 untuk mengumpulkan analitik penggunaan standar, termasuk tampilan halaman dan data sesi. Google Analytics dapat menyetel cookie di browser Anda. Versi mobile dan desktop native Aplikasi tidak menyertakan pelacakan analitik.

  • Preferensi Pengguna: Aplikasi menyimpan preferensi lokal Anda (seperti tema, opsi tampilan, lebar kolom, dan daftar file terbaru) di perangkat Anda menggunakan mekanisme penyimpanan platform standar. Preferensi ini tidak dikirimkan ke server kami.

1.3 Informasi yang Tidak Kami Kumpulkan

  • Kami tidak menggunakan data Anda untuk melatih model kecerdasan buatan atau pembelajaran mesin.
  • Kami tidak menayangkan iklan dan tidak mengintegrasikan SDK iklan apa pun.
  • Kami tidak mengumpulkan data lokasi, kontak, foto, input kamera, atau input mikrofon.
  • Kami tidak menggunakan identifikasi perangkat untuk tujuan pelacakan.
  • Pencadangan cloud Android secara eksplisit dinonaktifkan untuk Aplikasi.

2. Bagaimana Kami Menggunakan Informasi Anda

Kami menggunakan informasi yang kami kumpulkan untuk tujuan berikut:

  • Menyediakan dan mengoperasikan Aplikasi: Memproses permintaan Anda, mengaktifkan penyimpanan dan sinkronisasi file melalui Google Drive, mengonversi format file, dan menyediakan pembuatan proyek bertenaga AI.
  • Memproses langganan (khusus Web): Memproses pembayaran langganan Web, memvalidasi status langganan, dan mengelola akun Anda.
  • Mengirim email transaksional (khusus Web): Mengirimkan pemberitahuan terkait akun seperti pesan selamat datang, pengingat uji coba, konfirmasi langganan, dan pemberitahuan pembatalan.
  • Meningkatkan stabilitas dan kinerja: Menganalisis laporan kerusakan, data error, dan telemetri sisi server untuk mengidentifikasi dan memperbaiki bug.
  • Memahami pola penggunaan (khusus Web): Menggunakan data analitik agregat untuk memahami bagaimana aplikasi Web digunakan dan untuk meningkatkan pengalaman pengguna.
  • Menanggapi permintaan dukungan: Menggunakan kiriman umpan balik untuk mendiagnosis masalah dan meningkatkan Aplikasi.

3. Bagaimana Kami Membagikan Informasi Anda

Kami tidak menjual informasi pribadi Anda. Kami membagikan informasi hanya dalam keadaan berikut:

  • Penyedia Layanan: Kami menggunakan penyedia layanan pihak ketiga berikut untuk mengoperasikan Aplikasi:

    • Sentry (Functional Software, Inc.) — pelaporan kerusakan dan error sisi klien, data diproses di UE
    • Microsoft Application Insights (Microsoft Corporation) — telemetri dan pemantauan error sisi server
    • Google Analytics (Google LLC) — analitik penggunaan web (khusus versi Web)
    • Google APIs (Google LLC) — autentikasi dan penyimpanan file Google Drive
    • WayForPay — pemrosesan pembayaran langganan (khusus versi Web)
    • Resend (Resend, Inc.) — pengiriman email transaksional (khusus versi Web)
    • Anthropic (Claude) — memproses prompt pembuatan proyek bertenaga AI
    • Microsoft Azure — hosting backend dan penyimpanan data (wilayah East US)

  • Persyaratan Hukum: Kami dapat mengungkapkan informasi jika diwajibkan oleh hukum atau sebagai tanggapan terhadap proses hukum yang sah.

4. Penyimpanan dan Keamanan Data

  • Data pengguna sisi server (khusus Web): Alamat email, nama, lokale, tanggal pendaftaran, dan status langganan Anda disimpan di Azure Table Storage di server backend kami.
  • Token OAuth (token akses, refresh, dan ID Google) disimpan di penyimpanan aman platform (iOS Keychain, Android Keystore, macOS Keychain, atau yang setara di platform lain).
  • Status langganan (khusus Web) di-cache secara lokal di penyimpanan aman untuk akses offline.
  • Permintaan API ke backend kami ditandatangani dengan HMAC-MD5 untuk verifikasi integritas.
  • Aplikasi beroperasi dalam sandbox platform jika tersedia (macOS App Sandbox, iOS App Sandbox).
  • Kami menggunakan HTTPS untuk semua komunikasi jaringan.

4a. Retensi Data

  • Catatan pengguna sisi server (khusus Web): Catatan pengguna dan riwayat langganan Anda disimpan selama akun Anda ada. Untuk meminta penghapusan data Anda, hubungi kami di support@ingantt.com.
  • File proyek: File yang dikirim untuk konversi format diproses di memori dan tidak disimpan.
  • Prompt AI: Prompt teks yang dikirim untuk pembuatan proyek bertenaga AI tidak disimpan di server kami.
  • Umpan balik: Pesan umpan balik diteruskan ke tim dukungan kami melalui email dan tidak disimpan dalam basis data.
  • Data lokal: Preferensi, token OAuth, dan status langganan yang di-cache tetap ada di perangkat Anda hingga Anda keluar atau menghapus instalasi Aplikasi.

5. Pilihan dan Hak Anda

  • Masuk bersifat opsional pada platform native: Pada iOS, Android, macOS, dan Windows, Anda dapat menggunakan Aplikasi tanpa masuk ke Google. Masuk mengaktifkan penyimpanan file Google Drive. Pada versi Web, Google Sign-In diperlukan untuk fungsionalitas aplikasi, penyimpanan file, dan manajemen langganan.
  • Akses Google Drive: Aplikasi menggunakan cakupan terbatas drive.file, yang hanya mengizinkan akses ke file yang dibuat oleh Aplikasi atau yang secara eksplisit Anda buka di Aplikasi — bukan seluruh Google Drive Anda.
  • Lampiran umpan balik: Melampirkan file proyek Anda saat mengirimkan umpan balik bersifat opsional dan memerlukan persetujuan eksplisit Anda.
  • Analitik web: Anda dapat menggunakan pemblokiran cookie berbasis browser atau alat opt-out untuk membatasi pengumpulan data Google Analytics pada versi Web.
  • Email transaksional: Pada versi Web, Anda akan menerima email terkait akun (selamat datang, pengingat uji coba, pemberitahuan langganan). Ini adalah komunikasi layanan yang diperlukan untuk mengelola langganan Anda.
  • Penghapusan data: Anda dapat keluar dari akun Google Anda di dalam Aplikasi kapan saja, yang akan menghapus token OAuth yang tersimpan dari perangkat. Untuk meminta penghapusan catatan pengguna Anda dan data terkait di server kami, hubungi kami di support@ingantt.com.

6. Privasi Anak-Anak

Aplikasi tidak ditujukan untuk anak-anak di bawah usia 13 tahun (atau usia yang berlaku di yurisdiksi Anda). Kami tidak dengan sengaja mengumpulkan informasi pribadi dari anak-anak. Jika Anda yakin seorang anak telah memberikan informasi pribadi kepada kami, silakan hubungi kami di support@ingantt.com agar kami dapat mengambil tindakan yang tepat.

7. Transfer Data Internasional

Server backend kami dihosting di Microsoft Azure di wilayah East US. Laporan kerusakan diproses oleh Sentry di UE. Jika Anda berada di luar Amerika Serikat, informasi Anda dapat ditransfer ke dan diproses di Amerika Serikat atau yurisdiksi lain tempat penyedia layanan kami beroperasi.

8. Perubahan Kebijakan Privasi Ini

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Kami akan memberi tahu Anda tentang perubahan material dengan memperbarui tanggal "Terakhir Diperbarui" di bagian atas kebijakan ini. Penggunaan Anda yang berkelanjutan atas Aplikasi setelah perubahan apa pun merupakan penerimaan Anda terhadap kebijakan yang diperbarui.

9. Pengungkapan Khusus Platform

iOS dan macOS (Apple)

  • Aplikasi adalah aplikasi berbayar yang didistribusikan melalui Apple App Store. Praktik privasi Apple berlaku untuk transaksi pembelian.
  • Aplikasi hanya meminta izin akses file yang Anda berikan secara eksplisit melalui pemilih file sistem.

Android

  • Aplikasi adalah aplikasi berbayar yang didistribusikan melalui Google Play. Praktik privasi Google berlaku untuk transaksi pembelian.
  • Aplikasi meminta izin baca/tulis penyimpanan untuk operasi file lokal dan izin internet untuk akses jaringan.

Windows

  • Aplikasi adalah aplikasi berbayar yang didistribusikan melalui Microsoft Store. Praktik privasi Microsoft berlaku untuk transaksi pembelian.

Web

  • Versi Web menggunakan model langganan. Pembayaran langganan diproses melalui WayForPay.
  • Aplikasi juga tersedia melalui Google Workspace Marketplace.
  • Cookie Google Analytics disetel pada domain web.ingantt.com.
  • Autentikasi Google OAuth menggunakan entri localStorage sementara untuk callback masuk, yang segera dihapus setelah digunakan.
