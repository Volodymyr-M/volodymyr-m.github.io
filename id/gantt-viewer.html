Penampil Gantt — Pertanyaan Umum

Bagaimana cara melihat bagan Gantt tanpa Microsoft Project?

Buka file di Ingantt. Aplikasi ini mengimpor format .MPP asli Microsoft Project dan MS Project XML secara langsung, sehingga file dari rekan yang menggunakan MS Project terbuka sebagai bagan Gantt yang sepenuhnya interaktif — tanpa lisensi MS Project atau instalasi Windows. Tugas, dependensi, dan jalur kritis tetap utuh.

Apakah ada penampil Gantt gratis?

Ingantt menawarkan uji coba gratis 7 hari tanpa kartu kredit. Uji coba termasuk pelihatan dan pengeditan lengkap file .MPP dan MS Project XML di semua platform. Setelah uji coba, paket berbayar mulai dari $5,99 per pengguna per bulan. Sebagian besar kebutuhan melihat satu kali muat dalam uji coba gratis.

Bisakah saya mencetak atau membagikan bagan Gantt yang saya lihat?

Ya. Ingantt mengekspor bagan Gantt ke PDF atau PNG untuk dicetak, dilampirkan ke email, atau ditempelkan ke slide. Ekspor PDF mempertahankan tata letak timeline di tingkat zoom mana pun. Jika Anda perlu membuka .MPP yang sama di tempat lain nanti, lihat panduan kami tentang membuka file .MPP tanpa Microsoft Project.
