Pertanyaan yang Sering Diajukan

Apa itu Ingantt untuk iOS?

Ingantt untuk iOS adalah alat manajemen proyek profesional yang berjalan secara native di perangkat iOS Anda. Anda dapat membuat dan mengelola rencana proyek saat bepergian, dengan bantuan AI serta kompatibilitas MS Project.

Dapatkah saya mengimpor file Microsoft Project yang sudah ada?

Tentu! Ingantt sepenuhnya mendukung impor file .MPP dan .XML dari Microsoft Project. Semua data proyek Anda, termasuk tugas, sumber daya, dan ketergantungan, akan dipertahankan selama proses impor.

Bagaimana cara kerja pembuatan proyek dengan AI?

Asisten AI kami dapat menghasilkan rencana proyek lengkap dari deskripsi teks sederhana. Cukup jelaskan tujuan dan persyaratan proyek Anda, dan AI akan membuat rencana terstruktur dengan tugas, timeline, dan ketergantungan berdasarkan praktik terbaik.

Apakah ada uji coba gratis untuk versi iOS?

Tidak. Versi iOS memerlukan pembelian satu kali, tanpa langganan atau masa uji coba gratis.

Bagaimana cara membeli Ingantt untuk iOS?

Cukup kunjungi Apple App Store di perangkat iOS Anda, cari “Ingantt”, dan lakukan pembelian di sana. Anda akan memiliki aplikasi tersebut selamanya tanpa biaya berulang.
