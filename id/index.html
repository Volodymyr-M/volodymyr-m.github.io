Pertanyaan yang Sering Diajukan

Apa itu Ingantt dan bagaimana cara kerjanya?

Ingantt adalah alat manajemen proyek Gantt chart profesional yang memungkinkan Anda membuat, mengelola, dan berbagi rencana proyek langsung dari Google Drive Anda, sehingga kolaborasi menjadi mudah untuk tim yang sudah menggunakan ekosistem Google.

Bisakah saya mengimpor file Microsoft Project yang sudah saya miliki?

Ya! Ingantt sepenuhnya mendukung impor file .MPP dan .XML Microsoft Project. Semua data proyek Anda—termasuk tugas, sumber daya, dan dependensi—akan tetap utuh selama proses impor.

Bagaimana cara kerja generasi proyek AI?

Asisten AI kami dapat membuat rencana proyek lengkap dari deskripsi teks sederhana. Cukup jelaskan tujuan dan kebutuhan proyek Anda, dan AI akan menghasilkan rencana terstruktur dengan tugas, tenggat waktu, dan dependensi berdasarkan praktik terbaik manajemen proyek.

Apa saja yang termasuk dalam uji coba gratis?

Uji coba gratis 7 hari mencakup semua fitur Ingantt. Tidak perlu kartu kredit untuk memulai.

Bisakah saya menggunakan Ingantt Web di perangkat seluler?

Ya! Ingantt Web berfungsi di perangkat apa pun dengan browser web. Antarmuka responsif kami sepenuhnya dioptimalkan untuk ponsel dan tablet, sehingga Anda dapat mengakses proyek Anda di mana saja.

Apa yang terjadi pada data saya jika saya membatalkan langganan?

File proyek Anda akan tetap aman di Google Drive Anda bahkan setelah langganan dibatalkan. File proyek Anda disimpan dalam format MS Project, sehingga Anda tidak pernah kehilangan akses ke data Anda.

Bagaimana cara membeli Ingantt untuk rekan tim saya?

Masuk ke Ingantt dengan akun Google Anda, klik avatar di kanan atas, lalu pilih "Beli Langganan". Pilih paket (tahunan atau bulanan), lalu centang "Saya ingin membeli langganan untuk orang lain" dan masukkan akun Google rekan Anda.
