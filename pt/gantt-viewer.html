Visualizador Gantt — Perguntas frequentes

O que é um visualizador Gantt?

Um visualizador Gantt é uma ferramenta que abre um arquivo de gráfico de Gantt — geralmente um Microsoft Project .MPP ou .XML — e exibe o projeto como uma linha do tempo interativa. Um visualizador real permite rolar o cronograma, dar zoom, expandir ou recolher fases e passar o cursor sobre as tarefas para ver dependências, datas de início e fim, e recursos. O Ingantt é um visualizador Gantt que roda em Mac, iOS, iPad, Windows, Android e no navegador, permitindo abrir o mesmo arquivo em qualquer dispositivo.

Como visualizo um gráfico de Gantt sem o Microsoft Project?

Abra o arquivo no Ingantt. Ele importa diretamente o formato .MPP nativo do Microsoft Project e MS Project XML, então um arquivo recebido de um colega que usa MS Project abre como um Gantt completamente interativo — sem licença MS Project ou instalação Windows. Use a versão navegador para um uso pontual ou instale o teste grátis do app Mac, iOS, Windows ou Android. Tarefas, dependências e o caminho crítico chegam intactos.

Posso visualizar um arquivo .MPP no Mac?

Sim. O Ingantt tem um app macOS nativo que abre arquivos .MPP e MS Project XML direto em qualquer Mac recente. Você não precisa de máquina virtual, Parallels ou instalação Windows. Baixe o Ingantt da Mac App Store, abra o arquivo .MPP e verá o Gantt completo com fases, tarefas, dependências e caminho crítico. Se não quiser instalar nada, a versão navegador funciona igual no Safari, Chrome ou Firefox.

Posso visualizar um Gantt no iPhone ou iPad?

Sim. O Ingantt tem apps iOS nativos para iPhone e iPad que abrem arquivos .MPP e MS Project XML igual à versão Mac. No iPad você recebe uma linha do tempo Gantt em largura total com zoom por pinça e rolagem por toque; no iPhone o layout se adapta para que tarefas e marcos continuem legíveis. É a forma mais rápida de olhar um Gantt que um colega acabou de enviar quando você está longe da mesa.

Existe um visualizador Gantt gratuito?

O Ingantt oferece um teste grátis de 7 dias sem cartão de crédito. O teste inclui a visualização e edição completas de arquivos .MPP e MS Project XML em todas as plataformas — nada fica bloqueado atrás de uma "edição visualizador" vs "edição editor". Após o teste, os planos pagos começam em R$29,99/usuário/mês. A maioria das necessidades pontuais de visualização cabe dentro do teste grátis.

Posso ver um Gantt no navegador sem instalar nada?

Sim. O Ingantt tem uma versão navegador que abre arquivos Gantt em Chrome, Safari, Firefox ou Edge no Mac, Windows, Linux, Chromebook ou celular. Entre com uma conta Google, abra um arquivo .MPP do Google Drive ou faça upload, e o Gantt carrega no navegador com zoom, rolagem e detalhes ao passar o cursor. Nada é instalado, nada é baixado.

Preciso saber gestão de projetos?

Não. O Ingantt foi desenhado para que um não especialista — um gestor verificando uma data de entrega, um cliente checando um marco, um membro novo de equipe se orientando — possa abrir um Gantt e lê-lo sem aprender nada. Role a linha do tempo para ver quando as coisas acontecem, passe o cursor sobre uma tarefa para ver quem está nela, expanda uma fase para ver seu conteúdo. Sem jargão, sem configuração.

Posso imprimir ou compartilhar o Gantt que estou vendo?

Sim. O Ingantt exporta o Gantt para PDF ou PNG para imprimir, anexar a um e-mail ou colar numa apresentação. A exportação PDF preserva o layout da linha do tempo em qualquer nível de zoom, útil para entregar uma versão somente leitura a um interessado. Se você precisa reabrir o mesmo .MPP em outro lugar depois, veja nosso guia de abrir arquivos .MPP sem Microsoft Project.
