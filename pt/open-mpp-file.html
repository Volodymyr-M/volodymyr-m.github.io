04Abrir e Visualizar Arquivos .MPP — Perguntas que as Pessoas Realmente Fazem

Como abro um arquivo .MPP sem o Microsoft Project?

Use o Ingantt no seu navegador. Ele lê o formato binário .MPP diretamente — sem necessidade de licença do Microsoft Project, máquina virtual Windows ou um serviço de conversão que te manda um PDF por e-mail. O Ingantt importa cada tarefa, subtarefa, dependência, atribuição de recurso, marco e o caminho crítico, e renderiza o projeto como um gráfico de Gantt interativo completo. O teste grátis de 7 dias é suficiente para a maioria das situações pontuais de abertura de arquivo, e não é preciso cartão de crédito para começar.

O que é um visualizador Gantt, e o Ingantt é um?

Um visualizador Gantt é uma ferramenta que abre um arquivo de gráfico de Gantt — normalmente um .MPP ou .XML do Microsoft Project — e exibe o projeto como uma linha do tempo interativa. Um visualizador de verdade permite rolar pelo cronograma, dar zoom, expandir ou recolher fases, e passar o cursor sobre as tarefas para ver dependências, datas de início e fim, e recursos. O Ingantt é um visualizador e editor de Gantt baseado em navegador, então você pode abrir o mesmo arquivo em qualquer dispositivo à mão — e exportar as alterações de volta para MS Project XML quando precisar.

Posso abrir um arquivo .MPP ou visualizar um gráfico de Gantt no navegador?

Sim. O Ingantt roda inteiramente no navegador — Chrome, Safari, Firefox ou Edge — no Mac, Windows, Linux, Chromebook, iPhone, iPad ou Android. Entre com uma conta Google, envie o arquivo .MPP ou escolha-o no Google Drive, e o projeto completo abre no navegador com zoom, rolagem e detalhes ao passar o cursor. Nada é instalado localmente e nada é baixado. Esse é o caminho certo se você está numa máquina de trabalho onde não dá para instalar software novo, ou em Linux ou Chromebook onde o Microsoft Project nunca foi opção.

Posso imprimir meu gráfico de Gantt, ou exportá-lo para PDF ou PNG?

Sim, os três. O Ingantt imprime o gráfico de Gantt direto do navegador, e exporta para PDF e PNG em qualquer nível de zoom — seja uma visão geral em página única ou uma linha do tempo do tamanho de uma parede por várias páginas, o layout fica limpo. O PDF é a escolha habitual para compartilhamento somente leitura com stakeholders que não precisam mexer no arquivo original; o PNG é prático para jogar em apresentações, documentos do Notion ou mensagens do Slack; a impressão nativa é a mais rápida quando você só quer uma cópia em papel.

Posso salvar o arquivo .MPP de volta depois de abri-lo no Ingantt?

Não como .MPP — o Ingantt exporta para MS Project XML, que o Microsoft Project abre diretamente. O formato binário .MPP é proprietário, não documentado e, na prática, apenas a Microsoft consegue escrever nele; o MS Project XML é o formato de intercâmbio da própria Microsoft e faz o ida e volta sem perdas. Então você pode abrir um .MPP no Ingantt, fazer edições, exportar como MS Project XML e entregar o arquivo .xml a um colega que ainda use o Microsoft Project — ele abre como abriria qualquer arquivo de projeto. Para compartilhamento somente leitura com pessoas que não usam o Project, PDF ou PNG é a exportação certa.

Vou perder dependências, recursos ou o caminho crítico ao abrir um .MPP fora do Microsoft Project?

No Ingantt, não — a importação é sem perdas para os campos que importam. Tarefas, subtarefas, dependências finish-to-start, atribuições de recursos, trabalho e duração, marcos e o cálculo do caminho crítico chegam todos vindos do arquivo .MPP original. O Ingantt usa a mesma estrutura de projeto que o Microsoft Project usa internamente, então um gráfico de Gantt que levou uma tarde para ser montado no Project continua com a mesma cara depois de aberto no Ingantt. Visualizadores gratuitos que só mostram listas de tarefas ou achatam o projeto numa imagem perdem o grafo de dependências, que normalmente é a parte mais importante do plano.

Existe um abridor de .MPP ou visualizador Gantt gratuito?

O Ingantt tem um teste grátis de 7 dias sem cartão de crédito, e o teste inclui importação completa de .MPP, visualização, edição, impressão e exportação para PDF/PNG/XML — nada fica trancado atrás de uma divisão "edição visualizador" vs "edição editor". Para a maioria das pessoas que só precisa abrir ou visualizar um arquivo .MPP uma ou poucas vezes, o teste grátis cobre a situação por completo. Depois do teste, os planos pagos começam em $5.99 por usuário por mês. Se você trabalha com fornecedores ou clientes que ainda usam o Microsoft Project, o plano pago custa menos do que um único mês de Microsoft 365 Project.

Preciso saber gestão de projetos para usar um visualizador Gantt?

Não. O Ingantt foi desenhado para que um stakeholder não especialista — um executivo revisando uma data de entrega, um cliente checando um marco, um novo membro de equipe se orientando — possa abrir um gráfico de Gantt e lê-lo sem aprender nada novo. Você rola a linha do tempo para ver quando as coisas acontecem, passa o cursor sobre uma tarefa para ver quem está nela, e expande uma fase para ver as tarefas dentro. Não há jargão para decifrar nem passo de configuração além de abrir o arquivo.
