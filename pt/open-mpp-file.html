Abrir arquivos .MPP — perguntas de quem está preso

Como abro um arquivo .MPP?

O caminho mais rápido é abrir o arquivo no Ingantt. Um arquivo .MPP é o formato binário nativo do Microsoft Project, e por padrão só o Microsoft Project o abre limpo — mas o MS Project só roda no Windows, custa várias centenas de reais e provavelmente não é o que você quer comprar por um único arquivo. O Ingantt importa .MPP direto, funciona no Mac, iPhone, iPad, Windows, Android e no navegador, com um teste grátis de 7 dias sem cartão de crédito. Abra o app, solte o .MPP, e o projeto completo aparece em cerca de 30 segundos.

Como abro um arquivo .MPP sem Microsoft Project?

Use o Ingantt. Ele lê o formato binário .MPP direto — sem licença Microsoft Project, sem máquina virtual Windows, sem serviço de conversão que manda um PDF por e-mail. O Ingantt importa cada tarefa, subtarefa, dependência, atribuição de recurso, marco e o caminho crítico, e renderiza o projeto como um gráfico de Gantt completo. O teste grátis de 7 dias cobre a maioria das situações pontuais, sem cartão.

Como abro um arquivo .MPP no Mac?

O Microsoft Project não roda no macOS — não existe versão Mac. Então a resposta não é "instale o Project no Mac", é "use algo que leia .MPP nativamente no Mac". O Ingantt tem um app macOS nativo na Mac App Store que abre arquivos .MPP direto em Macs Apple Silicon e Intel. Baixe-o, abra o .MPP e veja o Gantt completo com tarefas, dependências e caminho crítico. Se não quiser instalar nada, a versão navegador funciona igual no Safari, Chrome ou Firefox.

Posso abrir um arquivo .MPP no iPhone ou iPad?

Sim. O Ingantt tem apps iOS nativos para iPhone e iPad na App Store que abrem arquivos .MPP e MS Project XML direto. No iPad você recebe uma linha do tempo larga com zoom por pinça e rolagem por toque — o mais próximo da experiência de Microsoft Project desktop num dispositivo iOS. No iPhone o layout se adapta ao tamanho da tela para que tarefas, datas e marcos continuem legíveis. Normalmente é a forma mais rápida de abrir um .MPP que um colega acabou de enviar.

Posso abrir um arquivo .MPP online no navegador?

Sim. O Ingantt tem uma versão navegador que abre arquivos .MPP em Chrome, Safari, Firefox ou Edge no Mac, Windows, Linux, Chromebook ou celular. Entre com uma conta Google, suba o .MPP ou escolha-o no Google Drive, e o projeto completo abre no navegador. Nada é instalado localmente. É o caminho certo numa máquina de trabalho onde não dá para instalar software, ou no Linux/Chromebook onde o Microsoft Project nunca foi opção.

Existe uma forma gratuita de abrir arquivos .MPP?

O Ingantt oferece um teste grátis de 7 dias sem cartão de crédito, e o teste inclui a importação e edição completas de .MPP — nada fica bloqueado. Para quem só precisa abrir um .MPP uma ou duas vezes, o teste cobre a situação inteira. Depois, os planos pagos começam em R$29,99/usuário/mês. Se você trabalha com .MPP regularmente, o plano é menos do que um único mês de Microsoft 365 Project.

Vou perder dependências, recursos ou o caminho crítico?

No Ingantt, não — a importação é sem perdas para os campos que importam. Tarefas, subtarefas, dependências finish-to-start, atribuições de recursos, trabalho e duração, marcos e o cálculo do caminho crítico chegam todos do arquivo .MPP original. O Ingantt usa a mesma estrutura de projeto que o Microsoft Project usa internamente, então um Gantt que levou uma tarde para montar no Project fica igual depois de aberto no Ingantt. Visualizadores gratuitos que mostram só listas de tarefas ou achatam o projeto numa imagem perdem o grafo de dependências.

Posso salvar o arquivo .MPP depois de abrir?

Sim. O Ingantt salva projetos de volta no formato nativo .MPP do Microsoft Project, além de .XML, PDF e PNG. Você pode abrir um .MPP no Mac, iPhone ou no navegador, fazer edições, salvar de volta como .MPP e entregar o arquivo a um colega que ainda usa Microsoft Project — ele vai abri-lo como sempre. Se você só quer olhar o gráfico sem editar ou re-salvar, nossa página de Visualizador Gantt cobre o caso somente leitura.
