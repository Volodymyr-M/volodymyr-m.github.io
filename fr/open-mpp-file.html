Comment ouvrir un fichier .MPP sans Microsoft Project ?

Utilisez Ingantt dans votre navigateur. Il lit directement le format binaire .MPP — pas besoin de licence Microsoft Project, de machine virtuelle Windows, ni d'un service de conversion qui vous envoie un PDF par e-mail. Ingantt importe chaque tâche, sous-tâche, dépendance, affectation de ressource, jalon et le chemin critique, et restitue le projet sous forme de diagramme de Gantt interactif complet. L'essai gratuit de 7 jours suffit pour la plupart des situations ponctuelles d'ouverture de fichier, et aucune carte bancaire n'est requise pour démarrer.

Qu'est-ce qu'une visionneuse Gantt, et Ingantt en est-il une ?

Une visionneuse Gantt est un outil qui ouvre un fichier de diagramme de Gantt — généralement un fichier .MPP ou .XML de Microsoft Project — et affiche le projet sous forme de chronologie interactive. Une vraie visionneuse vous permet de faire défiler le calendrier, de zoomer et dézoomer, de développer ou replier les phases, et de survoler les tâches pour voir les dépendances, les dates de début et de fin, et les ressources. Ingantt est une visionneuse et un éditeur de Gantt basés sur navigateur, donc vous pouvez ouvrir le même fichier sur l'appareil que vous avez sous la main — et exporter les modifications en MS Project XML lorsque vous en avez besoin.

Puis-je ouvrir un fichier .MPP ou visualiser un diagramme de Gantt dans le navigateur ?

Oui. Ingantt tourne entièrement dans le navigateur — Chrome, Safari, Firefox ou Edge — sur Mac, Windows, Linux, Chromebook, iPhone, iPad ou Android. Connectez-vous avec un Google Account, téléchargez le fichier .MPP ou choisissez-le depuis Google Drive, et le projet complet s'ouvre dans le navigateur avec zoom, défilement et survol pour les détails. Rien n'est installé localement et rien n'est téléchargé. C'est la bonne voie si vous êtes sur une machine de travail sur laquelle vous ne pouvez pas installer de nouveau logiciel, ou sur Linux ou Chromebook où Microsoft Project n'a jamais été une option.

Puis-je imprimer mon diagramme de Gantt, ou l'exporter en PDF ou PNG ?

Oui, les trois. Ingantt imprime le diagramme de Gantt directement depuis le navigateur, et il exporte en PDF et PNG à n'importe quel niveau de zoom — que vous ayez besoin d'un aperçu sur une seule page ou d'une chronologie murale étalée sur plusieurs pages, la mise en page reste nette. Le PDF est le choix habituel pour un partage en lecture seule avec des parties prenantes qui n'ont pas besoin de toucher au fichier d'origine ; le PNG est pratique pour glisser dans des présentations, des documents Notion ou des messages Slack ; l'impression native est la plus rapide quand vous voulez simplement une copie papier.

Puis-je enregistrer le fichier .MPP à nouveau après l'avoir ouvert dans Ingantt ?

Pas au format .MPP — Ingantt exporte en MS Project XML, que Microsoft Project ouvre directement. Le format binaire .MPP est propriétaire, non documenté, et essentiellement une cible d'écriture réservée à Microsoft ; le MS Project XML est le propre format d'échange de Microsoft et fait l'aller-retour proprement. Vous pouvez donc ouvrir un .MPP dans Ingantt, faire vos modifications, exporter en MS Project XML, et remettre le fichier .xml à un collègue qui utilise encore Microsoft Project — il l'ouvre comme n'importe quel fichier de projet. Pour un partage en lecture seule avec des personnes qui n'utilisent pas Project du tout, le PDF ou le PNG est le bon export.

Vais-je perdre les dépendances, les ressources ou le chemin critique en ouvrant un .MPP en dehors de Microsoft Project ?

Dans Ingantt, non — l'import est sans perte pour les champs qui comptent. Tâches, sous-tâches, dépendances finish-to-start, affectations de ressources, travail et durée, jalons, et le calcul du chemin critique arrivent tous depuis le fichier .MPP d'origine. Ingantt utilise la même structure de projet que Microsoft Project en interne, donc un diagramme de Gantt qui a demandé un après-midi à construire dans Project ressemble toujours au même diagramme de Gantt après son ouverture dans Ingantt. Les visionneuses freemium qui ne montrent qu'une liste de tâches ou aplatissent le projet en image perdent le graphe de dépendances, qui est généralement la partie la plus importante du plan.

Existe-t-il un lecteur .MPP gratuit ou une visionneuse de diagramme de Gantt gratuite ?

Ingantt propose un essai gratuit de 7 jours sans carte bancaire requise pour démarrer, et l'essai inclut l'import .MPP complet, la visualisation, l'édition, l'impression et l'export PDF/PNG/XML — rien n'est verrouillé derrière un découpage « édition visionneuse » contre « édition éditeur ». Pour la plupart des personnes qui n'ont besoin d'ouvrir ou de visualiser un fichier .MPP qu'une fois ou quelques fois, l'essai gratuit couvre entièrement la situation. Après l'essai, les plans payants commencent à 5,99 $ par utilisateur et par mois. Si vous travaillez avec des prestataires ou des clients qui utilisent encore Microsoft Project, le plan payant coûte moins qu'un seul mois de Microsoft 365 Project.

Faut-il connaître la gestion de projet pour utiliser une visionneuse Gantt ?

Non. Ingantt est conçu pour qu'une partie prenante non chef de projet — un dirigeant vérifiant une date de livraison, un client contrôlant un jalon, un nouveau membre d'équipe qui s'oriente — puisse ouvrir un diagramme de Gantt et le lire sans rien apprendre de nouveau. Vous faites défiler la chronologie pour voir quand les choses se passent, survolez une tâche pour voir qui s'en occupe, et développez une phase pour voir les tâches qu'elle contient. Aucun jargon à décoder, aucune étape de configuration au-delà de l'ouverture du fichier.
