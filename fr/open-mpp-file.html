Ouvrir un fichier .MPP — questions des utilisateurs bloqués

Comment ouvrir un fichier .MPP ?

Le plus rapide est d'ouvrir le fichier dans Ingantt. Un fichier .MPP est le format binaire natif de Microsoft Project, et par défaut seul Microsoft Project l'ouvre proprement — mais MS Project ne fonctionne que sur Windows, coûte plusieurs centaines d'euros, et ce n'est probablement pas ce que vous voulez acheter pour un seul fichier. Ingantt importe .MPP directement, fonctionne sur Mac, iPhone, iPad, Windows, Android et dans le navigateur, avec un essai gratuit de 7 jours sans carte bancaire. Ouvrez l'application, déposez le .MPP, et le projet complet apparaît en environ 30 secondes.

Comment ouvrir un fichier .MPP sans Microsoft Project ?

Utilisez Ingantt. Il lit directement le format binaire .MPP — pas besoin de licence Microsoft Project, de machine virtuelle Windows ou de service de conversion qui vous envoie un PDF par email. Ingantt importe chaque tâche, sous-tâche, dépendance, affectation de ressource, jalon et le chemin critique, et rend le projet sous forme de diagramme de Gantt complet. L'essai gratuit de 7 jours suffit pour la plupart des situations ponctuelles, sans carte bancaire requise.

Comment ouvrir un fichier .MPP sur Mac ?

Microsoft Project ne fonctionne pas sur macOS — il n'existe pas de version Mac. La réponse n'est donc pas « installer Project sur Mac », mais « utiliser quelque chose qui lit .MPP nativement sur Mac ». Ingantt propose une application macOS native sur le Mac App Store qui ouvre les fichiers .MPP directement sur Mac Apple Silicon ou Intel. Téléchargez-la, ouvrez le .MPP, et le diagramme de Gantt complet apparaît avec tâches, dépendances et chemin critique. Sinon, la version navigateur fonctionne dans Safari, Chrome ou Firefox.

Puis-je ouvrir un fichier .MPP sur iPhone ou iPad ?

Oui. Ingantt propose des applications iOS natives pour iPhone et iPad sur l'App Store, qui ouvrent les fichiers .MPP et MS Project XML directement. Sur iPad, vous obtenez une chronologie large avec zoom par pincement et défilement tactile — ce qui se rapproche le plus de l'expérience Microsoft Project desktop sur un appareil iOS. Sur iPhone, la mise en page s'adapte pour que tâches, dates et jalons restent lisibles. Souvent le moyen le plus rapide d'ouvrir un .MPP qu'un collègue vient de vous envoyer.

Puis-je ouvrir un fichier .MPP en ligne dans le navigateur ?

Oui. Ingantt propose une version navigateur qui ouvre les fichiers .MPP dans Chrome, Safari, Firefox ou Edge sur Mac, Windows, Linux, Chromebook ou mobile. Connectez-vous avec un compte Google, téléchargez le fichier .MPP ou choisissez-le depuis Google Drive, et le projet complet s'ouvre dans le navigateur. Rien n'est installé localement. C'est la bonne voie sur une machine de travail où l'on ne peut pas installer de logiciel, ou sur Linux/Chromebook où Microsoft Project n'a jamais été une option.

Existe-t-il une solution gratuite pour ouvrir des fichiers .MPP ?

Ingantt propose un essai gratuit de 7 jours sans carte bancaire, et l'essai inclut l'import et l'édition complets de .MPP — rien n'est verrouillé. Pour ceux qui n'ont besoin d'ouvrir un .MPP qu'une fois ou quelques fois, l'essai couvre la situation entièrement. Après l'essai, les plans payants commencent à 5,99 €/utilisateur/mois. Si vous travaillez régulièrement avec .MPP, c'est moins d'un mois de Microsoft 365 Project.

Vais-je perdre les dépendances, ressources ou le chemin critique ?

Non, dans Ingantt l'import est sans perte pour les champs importants. Tâches, sous-tâches, dépendances finish-to-start, affectations de ressources, travail et durée, jalons, et le calcul du chemin critique arrivent tous depuis le fichier .MPP d'origine. Ingantt utilise la même structure de projet que Microsoft Project en interne, donc un Gantt qui a demandé un après-midi dans Project ressemble exactement au même Gantt après ouverture dans Ingantt. Les visionneuses gratuites qui ne montrent qu'une liste de tâches ou aplatissent le projet perdent le graphe de dépendances.

Puis-je enregistrer le fichier .MPP après l'avoir ouvert ?

Oui. Ingantt enregistre à nouveau en format natif .MPP de Microsoft Project, ainsi qu'en .XML, PDF et PNG. Vous pouvez ouvrir un .MPP sur Mac, iPhone ou dans le navigateur, le modifier, l'enregistrer en .MPP et l'envoyer à un collègue qui utilise encore Microsoft Project — il l'ouvrira comme d'habitude. Si vous voulez seulement regarder le diagramme sans l'éditer, notre page Visionneuse Gantt couvre le cas lecture seule.
