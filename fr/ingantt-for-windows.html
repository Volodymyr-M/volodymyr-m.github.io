07Foire aux questions

Qu’est-ce qu’Ingantt pour Windows ?

Ingantt pour Windows est un outil professionnel de gestion de projet qui s’exécute nativement sur votre PC. Il vous permet de créer et de gérer des plans de projet avec une planification assistée par IA et une compatibilité MS Project.

Puis-je importer mes fichiers Microsoft Project existants ?

Oui ! Ingantt prend entièrement en charge l’importation de fichiers .MPP et .XML de Microsoft Project. Vos données de projet, y compris les tâches, ressources et dépendances, seront préservées lors de l’import.

Comment fonctionne la génération de projet avec IA ?

Notre assistant IA peut générer des plans de projet complets à partir de descriptions textuelles simples. Décrivez simplement vos objectifs et besoins, et l’IA créera un plan structuré avec tâches, calendriers et dépendances basés sur les meilleures pratiques.

Existe-t-il une version d’essai gratuite pour la version Windows ?

Oui. Une fois l’application installée depuis le Microsoft Store, vous disposez d’un essai gratuit de 7 jours.

Comment puis-je acheter Ingantt pour Windows ?

Rendez-vous sur le Microsoft Store et achetez-le là-bas. Vous posséderez l’application sans frais récurrents.
