Politique de confidentialité

Date d'entrée en vigueur: 22 février 2026 Dernière mise à jour: 22 février 2026

Ingantt Development (« nous », « notre » ou « nos ») exploite l'application Ingantt (l'« Application ») et le site Web https://ingantt.com (le « Site Web »). La présente Politique de confidentialité décrit la manière dont nous collectons, utilisons et partageons les informations lorsque vous utilisez l'Application sur toutes les plateformes prises en charge (iOS, Android, macOS, Windows et Web, y compris via Google Workspace Marketplace).

1. Informations que nous collectons

1.1 Informations que vous fournissez

  • Informations du compte Google : Si vous vous connectez avec Google, nous recevons de Google votre adresse e-mail, votre nom d'affichage et l'URL de votre photo de profil. Sur les plateformes natives (iOS, Android, macOS, Windows), la connexion est facultative et permet le stockage de fichiers sur Google Drive. Sur la version Web, Google Sign-In est obligatoire et est utilisé pour le stockage de fichiers sur Google Drive et la gestion des abonnements. Lors de votre première utilisation de la version Web, un enregistrement utilisateur est créé sur notre serveur contenant votre adresse e-mail, votre nom, votre langue et votre date d'inscription.

  • Informations d'abonnement et de facturation (Web uniquement) : Lors de l'achat d'un abonnement via le Web, nous collectons votre nom, votre adresse e-mail, votre adresse postale, votre numéro de téléphone et les détails relatifs au paiement. Ces informations sont transmises à WayForPay, notre prestataire de paiement, pour le traitement des transactions.

  • Retours : Lorsque vous soumettez des retours via l'Application, nous collectons le texte de votre message. Vous pouvez éventuellement choisir de joindre votre fichier de projet actuellement ouvert pour nous aider à diagnostiquer les problèmes.

  • Saisie pour la fonctionnalité IA : Lorsque vous utilisez la fonctionnalité « Create with AI », le prompt textuel que vous fournissez est envoyé à notre serveur backend pour traitement par Anthropic (Claude).

  • Fichiers de projet : Les fichiers de projet que vous créez et modifiez restent sous votre contrôle. Les fichiers sont stockés localement sur votre appareil ou dans votre compte Google Drive (dans un dossier appelé « Ingantt Schedules »). Lorsque vous ouvrez un fichier dans un format non natif (tel que MPP, GAN, XER ou d'autres formats de fichiers de projet pris en charge), le fichier est transmis à notre serveur pour conversion de format. Les fichiers sont traités uniquement en mémoire et ne sont pas stockés sur notre serveur.

1.2 Informations collectées automatiquement

  • Métadonnées de l'application : Chaque requête envoyée à notre backend inclut la version de votre application, le nom de la plateforme (iOS, Android, macOS, Windows ou Web), la langue du système et, si vous êtes connecté, votre adresse e-mail.

  • Rapports de plantage et d'erreurs : Nous utilisons Sentry (hébergé dans l'UE) pour collecter les rapports de plantage et les journaux d'erreurs en production. Cela inclut les traces de pile, les messages d'erreur, la version de l'application et les informations sur la plateforme. Nous ne collectons pas intentionnellement de données personnelles via les rapports de plantage.

  • Analyse Web (Web uniquement) : La version Web de l'Application utilise Google Analytics 4 pour collecter des analyses d'utilisation standard, y compris les pages consultées et les données de session. Google Analytics peut déposer des cookies sur votre navigateur. Les versions natives mobiles et de bureau de l'Application n'incluent pas de suivi analytique.

  • Préférences utilisateur : L'Application stocke vos préférences locales (telles que le thème, les options d'affichage, les largeurs de colonnes et la liste des fichiers récents) sur votre appareil en utilisant les mécanismes de stockage standard de la plateforme. Ces préférences ne sont pas transmises à nos serveurs.

1.3 Informations que nous ne collectons pas

  • Nous n'utilisons pas vos données pour entraîner des modèles d'intelligence artificielle ou d'apprentissage automatique.
  • Nous n'affichons pas de publicités et n'intégrons aucun SDK publicitaire.
  • Nous ne collectons pas de données de localisation, de contacts, de photos, de données de caméra ou d'entrée microphone.
  • Nous n'utilisons pas d'identifiants d'appareil à des fins de suivi.
  • La sauvegarde cloud Android est explicitement désactivée pour l'Application.

2. Comment nous utilisons vos informations

Nous utilisons les informations que nous collectons aux fins suivantes :

  • Fournir et exploiter l'Application : Traiter vos demandes, permettre le stockage et la synchronisation de fichiers via Google Drive, convertir les formats de fichiers et fournir la création de projets assistée par l'IA.
  • Traiter les abonnements (Web uniquement) : Traiter les paiements d'abonnement Web, valider le statut de l'abonnement et gérer votre compte.
  • Envoyer des e-mails transactionnels (Web uniquement) : Vous envoyer des notifications relatives à votre compte telles que les messages de bienvenue, les rappels d'essai, les confirmations d'abonnement et les avis d'annulation.
  • Améliorer la stabilité et les performances : Analyser les rapports de plantage, les données d'erreur et la télémétrie côté serveur pour identifier et corriger les bogues.
  • Comprendre les schémas d'utilisation (Web uniquement) : Utiliser les données d'analyse agrégées pour comprendre comment l'application Web est utilisée et améliorer l'expérience utilisateur.
  • Répondre aux demandes d'assistance : Utiliser les retours soumis pour diagnostiquer les problèmes et améliorer l'Application.

3. Comment nous partageons vos informations

Nous ne vendons pas vos informations personnelles. Nous partageons les informations uniquement dans les circonstances suivantes :

  • Prestataires de services : Nous utilisons les prestataires de services tiers suivants pour exploiter l'Application :

    • Sentry (Functional Software, Inc.) — signalement des plantages et des erreurs côté client, données traitées dans l'UE
    • Microsoft Application Insights (Microsoft Corporation) — télémétrie côté serveur et surveillance des erreurs
    • Google Analytics (Google LLC) — analyse d'utilisation Web (version Web uniquement)
    • Google APIs (Google LLC) — authentification et stockage de fichiers sur Google Drive
    • WayForPay — traitement des paiements d'abonnement (version Web uniquement)
    • Resend (Resend, Inc.) — envoi d'e-mails transactionnels (version Web uniquement)
    • Anthropic (Claude) — traitement des prompts de création de projets assistée par l'IA
    • Microsoft Azure — hébergement du backend et stockage des données (région East US)

  • Obligations légales : Nous pouvons divulguer des informations si la loi l'exige ou en réponse à une procédure juridique valide.

4. Stockage et sécurité des données

  • Données utilisateur côté serveur (Web uniquement) : Votre adresse e-mail, votre nom, votre langue, votre date d'inscription et votre statut d'abonnement sont stockés dans Azure Table Storage sur notre serveur backend.
  • Les jetons OAuth (jetons d'accès, de rafraîchissement et d'identification Google) sont stockés dans le stockage sécurisé de la plateforme (iOS Keychain, Android Keystore, macOS Keychain ou équivalent de la plateforme).
  • Le statut d'abonnement (Web uniquement) est mis en cache localement dans un stockage sécurisé pour l'accès hors ligne.
  • Les requêtes API vers notre backend sont signées avec HMAC-MD5 pour la vérification de l'intégrité.
  • L'Application fonctionne dans les environnements sandbox de la plateforme lorsqu'ils sont disponibles (macOS App Sandbox, iOS App Sandbox).
  • Nous utilisons HTTPS pour toutes les communications réseau.

4a. Conservation des données

  • Enregistrements utilisateur côté serveur (Web uniquement) : Votre enregistrement utilisateur et votre historique d'abonnement sont conservés aussi longtemps que votre compte existe. Pour demander la suppression de vos données, contactez-nous à support@ingantt.com.
  • Fichiers de projet : Les fichiers soumis pour conversion de format sont traités en mémoire et ne sont pas conservés.
  • Prompts IA : Les prompts textuels envoyés pour la création de projets assistée par l'IA ne sont pas stockés sur notre serveur.
  • Retours : Les messages de retour sont transmis à notre équipe d'assistance par e-mail et ne sont pas stockés dans une base de données.
  • Données locales : Les préférences, les jetons OAuth et le statut d'abonnement mis en cache restent sur votre appareil jusqu'à ce que vous vous déconnectiez ou désinstalliez l'Application.

5. Vos choix et droits

  • La connexion est facultative sur les plateformes natives : Sur iOS, Android, macOS et Windows, vous pouvez utiliser l'Application sans vous connecter à Google. La connexion permet le stockage de fichiers sur Google Drive. Sur la version Web, Google Sign-In est requis pour le fonctionnement de l'application, le stockage de fichiers et la gestion des abonnements.
  • Accès à Google Drive : L'Application utilise le périmètre limité drive.file, qui ne permet l'accès qu'aux fichiers créés par l'Application ou explicitement ouverts par vous dans l'Application — et non à l'ensemble de votre Google Drive.
  • Pièces jointes de retour : Joindre votre fichier de projet lors de la soumission de retours est facultatif et nécessite votre consentement explicite.
  • Analyse Web : Vous pouvez utiliser les outils de blocage des cookies de votre navigateur ou les outils de désactivation pour limiter la collecte de données de Google Analytics sur la version Web.
  • E-mails transactionnels : Sur la version Web, vous recevrez des e-mails relatifs à votre compte (bienvenue, rappels d'essai, notifications d'abonnement). Il s'agit de communications de service nécessaires à la gestion de votre abonnement.
  • Suppression des données : Vous pouvez vous déconnecter de votre compte Google dans l'Application à tout moment, ce qui supprime vos jetons OAuth stockés de l'appareil. Pour demander la suppression de votre enregistrement utilisateur et de toute donnée associée sur nos serveurs, contactez-nous à support@ingantt.com.

6. Confidentialité des enfants

L'Application n'est pas destinée aux enfants de moins de 13 ans (ou l'âge applicable dans votre juridiction). Nous ne collectons pas sciemment d'informations personnelles auprès d'enfants. Si vous pensez qu'un enfant nous a fourni des informations personnelles, veuillez nous contacter à support@ingantt.com afin que nous puissions prendre les mesures appropriées.

7. Transferts internationaux de données

Notre serveur backend est hébergé sur Microsoft Azure dans la région East US. Les rapports de plantage sont traités par Sentry dans l'UE. Si vous vous trouvez en dehors des États-Unis, vos informations peuvent être transférées et traitées aux États-Unis ou dans d'autres juridictions où nos prestataires de services opèrent.

8. Modifications de la présente Politique de confidentialité

Nous pouvons mettre à jour la présente Politique de confidentialité de temps à autre. Nous vous informerons de tout changement important en mettant à jour la date de « Dernière mise à jour » en haut de cette politique. Votre utilisation continue de l'Application après toute modification constitue votre acceptation de la politique mise à jour.

9. Informations spécifiques aux plateformes

iOS et macOS (Apple)

  • L'Application est une application payante distribuée via l'Apple App Store. Les pratiques de confidentialité d'Apple s'appliquent à la transaction d'achat.
  • L'Application ne demande que les autorisations d'accès aux fichiers que vous accordez explicitement via le sélecteur de fichiers du système.

Android

  • L'Application est une application payante distribuée via Google Play. Les pratiques de confidentialité de Google s'appliquent à la transaction d'achat.
  • L'Application demande les autorisations de lecture/écriture du stockage pour les opérations de fichiers locaux et l'autorisation Internet pour l'accès réseau.

Windows

  • L'Application est une application payante distribuée via le Microsoft Store. Les pratiques de confidentialité de Microsoft s'appliquent à la transaction d'achat.

Web

  • La version Web utilise un modèle d'abonnement. Les paiements d'abonnement sont traités par WayForPay.
  • L'Application est également disponible via Google Workspace Marketplace.
  • Les cookies Google Analytics sont déposés sur le domaine web.ingantt.com.
  • L'authentification Google OAuth utilise une entrée temporaire localStorage pour le callback de connexion, qui est immédiatement effacée après utilisation.
