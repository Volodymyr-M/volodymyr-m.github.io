Visionneuse Gantt — Questions fréquentes

Qu'est-ce qu'une visionneuse Gantt ?

Une visionneuse Gantt est un outil qui ouvre un fichier de diagramme de Gantt — généralement un fichier Microsoft Project .MPP ou .XML — et affiche le projet sous forme de chronologie interactive. Une vraie visionneuse permet de faire défiler la chronologie, de zoomer, de déplier les phases et de survoler les tâches pour voir dépendances, dates de début et de fin, et ressources. Ingantt est une visionneuse Gantt qui fonctionne sur Mac, iOS, iPad, Windows, Android et dans le navigateur, pour ouvrir le même fichier sur n'importe quel appareil.

Comment visualiser un diagramme de Gantt sans Microsoft Project ?

Ouvrez le fichier dans Ingantt. Il importe directement le format .MPP natif de Microsoft Project et MS Project XML, donc un fichier reçu d'un collègue s'ouvre comme un diagramme de Gantt complètement interactif — sans licence MS Project ni installation Windows. Utilisez la version navigateur pour un besoin ponctuel ou installez l'essai gratuit de l'application Mac, iOS, Windows ou Android. Les tâches, les dépendances et le chemin critique arrivent intacts.

Puis-je visualiser un fichier .MPP sur Mac ?

Oui. Ingantt propose une application macOS native qui ouvre directement les fichiers .MPP et MS Project XML sur tout Mac récent. Pas besoin de machine virtuelle, de Parallels ou d'installation Windows. Téléchargez Ingantt depuis le Mac App Store, ouvrez le fichier .MPP et vous voyez le diagramme de Gantt complet avec phases, tâches, dépendances et chemin critique. Si vous ne voulez rien installer, la version navigateur fonctionne dans Safari, Chrome ou Firefox.

Puis-je visualiser un diagramme de Gantt sur iPhone ou iPad ?

Oui. Ingantt propose des applications iOS natives pour iPhone et iPad, qui ouvrent les fichiers .MPP et MS Project XML comme la version Mac. Sur iPad, vous obtenez une chronologie Gantt pleine largeur avec zoom par pincement et défilement tactile ; sur iPhone, la mise en page s'adapte pour que tâches et jalons restent lisibles. C'est le moyen le plus rapide de regarder un Gantt qu'un collègue vient de vous envoyer par email.

Existe-t-il une visionneuse Gantt gratuite ?

Ingantt propose un essai gratuit de 7 jours sans carte bancaire. L'essai inclut la visualisation et l'édition complètes des fichiers .MPP et MS Project XML sur toutes les plateformes — rien n'est verrouillé derrière une « édition visionneuse » vs « édition éditeur ». Après l'essai, les plans payants commencent à 5,99 €/utilisateur/mois. La plupart des besoins de visualisation ponctuelle rentrent dans l'essai gratuit.

Puis-je visualiser un Gantt dans le navigateur sans rien installer ?

Oui. Ingantt propose une version navigateur qui ouvre les fichiers Gantt dans Chrome, Safari, Firefox ou Edge sur Mac, Windows, Linux, Chromebook ou mobile. Connectez-vous avec un compte Google, ouvrez un fichier .MPP depuis Google Drive ou téléchargez-en un, et le diagramme se charge avec zoom, défilement et détails au survol. Rien n'est installé, rien n'est téléchargé.

Faut-il connaître la gestion de projet ?

Non. Ingantt est conçu pour qu'un non-spécialiste — un dirigeant vérifiant une date de livraison, un client contrôlant un jalon, un nouvel arrivant s'orientant — puisse ouvrir un Gantt et le lire sans rien apprendre. Faites défiler la chronologie pour voir quand les choses arrivent, survolez une tâche pour voir qui la fait, dépliez une phase pour voir son contenu. Pas de jargon, pas d'étape de configuration.

Puis-je imprimer ou partager le Gantt que je visualise ?

Oui. Ingantt exporte le diagramme en PDF ou PNG pour l'imprimer, l'attacher à un email ou l'insérer dans une présentation. L'export PDF préserve la mise en page à n'importe quel niveau de zoom, ce qui est utile pour remettre une version en lecture seule à un intervenant. Si vous devez rouvrir le même .MPP ailleurs plus tard, voyez notre guide pour ouvrir des fichiers .MPP sans Microsoft Project.
