Preguntas frecuentes

¿Qué es Ingantt para macOS?

Ingantt para macOS es una herramienta profesional de gestión de proyectos que funciona de manera nativa en tu Mac. Te permite crear y gestionar planes de proyecto con planificación asistida por IA y compatibilidad con MS Project.

¿Puedo importar mis archivos existentes de Microsoft Project?

¡Sí! Ingantt admite completamente la importación de archivos .MPP y .XML de Microsoft Project. Tus datos de proyecto, incluidas tareas, recursos y dependencias, se conservarán durante el proceso de importación.

¿Cómo funciona la generación de proyectos con IA?

Nuestro asistente de IA puede generar planes de proyecto completos a partir de descripciones de texto simples. Solo describe tus objetivos y requisitos, y la IA creará un plan estructurado con tareas, cronogramas y dependencias basadas en las mejores prácticas.

¿Hay una versión de prueba gratuita para la versión de macOS?

No. La versión de macOS se compra una sola vez, sin suscripción ni prueba gratuita.

¿Cómo puedo comprar Ingantt para macOS?

Visita la Mac App Store y cómprala allí. Serás dueño de la aplicación de forma perpetua sin cargos recurrentes.
