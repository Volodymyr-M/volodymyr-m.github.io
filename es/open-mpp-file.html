04Abrir y visualizar archivos .MPP — Preguntas que la Gente Realmente Hace

¿Cómo abro un archivo .MPP sin Microsoft Project?

Usa Ingantt en tu navegador. Lee el formato binario .MPP directamente — sin necesidad de licencia de Microsoft Project, sin máquina virtual Windows, sin servicio de conversión que te mande un PDF por correo. Ingantt importa cada tarea, subtarea, dependencia, asignación de recursos, hito y la ruta crítica, y renderiza el proyecto como un diagrama de Gantt interactivo completo. La prueba gratuita de 7 días cubre la mayoría de situaciones puntuales de abrir un archivo, y no se requiere tarjeta de crédito para empezar.

¿Qué es un visor de Gantt, y lo es Ingantt?

Un visor de Gantt es una herramienta que abre un archivo de diagrama de Gantt — normalmente un .MPP o .XML de Microsoft Project — y muestra el proyecto como una línea de tiempo interactiva. Un visor de verdad te permite desplazarte por el cronograma, ampliar y reducir, expandir o colapsar fases, y pasar el cursor sobre tareas para ver dependencias, fechas de inicio y fin, y recursos. Ingantt es un visor y editor de Gantt basado en navegador, así que puedes abrir el mismo archivo en el dispositivo que tengas a mano — y exportar los cambios de vuelta a MS Project XML cuando lo necesites.

¿Puedo abrir un archivo .MPP o ver un diagrama de Gantt en el navegador?

Sí. Ingantt se ejecuta íntegramente en el navegador — Chrome, Safari, Firefox o Edge — sobre Mac, Windows, Linux, Chromebook, iPhone, iPad o Android. Inicia sesión con una Google Account, sube el archivo .MPP o selecciónalo desde Google Drive, y el proyecto completo se abre en el navegador con zoom, desplazamiento y detalles al pasar el cursor. No se instala nada localmente y no se descarga nada. Es el camino correcto si estás en un equipo de trabajo donde no puedes instalar software nuevo, o en Linux o Chromebook donde Microsoft Project nunca fue una opción.

¿Puedo imprimir mi diagrama de Gantt, o exportarlo a PDF o PNG?

Sí, las tres opciones. Ingantt imprime el diagrama de Gantt directamente desde el navegador, y lo exporta a PDF y PNG a cualquier nivel de zoom — así que, tanto si necesitas una vista general de una sola página como una línea de tiempo del tamaño de una pared en varias páginas, el diseño se mantiene limpio. El PDF es la elección habitual para compartir en modo solo lectura con partes interesadas que no necesitan tocar el archivo original; el PNG es útil para insertar en presentaciones, documentos de Notion o mensajes de Slack; la impresión nativa es lo más rápido cuando solo quieres una copia en papel.

¿Puedo guardar el archivo .MPP de vuelta tras abrirlo en Ingantt?

No como .MPP — Ingantt exporta a MS Project XML, que Microsoft Project abre directamente. El formato binario .MPP es propietario, no está documentado y, en esencia, solo Microsoft puede escribir en él; MS Project XML es el formato de intercambio del propio Microsoft y hace el recorrido de ida y vuelta sin problemas. Así que puedes abrir un .MPP en Ingantt, hacer ediciones, exportar como MS Project XML y entregarle el archivo .xml a un compañero que siga usando Microsoft Project — lo abrirá como abre cualquier archivo de proyecto. Para compartir en modo solo lectura con gente que no usa Project en absoluto, PDF o PNG es la exportación adecuada.

¿Perderé dependencias, recursos o la ruta crítica al abrir un .MPP fuera de Microsoft Project?

En Ingantt, no — la importación es sin pérdidas en los campos que importan. Tareas, subtareas, dependencias finish-to-start, asignaciones de recursos, trabajo y duración, hitos y el cálculo de la ruta crítica llegan todos desde el archivo .MPP original. Ingantt usa la misma estructura de proyecto que Microsoft Project usa internamente, así que un diagrama de Gantt que costó una tarde construir en Project sigue pareciendo el mismo diagrama de Gantt después de abrirlo en Ingantt. Los visores gratuitos que solo muestran listas de tareas o aplanan el proyecto como imagen pierden el grafo de dependencias, que suele ser la parte más importante del plan.

¿Hay un abridor de .MPP o visor de Gantt gratuito?

Ingantt tiene una prueba gratuita de 7 días sin tarjeta de crédito para empezar, y la prueba incluye importación completa de .MPP, visualización, edición, impresión y exportación a PDF/PNG/XML — nada está bloqueado detrás de una división "edición visor" frente a "edición editor". Para la mayoría de personas que solo necesitan abrir o ver un archivo .MPP una o un par de veces, la prueba gratuita cubre la situación por completo. Después de la prueba, los planes de pago empiezan en 5,99 $ por usuario al mes. Si trabajas con proveedores o clientes que siguen usando Microsoft Project, el plan de pago es menos que el coste de un solo mes de Microsoft 365 Project.

¿Necesito saber de gestión de proyectos para usar un visor de Gantt?

No. Ingantt está diseñado para que una parte interesada no técnica — un directivo revisando una fecha de entrega, un cliente comprobando un hito, un miembro nuevo del equipo poniéndose al día — pueda abrir un diagrama de Gantt y leerlo sin aprender nada nuevo. Te desplazas por la línea de tiempo para ver cuándo ocurren las cosas, pasas el cursor sobre una tarea para ver quién la lleva, y expandes una fase para ver las tareas que contiene. No hay jerga que descifrar ni un paso de configuración más allá de abrir el archivo.
