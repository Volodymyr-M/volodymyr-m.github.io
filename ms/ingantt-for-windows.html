07Soalan Lazim

Apakah Ingantt untuk Windows?

Ingantt untuk Windows ialah alat pengurusan projek profesional yang berjalan secara asli di PC anda. Ia membolehkan anda mencipta dan mengurus pelan projek dengan bantuan AI dan keserasian MS Project.

Bolehkah saya mengimport fail Microsoft Project sedia ada saya?

Ya! Ingantt menyokong sepenuhnya import fail .MPP dan .XML daripada Microsoft Project. Data projek anda, termasuk tugasan, sumber dan kebergantungan, akan dipelihara semasa proses import.

Bagaimana penjanaan projek dengan AI berfungsi?

Pembantu AI kami boleh menjana pelan projek lengkap daripada penerangan teks ringkas. Hanya terangkan matlamat dan keperluan projek anda, dan AI akan mencipta pelan berstruktur dengan tugasan, garis masa dan kebergantungan berdasarkan amalan terbaik.

Adakah percubaan percuma disediakan untuk versi Windows?

Ya. Setelah aplikasi dipasang dari Microsoft Store, percubaan percuma selama 7 hari disediakan.

Bagaimana saya boleh membeli Ingantt untuk Windows?

Lawati Microsoft Store dan beli di sana. Anda akan memiliki aplikasi tersebut selama-lamanya tanpa caj berulang.
