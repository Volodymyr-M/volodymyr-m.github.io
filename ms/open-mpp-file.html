Bagaimana cara membuka fail .MPP tanpa Microsoft Project?

Gunakan Ingantt. Ia membaca format binari .MPP secara terus — tanpa lesen Microsoft Project, tanpa mesin maya Windows, tanpa perkhidmatan penukaran yang menghantar PDF melalui e-mel. Ingantt mengimport setiap tugasan, sub-tugasan, kebergantungan, tugasan sumber, pencapaian dan laluan kritikal, dan memaparkan projek sebagai carta Gantt lengkap. Percubaan percuma 7 hari meliputi kebanyakan situasi sekali sahaja.

Bagaimana cara membuka fail .MPP pada Mac?

Microsoft Project tidak berjalan pada macOS — tiada versi Mac. Jadi jawapannya bukan "pasang Project pada Mac" tetapi "gunakan sesuatu yang membaca .MPP secara asli pada Mac". Ingantt mempunyai aplikasi macOS asli di Mac App Store yang membuka fail .MPP secara terus pada Mac Apple Silicon dan Intel. Muat turun, buka .MPP, dan anda akan melihat carta Gantt penuh dengan tugasan, kebergantungan dan laluan kritikal.

Bolehkah saya menyimpan fail .MPP selepas membukanya?

Ya. Ingantt menyimpan projek semula dalam format .MPP asli Microsoft Project, serta .XML, PDF dan PNG. Anda boleh membuka .MPP pada Mac, iPhone atau dalam pelayar, membuat suntingan, menyimpannya semula sebagai .MPP dan menyerahkan fail kepada rakan sekerja yang masih menggunakan Microsoft Project. Jika anda hanya mahu melihat carta tanpa mengedit, halaman Pemapar Gantt kami meliputi kes baca sahaja.
