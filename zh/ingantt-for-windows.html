常见问题

什么是适用于 Windows 的 Ingantt？

Ingantt for Windows 是一款在 PC 上本地运行的专业项目管理工具。它结合了 AI 辅助的规划功能和对 MS Project 的兼容性，让您轻松创建并管理项目计划。

我能导入已有的 Microsoft Project 文件吗？

可以！Ingantt 完全支持从 Microsoft Project 导入 .MPP 和 .XML 文件。您的项目数据（包括任务、资源和依赖关系）都会在导入过程中得到保留。

AI 项目生成是如何工作的？

我们的 AI 助手能够根据简短的文本描述生成完整的项目计划。只需描述您的项目目标和需求，AI 将基于最佳实践创建包含任务、进度和依赖关系的结构化计划。

Windows 版本有免费试用吗？

有的。从 Microsoft Store 安装应用后，可享受 7 天免费试用。

我如何购买适用于 Windows 的 Ingantt？

前往 Microsoft Store 并在那里购买。购买后您可永久使用该应用，无需重复付费。
