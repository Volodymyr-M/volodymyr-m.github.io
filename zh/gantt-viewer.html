04甘特图查看器 — 常见问题

什么是甘特图查看器？

甘特图查看器是一种打开甘特图文件（通常是 Microsoft Project .MPP 或 .XML 文件）并将项目显示为交互式时间线的工具。真正的查看器允许您滚动时间表、缩放、展开或折叠阶段，以及悬停在任务上以查看依赖关系、开始和结束日期以及资源。Ingantt 是一款在 Mac、iOS、iPad、Windows、Android 和浏览器上运行的甘特图查看器。

如何在没有 Microsoft Project 的情况下查看甘特图？

在 Ingantt 中打开文件。它直接导入 Microsoft Project 的原生 .MPP 格式和 MS Project XML，因此来自使用 MS Project 的同事的文件可以作为完全交互式的甘特图打开 — 无需 MS Project 许可证或 Windows 安装。使用浏览器版本或安装 Mac、iOS、Windows 或 Android 应用的免费试用。任务、依赖关系和关键路径完整保留。

可以。Ingantt 将甘特图导出为 PDF 或 PNG，以便打印、附加到电子邮件或粘贴到幻灯片中。PDF 导出在任何缩放级别都保留时间线布局。如果您稍后需要在其他地方打开同一 .MPP 文件，请参阅我们的在没有 Microsoft Project 的情况下打开 .MPP 文件指南。
