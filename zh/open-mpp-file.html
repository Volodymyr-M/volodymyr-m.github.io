打开 .MPP 文件 — 被困住时的常见问题

如何打开 .MPP 文件？

最快的方法是在 Ingantt 中打开文件。.MPP 文件是 Microsoft Project 的原生二进制格式，默认情况下只有 Microsoft Project 可以正常打开 — 但 MS Project 只能在 Windows 上运行，价格昂贵，为一个文件购买并不划算。Ingantt 直接导入 .MPP，可在 Mac、iPhone、iPad、Windows、Android 和浏览器上运行，提供 7 天免费试用，无需信用卡。打开应用，拖入 .MPP，完整项目约 30 秒内显示。

如何在 Mac 上打开 .MPP 文件？

Microsoft Project 不能在 macOS 上运行 — 它没有 Mac 版本。所以答案不是"在 Mac 上安装 Project"，而是"使用能在 Mac 上原生读取 .MPP 的工具"。Ingantt 在 Mac App Store 上有原生 macOS 应用，可直接在 Apple Silicon 和 Intel Mac 上打开 .MPP 文件。下载它，打开 .MPP，您就能看到完整的甘特图，包含任务、依赖关系和关键路径。或者，浏览器版本在 Safari、Chrome 或 Firefox 中也能同样工作。

我可以在 iPhone 或 iPad 上打开 .MPP 文件吗？

可以。Ingantt 在 App Store 上有为 iPhone 和 iPad 设计的原生 iOS 应用，可直接打开 .MPP 和 MS Project XML 文件。在 iPad 上，您可以获得带有捏合缩放和触摸滚动的宽时间线 — 这是在 iOS 设备上最接近 Microsoft Project 桌面体验的方式。

打开 .MPP 后我可以再次保存吗？

可以。Ingantt 将项目保存回 Microsoft Project 的原生 .MPP 格式，以及 .XML、PDF 和 PNG。您可以在 Mac、iPhone 或浏览器中打开 .MPP，进行编辑，将其保存为 .MPP，并将文件交给仍在使用 Microsoft Project 的同事 — 他们会像往常一样打开它。如果您只想查看图表而不进行编辑，请参阅我们的甘特图查看器页面。
