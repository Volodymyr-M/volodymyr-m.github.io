04打开并查看 .MPP 文件 — 大家真正会问的问题

没有 Microsoft Project 怎么打开 .MPP 文件？

在浏览器里用 Ingantt 打开。它直接读取 .MPP 二进制格式 — 不需要 Microsoft Project 许可证、不需要 Windows 虚拟机，也不需要给你邮发 PDF 的转换服务。Ingantt 会导入每个任务、子任务、依赖关系、资源分配、里程碑和关键路径，并把项目渲染成一张完整的交互式甘特图。对于绝大多数只是偶尔要打开文件的场景，7 天免费试用已经足够，开始使用也无需信用卡。

什么是甘特图查看器，Ingantt 算吗？

甘特图查看器是一种可以打开甘特图文件 — 通常是 Microsoft Project 的 .MPP 或 .XML 文件 — 并把项目以交互式时间线形式显示出来的工具。真正的查看器让你在计划上滚动、放大缩小、展开或折叠阶段，并通过悬停任务查看依赖关系、开始和结束日期以及资源。Ingantt 是一个基于浏览器的甘特图查看器兼编辑器，所以不管你手边是什么设备，都可以打开同一个文件 — 需要时还能把修改导出回 MS Project XML。

我可以在浏览器中打开 .MPP 文件或查看甘特图吗？

可以。Ingantt 完全在浏览器中运行 — Chrome、Safari、Firefox 或 Edge — 在 Mac、Windows、Linux、Chromebook、iPhone、iPad 或 Android 上都能用。用 Google 账号登录，上传 .MPP 文件或从 Google Drive 中选一份，整个项目就会在浏览器里打开，带有缩放、滚动和悬停查看详情。本地不装任何东西，也不下载任何东西。如果你在一台无法安装新软件的公司电脑上，或者在从来就没有 Microsoft Project 可选的 Linux 或 Chromebook 上，这就是正确的选择。

我可以打印甘特图，或把它导出成 PDF 或 PNG 吗？

三种都行。Ingantt 直接从浏览器打印甘特图，并能按任意缩放级别导出为 PDF 和 PNG — 不管你要的是单页概览还是跨多页的超大时间线，版式都会保持干净。PDF 常用于向不需要动原始文件的干系人做只读分享；PNG 方便放进幻灯片、Notion 文档或 Slack 消息；原生打印则是想要纸质副本时最快的方式。

在 Ingantt 打开 .MPP 文件后可以再保存回去吗？

不能保存为 .MPP — Ingantt 导出为 MS Project XML，Microsoft Project 可以直接打开。.MPP 二进制格式是专有的、未公开的，基本上只有 Microsoft 自己才能写；而 MS Project XML 是 Microsoft 自己的交换格式，可以干净地双向互通。所以你可以在 Ingantt 里打开 .MPP，做编辑，导出为 MS Project XML，再把这个 .xml 文件交给仍在用 Microsoft Project 的同事 — 他们会像打开其他项目文件一样打开它。对完全不使用 Project 的人做只读分享时，PDF 或 PNG 才是更合适的导出格式。

在 Microsoft Project 之外打开 .MPP 会丢失依赖关系、资源或关键路径吗？

在 Ingantt 里不会 — 对重要的字段来说，导入是无损的。任务、子任务、完成到开始的依赖关系、资源分配、工作量和工期、里程碑以及关键路径计算，全都会从原始 .MPP 文件中完整带过来。Ingantt 使用的项目结构和 Microsoft Project 内部用的一样，所以在 Project 里花了一下午搭出来的甘特图，在 Ingantt 中打开后还是同一张甘特图。只显示任务列表或把项目扁平化成图片的免费查看器会丢掉依赖图，而那通常是一份计划里最重要的部分。

Ingantt 提供 7 天免费试用，开始使用无需信用卡；试用期间包含完整的 .MPP 导入、查看、编辑、打印以及 PDF/PNG/XML 导出 — 不会在"查看版"和"编辑版"之间切一刀把功能锁起来。对大多数只需要打开或查看 .MPP 文件一两次的人来说，免费试用完全够用。试用期后，付费套餐起价 $5.99 每用户每月。如果你要跟仍在用 Microsoft Project 的供应商或客户协作，付费套餐的价格比单月的 Microsoft 365 Project 还便宜。
