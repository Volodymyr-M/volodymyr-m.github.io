2023年5月27日

新版本：PDF 打印

Ingantt 是一个专业级工具，其核心思想是利用广受认可的甘特图来进行排程。有时，在完成了全面的项目计划后，您可能需要向他人展示。

现在可以了！在主菜单中新增了“可打印 PDF”，使用它可以生成 PDF 文件，然后进行打印或与他人分享。生成的 PDF 完全反映了您所见的甘特图，并使用相同的选项。例如，如果您在 选项 对话框中选择显示根摘要任务（root summary task），那么打印版 PDF 也会同样显示。

生成的 PDF 文件可以用多种常见应用程序打开并打印，不仅适合打印，还非常方便通过电子邮件、即时通讯或云服务与其他人分享。

2023年5月5日

新版本：资源使用

Ingantt 推出了新功能，以在所有支持的平台上进一步提升其规划功能的实用性。

自开始以来，Ingantt 就具备检测并自动修复资源过度分配的能力。现在，您可以准确查看资源在哪些期间被过度分配、他们在每个任务上花费了多少时间，以及他们的总工作量是多少。这些都得益于 Ingantt 中名为 Resource Usage 的新视图。

乍看之下，该视图类似于标准分割显示的甘特图，但其实有一些差异。在左侧，按资源分组显示任务；右侧的图表中，您能看到每个资源在每个任务上投入的工作量，以及资源的总工作量。如果某个资源在特定时间段内过度分配，则对应数据会以红色高亮显示。

利用这个新功能，您可以更好地规划资源的工作负荷。我们也在准备更多的新功能，敬请期待！

2023年4月16日

新版本：MPP 文件导入

我们很高兴地宣布 Ingantt 的一次更新，其中增添了重要的新功能！

如您所知，Ingantt 能打开由 Microsoft Project 创建的 XML 文件。但是，这并非该流行应用的原生格式；它通常以名为 MPP（“.mpp” 扩展名）的二进制格式保存文件。

在这次更新中，您可以像打开 XML 文件一样打开 MPP 文件。只需在文件选择器中选择一个 MPP 文件，即可打开。

请注意一点：与 Ingantt 大部分无需联网的功能不同，打开 MPP 文件需要有效的网络连接，因为需要将这些文件上传到我们的服务器以转换成 XML。数据传输通过安全连接进行，而且文件不会被存储在服务器上，所以无需担心隐私和安全问题。

希望您会和我们一样喜欢 Ingantt 的这个新功能。更多新功能即将推出！

2023年4月12日

1.0 版本正式发布！

经过数月的努力，Ingantt 终于准备就绪！

Ingantt 是我们对易用、面向大众的 Microsoft Project 替代方案这一愿景的成果。它可以在各种设备上使用，价格也相对实惠。适用于绝大多数类型的项目规划，对那些需要规划的任务超出简单列表的人而言非常实用。

无论是大或小的翻新工程、软件开发项目、小型建筑项目、婚礼或活动，只要需要规划，Ingantt 都能帮助您回答最重要的问题：项目何时完成？每个任务何时结束？这些的总成本是多少？并且，还会考虑节假日、休假以及任何其它影响排期和成本的重要信息。

与一些规划工具不同，Ingantt 会给您一种高效工作的感觉。我们将执行常见操作所需的点击数降至尽可能低。除了传统的左侧任务列表、右侧甘特图（通过对话框编辑）的分屏布局，我们还在 Ingantt 中引入了一种特殊的编辑视图。您可以像使用 Photoshop 一样实时调整计划，所有更改都会立即显示在甘特图上，无需打开其他对话框。

如果您熟悉 Microsoft Project 等工具，可以直接利用已有经验快速上手。如果不熟悉，我们准备了详尽的文档{:target="_blank"}和入门指南{:target="_blank"}供您参考。

作为一款现代应用，Ingantt 支持深色模式（Dark Mode）。为方便您将文件保存到云端，Ingantt 对 Google Drive 提供了完整支持，您可以在 Google Drive 上存储和打开 Ingantt 文件。

Ingantt 已在 Microsoft Store 和 Google Play{:target="_blank"} 上线，支持 Windows 和 Android，同时还提供网页版。

您的任何反馈、问题或意见，都欢迎随时联系我们。这只是初版，Ingantt 仍在不断升级，新功能也在路上。请持续关注！
