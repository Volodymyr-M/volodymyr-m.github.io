常见问题

什么是 Ingantt for macOS？

Ingantt for macOS 是一个专业的项目管理工具，可以在您的 Mac 上原生运行。它具备 AI 辅助规划和 MS Project 兼容性，帮助您轻松创建并管理项目计划。

我可以导入已有的 Microsoft Project 文件吗？

可以！Ingantt 完全支持导入来自 Microsoft Project 的 .MPP 和 .XML 文件。项目数据（包括任务、资源和依赖关系）在导入过程中都会被保留。

AI 项目生成是如何工作的？

我们的 AI 助手可以根据简短的文字描述生成完整的项目计划。只需描述项目目标和需求，AI 就会根据最佳实践创建带有任务、时间线和依赖关系的结构化计划。

macOS 版本有免费试用吗？

没有。macOS 版本是一次性购买，没有订阅或免费试用。

我该如何购买 Ingantt for macOS？

请访问 Mac App Store 购买。购买后您可以永久使用，无需支付任何后续费用。
