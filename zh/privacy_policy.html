隐私政策

生效日期: 2026年2月22日 最后更新: 2026年2月22日

Ingantt Development（"我们"）运营 Ingantt 应用程序（"应用程序"）和位于 https://ingantt.com 的网站（"网站"）。本隐私政策描述了当您在所有受支持的平台（iOS、Android、macOS、Windows 和 Web，包括通过 Google Workspace Marketplace）上使用本应用程序时，我们如何收集、使用和共享信息。

1. 我们收集的信息

1.1 您提供的信息

  • **Google 账户信息：**如果您通过 Google 登录，我们会从 Google 获取您的电子邮件地址、显示名称和头像 URL。在原生平台（iOS、Android、macOS、Windows）上，登录是可选的，登录后可启用 Google Drive 文件存储。在 Web 版本上，Google 登录是必需的，用于 Google Drive 文件存储和订阅管理。当您首次使用 Web 版本时，我们的服务器上会创建一条用户记录，包含您的电子邮件地址、姓名、区域设置和注册日期。

  • **订阅和账单信息（仅限 Web）：**通过 Web 购买订阅时，我们会收集您的姓名、电子邮件地址、邮寄地址、电话号码和付款相关详情。此信息将传输至我们的支付处理商 WayForPay 进行交易处理。

  • **反馈：**当您通过应用程序提交反馈时，我们会收集您的消息文本。您可以选择附上当前打开的项目文件，以帮助我们诊断问题。

  • **AI 功能输入：**当您使用"AI 创建"功能时，您提供的文本提示将发送到我们的后端服务器，由 Anthropic (Claude) 处理。

  • **项目文件：**您创建和编辑的项目文件由您控制。文件存储在您设备本地或您的 Google Drive 账户中（名为"Ingantt Schedules"的文件夹）。当您打开非原生格式的文件（如 MPP、GAN、XER 或其他受支持的项目文件格式）时，该文件将传输到我们的服务器进行格式转换。文件仅在内存中处理，不会存储在我们的服务器上。

1.2 自动收集的信息

  • **应用程序元数据：**对我们后端的每个请求都包含您的应用程序版本、平台名称（iOS、Android、macOS、Windows 或 Web）、系统区域设置，以及（如果已登录）您的电子邮件地址。

  • **崩溃和错误报告：**我们使用 Sentry（托管在欧盟）在生产环境中收集崩溃报告和错误日志。这包括堆栈跟踪、错误消息、应用程序版本和平台信息。我们不会有意通过崩溃报告收集个人数据。

  • **Web 分析（仅限 Web）：**本应用程序的 Web 版本使用 Google Analytics 4 收集标准使用分析数据，包括页面浏览量和会话数据。Google Analytics 可能会在您的浏览器中设置 Cookie。原生移动和桌面版本的应用程序不包含分析跟踪。

  • **用户偏好设置：**本应用程序使用标准平台存储机制在您的设备上存储本地偏好设置（如主题、显示选项、列宽和最近打开的文件列表）。这些偏好设置不会传输到我们的服务器。

1.3 我们不收集的信息

  • 我们不会使用您的数据来训练人工智能或机器学习模型。
  • 我们不投放广告，也不集成任何广告 SDK。
  • 我们不收集位置数据、通讯录、照片、相机输入或麦克风输入。
  • 我们不使用设备标识符进行跟踪。
  • Android 云备份已对本应用程序明确禁用。

2. 我们如何使用您的信息

我们将收集的信息用于以下目的：

  • **提供和运营应用程序：**处理您的请求，通过 Google Drive 启用文件存储和同步，转换文件格式，并提供 AI 驱动的项目创建。
  • **处理订阅（仅限 Web）：**处理 Web 订阅付款，验证订阅状态，并管理您的账户。
  • **发送事务性邮件（仅限 Web）：**向您发送与账户相关的通知，如欢迎消息、试用提醒、订阅确认和取消通知。
  • **提高稳定性和性能：**分析崩溃报告、错误数据和服务器端遥测数据，以识别和修复错误。
  • **了解使用模式（仅限 Web）：**使用汇总的分析数据了解 Web 应用程序的使用方式，并改善用户体验。
  • **回应支持请求：**使用反馈提交来诊断问题并改进应用程序。

3. 我们如何共享您的信息

我们不会出售您的个人信息。我们仅在以下情况下共享信息：

  • **服务提供商：**我们使用以下第三方服务提供商来运营应用程序：

    • Sentry（Functional Software, Inc.）——客户端崩溃和错误报告，数据在欧盟处理
    • Microsoft Application Insights（Microsoft Corporation）——服务器端遥测和错误监控
    • Google Analytics（Google LLC）——Web 使用分析（仅限 Web 版本）
    • Google APIs（Google LLC）——身份验证和 Google Drive 文件存储
    • WayForPay ——订阅付款处理（仅限 Web 版本）
    • Resend（Resend, Inc.）——事务性邮件投递（仅限 Web 版本）
    • Anthropic (Claude) ——处理 AI 驱动的项目创建提示
    • Microsoft Azure ——后端托管和数据存储（美国东部区域）

  • **法律要求：**如果法律要求或根据有效的法律程序，我们可能会披露信息。

4. 数据存储和安全

  • **服务器端用户数据（仅限 Web）：**您的电子邮件地址、姓名、区域设置、注册日期和订阅状态存储在我们后端服务器的 Azure Table Storage 中。
  • OAuth 令牌（Google 访问令牌、刷新令牌和 ID 令牌）存储在平台安全存储中（iOS Keychain、Android Keystore、macOS Keychain 或平台等效机制）。
  • **订阅状态（仅限 Web）**缓存在本地安全存储中，供离线访问使用。
  • 对我们后端的 API 请求使用 HMAC-MD5 进行签名以验证完整性。
  • 本应用程序在可用的平台沙箱内运行（macOS App Sandbox、iOS App Sandbox）。
  • 我们所有的网络通信都使用 HTTPS。

4a. 数据保留

  • **服务器端用户记录（仅限 Web）：**您的用户记录和订阅历史在您的账户存在期间保留。如需请求删除您的数据，请联系 support@ingantt.com。
  • **项目文件：**提交用于格式转换的文件在内存中处理，不会保留。
  • **AI 提示：**为 AI 驱动的项目创建发送的文本提示不会存储在我们的服务器上。
  • **反馈：**反馈消息通过电子邮件转发给我们的支持团队，不会存储在数据库中。
  • **本地数据：**偏好设置、OAuth 令牌和缓存的订阅状态保留在您的设备上，直到您退出登录或卸载应用程序。

5. 您的选择和权利

  • **原生平台上的登录是可选的：**在 iOS、Android、macOS 和 Windows 上，您无需登录 Google 即可使用本应用程序。登录后可启用 Google Drive 文件存储。在 Web 版本上，Google 登录是应用程序功能、文件存储和订阅管理所必需的。
  • **Google Drive 访问权限：**本应用程序使用受限的 drive.file 权限范围，仅允许访问由应用程序创建的文件或您在应用程序中明确打开的文件——而非您的整个 Google Drive。
  • **反馈附件：**提交反馈时附上项目文件是可选的，需要您的明确同意。
  • **Web 分析：**您可以使用基于浏览器的 Cookie 阻止或退出工具来限制 Web 版本上 Google Analytics 的数据收集。
  • **事务性邮件：**在 Web 版本上，您将收到与账户相关的电子邮件（欢迎邮件、试用提醒、订阅通知）。这些是管理您的订阅所必需的服务通信。
  • **数据删除：**您可以随时在应用程序中退出 Google 账户，这将从设备中删除您存储的 OAuth 令牌。如需请求删除您在我们服务器上的用户记录和任何相关数据，请联系 support@ingantt.com。

6. 儿童隐私

本应用程序不面向 13 岁以下（或您所在司法管辖区适用年龄以下）的儿童。我们不会有意收集儿童的个人信息。如果您认为某个儿童向我们提供了个人信息，请联系 support@ingantt.com，以便我们采取适当措施。

7. 国际数据传输

我们的后端服务器托管在美国东部区域的 Microsoft Azure 上。崩溃报告由位于欧盟的 Sentry 处理。如果您位于美国以外，您的信息可能会被传输到美国或我们的服务提供商运营的其他司法管辖区并在那里进行处理。

8. 本隐私政策的变更

我们可能会不时更新本隐私政策。我们将通过更新本政策顶部的"最后更新"日期来通知您任何重大变更。在任何变更后继续使用本应用程序即表示您接受更新后的政策。

9. 特定平台披露

iOS 和 macOS (Apple)

  • 本应用程序是通过 Apple App Store 分发的付费应用程序。Apple 的隐私惯例适用于购买交易。
  • 本应用程序仅请求您通过系统文件选择器明确授予的文件访问权限。

Android

  • 本应用程序是通过 Google Play 分发的付费应用程序。Google 的隐私惯例适用于购买交易。
  • 本应用程序请求存储读写权限（用于本地文件操作）和互联网权限（用于网络访问）。

Windows

  • 本应用程序是通过 Microsoft Store 分发的付费应用程序。Microsoft 的隐私惯例适用于购买交易。

Web

  • Web 版本使用订阅模式。订阅付款通过 WayForPay 处理。
  • 本应用程序还可通过 Google Workspace Marketplace 获取。
  • Google Analytics Cookie 设置在 web.ingantt.com 域上。
  • Google OAuth 身份验证使用临时的 localStorage 条目进行登录回调，该条目在使用后立即清除。
