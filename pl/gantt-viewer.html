Przeglądarka Gantta — częste pytania

Jak otworzyć wykres Gantta bez Microsoft Project?

Otwórz plik w Ingantt. Bezpośrednio importuje natywny format .MPP Microsoft Project oraz MS Project XML, więc plik od kolegi używającego MS Project otwiera się jako w pełni interaktywny wykres Gantta — bez licencji MS Project ani instalacji Windows. Zadania, zależności i ścieżka krytyczna są zachowane.

Czy istnieje darmowa przeglądarka Gantta?

Ingantt oferuje 7-dniowy darmowy okres próbny bez karty kredytowej. Próba obejmuje pełne przeglądanie i edycję plików .MPP i MS Project XML na wszystkich platformach. Po próbie plany płatne zaczynają się od 5,99 USD za użytkownika miesięcznie. Większość jednorazowych potrzeb przeglądania mieści się w darmowej próbie.

Czy mogę wydrukować lub udostępnić oglądany wykres Gantta?

Tak. Ingantt eksportuje wykres Gantta do PDF lub PNG, abyś mógł go wydrukować, dołączyć do e-maila lub wkleić do prezentacji. Eksport PDF zachowuje układ osi czasu na dowolnym poziomie powiększenia. Jeśli musisz później otworzyć ten sam .MPP gdzie indziej, zobacz nasz przewodnik otwierania plików .MPP bez Microsoft Project.
