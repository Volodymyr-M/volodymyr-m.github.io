Odczytuje prawdziwy binarny .MPP

Analizuje natywny format .MPP Microsoft Project i MS Project XML bezpośrednio — nie podsumowanie listy zadań, nie zrzut ekranu. Zadania, podzadania, zależności, kamienie milowe i zasoby przechodzą bez strat.

Interaktywny, a nie obraz

Przybliżaj oś czasu, przewijaj fazy, rozwijaj lub zwijaj grupy, najedź kursorem na dowolne zadanie, aby zobaczyć szczegóły. To prawdziwy wykres Gantta — a nie płaski PDF, który możesz tylko przewijać.

Drukuj i eksportuj wszędzie

Drukuj wykres Gantta bezpośrednio z przeglądarki lub eksportuj do PDF i PNG na dowolnym poziomie przybliżenia — gotowe do dołączenia do e-maila, wstawienia do prezentacji lub powieszenia na ścianie.

Zapisz z powrotem do MS Project XML

Po obejrzeniu lub edycji wyeksportuj plik jako MS Project XML, aby współpracownik, który nadal używa Microsoft Project, mógł go otworzyć po swojej stronie — bez blokady formatu.

Jak otworzyć plik .MPP bez Microsoft Project?

Użyj Ingantt w przeglądarce. Odczytuje format binarny .MPP bezpośrednio — bez potrzeby posiadania licencji Microsoft Project, maszyny wirtualnej Windows ani usługi konwersji wysyłającej PDF-y mailem. Ingantt importuje każde zadanie, podzadanie, zależność, przypisanie zasobu, kamień milowy i ścieżkę krytyczną, a następnie renderuje projekt jako pełny interaktywny wykres Gantta. 7-dniowy darmowy okres próbny wystarcza w większości jednorazowych sytuacji otwierania pliku, a rozpoczęcie nie wymaga karty kredytowej.

Czym jest przeglądarka Gantta i czy Ingantt nią jest?

Przeglądarka Gantta to narzędzie, które otwiera plik wykresu Gantta — zwykle plik .MPP lub .XML Microsoft Project — i wyświetla projekt jako interaktywną oś czasu. Prawdziwa przeglądarka pozwala przewijać harmonogram, przybliżać i oddalać, rozwijać lub zwijać fazy oraz najeżdżać kursorem na zadania, aby zobaczyć zależności, daty rozpoczęcia i zakończenia oraz zasoby. Ingantt to przeglądarka i edytor Gantta działający w przeglądarce, więc możesz otworzyć ten sam plik na dowolnym urządzeniu, które masz pod ręką — i w razie potrzeby wyeksportować zmiany z powrotem do MS Project XML.

Czy mogę otworzyć plik .MPP lub wyświetlić wykres Gantta w przeglądarce?

Tak. Ingantt działa w całości w przeglądarce — Chrome, Safari, Firefox lub Edge — na Mac, Windows, Linux, Chromebook, iPhone, iPad lub Android. Zaloguj się za pomocą konta Google, prześlij plik .MPP lub wybierz go z Google Drive, a pełny projekt otworzy się w przeglądarce z przybliżaniem, przewijaniem i podglądem po najechaniu. Nic nie jest instalowane lokalnie i nic nie jest pobierane. To właściwa droga, jeśli jesteś na służbowej maszynie, na której nie możesz instalować nowego oprogramowania, lub na Linux lub Chromebook, gdzie Microsoft Project nigdy nie był opcją.

Czy mogę wydrukować wykres Gantta lub wyeksportować go do PDF lub PNG?

Tak, wszystkie trzy opcje. Ingantt drukuje wykres Gantta bezpośrednio z przeglądarki i eksportuje do PDF i PNG na dowolnym poziomie przybliżenia — więc niezależnie od tego, czy potrzebujesz jednostronicowego przeglądu, czy osi czasu wielkości ściany rozciągniętej na kilka stron, układ pozostaje czysty. PDF to zazwyczaj wybór do udostępniania tylko do odczytu interesariuszom, którzy nie muszą dotykać oryginalnego pliku; PNG jest przydatny do wstawiania do prezentacji, dokumentów Notion lub wiadomości Slack; natywne drukowanie jest najszybsze, gdy po prostu chcesz papierową kopię.

Czy mogę zapisać plik .MPP po otwarciu go w Ingantt?

Nie jako .MPP — Ingantt eksportuje do MS Project XML, który Microsoft Project otwiera bezpośrednio. Format binarny .MPP jest zastrzeżony, nieudokumentowany i w zasadzie stanowi cel zapisu tylko dla Microsoft; MS Project XML to własny format wymiany Microsoftu, który wymienia się bez strat. Możesz więc otworzyć .MPP w Ingantt, wprowadzić zmiany, wyeksportować jako MS Project XML i przekazać plik .xml współpracownikowi, który nadal używa Microsoft Project — otwiera go tak, jak każdy inny plik projektu. Do udostępniania tylko do odczytu osobom, które w ogóle nie używają Project, PDF lub PNG są właściwym eksportem.

Czy stracę zależności, zasoby lub ścieżkę krytyczną, otwierając .MPP poza Microsoft Project?

W Ingantt — nie. Import jest bezstratny dla pól, które mają znaczenie. Zadania, podzadania, zależności finish-to-start, przypisania zasobów, praca i czas trwania, kamienie milowe oraz obliczenia ścieżki krytycznej — wszystko przychodzi z oryginalnego pliku .MPP. Ingantt używa tej samej struktury projektu, której Microsoft Project używa wewnętrznie, więc wykres Gantta, którego budowa zajęła popołudnie w Project, nadal wygląda tak samo po otwarciu w Ingantt. Darmowe przeglądarki, które pokazują jedynie listę zadań lub spłaszczają projekt do obrazu, tracą graf zależności, który jest zazwyczaj najważniejszą częścią planu.

Czy istnieje darmowy program do otwierania .MPP lub przeglądarka wykresów Gantta?

Ingantt oferuje 7-dniowy darmowy okres próbny bez konieczności podania karty kredytowej, a okres próbny obejmuje pełen import .MPP, przeglądanie, edycję, drukowanie oraz eksport PDF/PNG/XML — nic nie jest zablokowane za podziałem "wersja przeglądarki" vs. "wersja edytora". Dla większości osób, które potrzebują otworzyć lub wyświetlić plik .MPP raz lub kilka razy, darmowy okres próbny w pełni pokrywa sytuację. Po zakończeniu okresu próbnego plany płatne zaczynają się od $5.99 za użytkownika miesięcznie. Jeśli pracujesz z dostawcami lub klientami, którzy nadal używają Microsoft Project, plan płatny kosztuje mniej niż jeden miesiąc Microsoft 365 Project.

Czy muszę znać zarządzanie projektami, aby korzystać z przeglądarki Gantta?

Nie. Ingantt jest zaprojektowany tak, aby interesariusz niebędący PM — dyrektor sprawdzający datę dostawy, klient sprawdzający kamień milowy, nowy członek zespołu wdrażający się do projektu — mógł otworzyć wykres Gantta i przeczytać go bez nauki czegokolwiek nowego. Przewijasz oś czasu, aby zobaczyć, kiedy co się dzieje, najeżdżasz kursorem na zadanie, aby zobaczyć, kto się nim zajmuje, i rozwijasz fazę, aby zobaczyć zadania wewnątrz. Nie ma żargonu do rozszyfrowania i żadnego kroku konfiguracji poza otwarciem pliku.
