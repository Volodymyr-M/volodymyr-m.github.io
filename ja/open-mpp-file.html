04.MPP ファイルの表示と閲覧 — 実際によくある質問

Microsoft Project なしで .MPP ファイルを開くには？

ブラウザで Ingantt を使いましょう。.MPP のバイナリ形式を直接読み込むので、Microsoft Project ライセンスや Windows 仮想マシン、PDF をメールしてくる変換サービスは不要です。Ingantt はすべてのタスク、サブタスク、依存関係、リソース割り当て、マイルストーン、そしてクリティカルパスをインポートし、プロジェクトを完全なインタラクティブガントチャートとしてレンダリングします。7日間の無料体験は単発でファイルを開きたい用途のほとんどに十分で、開始にクレジットカードは不要です。

ガントビューアとは何ですか、Ingantt はそれに該当しますか？

ガントビューアとは、ガントチャートファイル — 通常は Microsoft Project の .MPP または .XML ファイル — を開き、プロジェクトをインタラクティブなタイムラインとして表示するツールです。本物のビューアはスケジュールをスクロールし、ズームイン・アウトし、フェーズを展開・折りたたみ、タスクにホバーして依存関係、開始日・終了日、リソースを確認できます。Ingantt はブラウザベースのガントビューア兼エディタなので、手元にあるどんなデバイスでも同じファイルを開き、必要なときに MS Project XML として変更を書き戻せます。

.MPP ファイルやガントチャートをブラウザで開けますか？

はい。Ingantt は完全にブラウザ内で動作します — Chrome、Safari、Firefox、Edge が Mac、Windows、Linux、Chromebook、iPhone、iPad、Android 上で利用できます。Google アカウントでサインインし、.MPP ファイルをアップロードするか Google Drive から選べば、プロジェクト全体がブラウザで開き、ズーム、スクロール、ホバーで詳細表示が可能です。ローカルへのインストールもダウンロードもありません。新しいソフトウェアをインストールできない業務用マシンや、Microsoft Project が選択肢にない Linux・Chromebook を使っているなら、これが正しいルートです。

ガントチャートを印刷したり PDF や PNG にエクスポートしたりできますか？

はい、3つすべて可能です。Ingantt はガントチャートをブラウザから直接印刷し、任意のズームレベルで PDF と PNG にエクスポートします — 1ページの概要でも、複数ページにまたがる壁サイズのタイムラインでも、レイアウトはきれいに保たれます。PDF は原本に触れる必要のないステークホルダーとの読み取り専用共有に、PNG はスライドや Notion ドキュメント、Slack メッセージへの貼り付けに便利。ネイティブ印刷はハードコピーだけ欲しいときに最速です。

Ingantt で開いた .MPP ファイルをそのまま保存し直せますか？

.MPP としては保存できません — Ingantt は MS Project XML にエクスポートし、Microsoft Project がそれを直接開けます。.MPP バイナリ形式はプロプライエタリで非公開、実質的に Microsoft だけが書き込めるターゲットですが、MS Project XML は Microsoft 自身の交換用フォーマットで、きれいに往復できます。なので Ingantt で .MPP を開き、編集し、MS Project XML としてエクスポートして、その .xml ファイルを Microsoft Project を使っている同僚に渡せます — 彼らは普通のプロジェクトファイルと同じ手順で開けます。Project をまったく使わない人との読み取り専用共有には、PDF や PNG が適切なエクスポートです。

Microsoft Project の外で .MPP を開くと依存関係、リソース、クリティカルパスは失われますか？

Ingantt では失われません — 重要なフィールドについてインポートはロスレスです。タスク、サブタスク、終了-開始の依存関係、リソース割り当て、作業量と期間、マイルストーン、そしてクリティカルパス計算がすべて元の .MPP ファイルから取り込まれます。Ingantt は Microsoft Project が内部で使うのと同じプロジェクト構造を使用するため、Project で一日かけて組み立てたガントチャートは、Ingantt で開いた後も同じガントチャートに見えます。タスクリストだけを表示したり、プロジェクトを画像に平面化したりするフリーミアムのビューアでは、依存グラフ — たいてい計画のうち最も重要な部分 — が失われます。

無料の .MPP オープナーやガントチャートビューアはありますか？

Ingantt には7日間の無料体験があり、開始にクレジットカードは不要です。体験には完全な .MPP インポート、表示、編集、印刷、PDF/PNG/XML エクスポートが含まれます — 「ビューア版」と「エディタ版」の分割で機能が隠されることはありません。.MPP を一度か数回だけ開けばいい人にとっては、無料体験でほぼすべての用途を網羅できます。体験終了後の有料プランは1ユーザー月額 $5.99 から。Microsoft Project をまだ使うベンダーや顧客と仕事をするなら、有料プランは Microsoft 365 Project の1か月分より安くつきます。

ガントビューアを使うにはプロジェクト管理の知識が必要ですか？

いいえ。Ingantt は PM ではないステークホルダー — 納期を確認する役員、マイルストーンをチェックする顧客、状況をつかみたい新しいチームメンバー — がガントチャートを開き、何も新しく学ばずに読めるように設計されています。タイムラインをスクロールすれば何がいつ起きるか、タスクにホバーすれば誰が担当か、フェーズを展開すれば中のタスクがわかります。解読すべき専門用語もなく、ファイルを開く以上のセットアップもありません。
