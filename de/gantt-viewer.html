04Gantt-Viewer — Häufig gestellte Fragen

Was ist ein Gantt-Viewer?

Ein Gantt-Viewer ist ein Tool, das eine Gantt-Diagramm-Datei — meist eine Microsoft-Project-.MPP- oder -.XML-Datei — öffnet und das Projekt als interaktive Zeitachse anzeigt. Ein echter Viewer erlaubt Scrollen, Zoomen, Aufklappen von Phasen und das Anzeigen von Aufgabendetails wie Abhängigkeiten, Start- und Endtermin sowie Ressourcen. Ingantt ist ein Gantt-Viewer für Mac, iOS, iPad, Windows, Android und den Browser, sodass Sie dieselbe Datei auf jedem Gerät öffnen können.

Wie öffne ich ein Gantt-Diagramm ohne Microsoft Project?

Öffnen Sie die Datei in Ingantt. Es importiert das native .MPP-Format von Microsoft Project sowie MS Project XML direkt — eine Datei eines Kollegen mit MS Project öffnet sich als vollständig interaktives Gantt-Diagramm, ohne MS-Project-Lizenz oder Windows-Installation. Nutzen Sie die Browser-Version für einen einmaligen Bedarf oder installieren Sie die kostenlose Testversion für Mac, iOS, Windows oder Android. Aufgaben, Abhängigkeiten und der kritische Pfad kommen intakt an.

Kann ich eine .MPP-Datei auf dem Mac öffnen?

Ja. Ingantt bietet eine native macOS-App, die .MPP- und MS-Project-XML-Dateien direkt auf jedem aktuellen Mac öffnet. Sie brauchen keine virtuelle Maschine, Parallels oder eine Windows-Installation. Laden Sie Ingantt aus dem Mac App Store, öffnen Sie die .MPP-Datei, und Sie sehen das komplette Gantt-Diagramm mit Phasen, Aufgaben, Abhängigkeiten und kritischem Pfad. Wer nichts installieren möchte, nutzt die Browser-Version in Safari, Chrome oder Firefox.

Kann ich ein Gantt-Diagramm auf iPhone oder iPad ansehen?

Ja. Ingantt bietet native iOS-Apps für iPhone und iPad, die .MPP- und MS-Project-XML-Dateien wie die Mac-Version öffnen. Auf dem iPad gibt es eine volle Gantt-Zeitachse mit Pinch-to-Zoom und Touch-Scrollen; auf dem iPhone passt sich das Layout an, sodass Aufgaben und Meilensteine auch auf kleinem Bildschirm lesbar bleiben. Das ist der schnellste Weg, ein Gantt anzusehen, das Ihnen jemand unterwegs geschickt hat.

Gibt es einen kostenlosen Gantt-Viewer?

Ingantt bietet eine 7-tägige kostenlose Testversion ohne Kreditkarte. Die Testversion umfasst das vollständige Öffnen und Bearbeiten von .MPP- und MS-Project-XML-Dateien auf allen Plattformen — nichts ist hinter einer „Viewer-Edition" vs. „Editor-Edition" gesperrt. Nach dem Test kosten kostenpflichtige Pläne ab 5,99 € pro Nutzer und Monat. Die meisten einmaligen Anzeigen passen in die kostenlose Testphase.

Kann ich ein Gantt-Diagramm im Browser ohne Installation ansehen?

Ja. Ingantt bietet eine Browser-Version, die Gantt-Dateien in jedem modernen Browser — Chrome, Safari, Firefox oder Edge — auf Mac, Windows, Linux, Chromebook oder Mobilgerät öffnet. Melden Sie sich mit einem Google-Konto an, öffnen Sie eine .MPP-Datei aus Google Drive oder laden Sie eine hoch, und das Gantt-Diagramm erscheint im Browser mit vollem Zoom, Scrollen und Detailanzeige. Nichts wird installiert oder heruntergeladen.

Brauche ich Projektmanagement-Kenntnisse?

Nein. Ingantt ist so gestaltet, dass Nicht-Projektmanager — eine Führungskraft, die einen Liefertermin prüft, ein Kunde, der einen Meilenstein kontrolliert, ein neues Teammitglied zur Orientierung — ein Gantt öffnen und lesen können, ohne Neues zu lernen. Sie scrollen die Zeitachse, halten über einer Aufgabe, um zu sehen, wer daran arbeitet, und klappen eine Phase auf, um die Aufgaben darin zu sehen. Kein Fachjargon, keine Einrichtung.

Kann ich das angezeigte Gantt-Diagramm drucken oder teilen?

Ja. Ingantt exportiert das Gantt-Diagramm nach PDF oder PNG — zum Drucken, als E-Mail-Anhang oder für eine Folie. Der PDF-Export behält das Zeitachsen-Layout auf jeder Zoom-Stufe bei, was nützlich ist, wenn Sie eine schreibgeschützte Version an Stakeholder weitergeben. Wenn Sie dieselbe .MPP-Datei später an anderer Stelle öffnen müssen, siehe unsere Anleitung zum Öffnen von .MPP-Dateien ohne Microsoft Project.
