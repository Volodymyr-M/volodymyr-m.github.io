04.MPP-Dateien öffnen und ansehen — Fragen, die wirklich gestellt werden

Wie öffne ich eine .MPP-Datei ohne Microsoft Project?

Nutzen Sie Ingantt im Browser. Es liest das .MPP-Binärformat direkt — keine Microsoft-Project-Lizenz nötig, keine virtuelle Windows-Maschine, kein Konvertierungsdienst, der Ihnen ein PDF per E-Mail schickt. Ingantt importiert jede Aufgabe, Teilaufgabe, Abhängigkeit, Ressourcenzuweisung, jeden Meilenstein und den kritischen Pfad und rendert das Projekt als vollständiges interaktives Gantt-Diagramm. Die 7-tägige kostenlose Testphase reicht für die meisten einmaligen Situationen, in denen eine Datei zu öffnen ist, und für den Start ist keine Kreditkarte erforderlich.

Was ist ein Gantt-Viewer, und ist Ingantt einer?

Ein Gantt-Viewer ist ein Tool, das eine Gantt-Diagramm-Datei öffnet — meist eine Microsoft Project .MPP- oder .XML-Datei — und das Projekt als interaktive Zeitachse darstellt. Ein echter Viewer lässt Sie durch den Zeitplan scrollen, hinein- und herauszoomen, Phasen auf- und zuklappen und über Aufgaben fahren, um Abhängigkeiten, Start- und Endtermine sowie Ressourcen zu sehen. Ingantt ist ein browserbasierter Gantt-Viewer und -Editor, sodass Sie dieselbe Datei auf jedem verfügbaren Gerät öffnen können — und Änderungen bei Bedarf zurück als MS Project XML exportieren.

Kann ich eine .MPP-Datei öffnen oder ein Gantt-Diagramm im Browser ansehen?

Ja. Ingantt läuft vollständig im Browser — Chrome, Safari, Firefox oder Edge — auf Mac, Windows, Linux, Chromebook, iPhone, iPad oder Android. Melden Sie sich mit einem Google Account an, laden Sie die .MPP-Datei hoch oder wählen Sie sie aus Google Drive, und das vollständige Projekt öffnet sich im Browser mit Zoom, Scrollen und Details-beim-Hover. Nichts wird lokal installiert und nichts wird heruntergeladen. Das ist der richtige Weg, wenn Sie an einem Firmengerät arbeiten, auf dem Sie keine neue Software installieren dürfen, oder an einem Linux- oder Chromebook, für das Microsoft Project ohnehin nie eine Option war.

Kann ich mein Gantt-Diagramm drucken oder nach PDF oder PNG exportieren?

Ja, alle drei. Ingantt druckt das Gantt-Diagramm direkt aus dem Browser und exportiert auf jeder Zoomstufe nach PDF und PNG — ob Sie eine einseitige Übersicht oder eine wandgroße Zeitachse über mehrere Seiten brauchen, das Layout bleibt sauber. PDF ist die übliche Wahl für die schreibgeschützte Weitergabe an Stakeholder, die die Originaldatei nicht anfassen müssen; PNG eignet sich zum Einfügen in Präsentationen, Notion-Dokumente oder Slack-Nachrichten; der native Druck ist am schnellsten, wenn Sie einfach einen Ausdruck möchten.

Kann ich die .MPP-Datei nach dem Öffnen in Ingantt wieder speichern?

Nicht als .MPP — Ingantt exportiert nach MS Project XML, das Microsoft Project direkt öffnet. Das .MPP-Binärformat ist proprietär, undokumentiert und im Grunde ein Microsoft-eigenes Schreibziel; MS Project XML ist Microsofts eigenes Austauschformat und läuft sauber im Roundtrip. Sie können also eine .MPP in Ingantt öffnen, Änderungen vornehmen, als MS Project XML exportieren und die .xml-Datei an eine Kollegin weitergeben, die noch Microsoft Project nutzt — sie öffnet sie wie jede andere Projektdatei. Für die schreibgeschützte Weitergabe an Personen, die Project gar nicht einsetzen, ist PDF oder PNG der richtige Export.

Verliere ich Abhängigkeiten, Ressourcen oder den kritischen Pfad, wenn ich eine .MPP außerhalb von Microsoft Project öffne?

In Ingantt nicht — der Import ist verlustfrei für die Felder, auf die es ankommt. Aufgaben, Teilaufgaben, Finish-to-Start-Abhängigkeiten, Ressourcenzuweisungen, Arbeit und Dauer, Meilensteine und die Berechnung des kritischen Pfads kommen alle aus der Original-.MPP-Datei durch. Ingantt verwendet dieselbe Projektstruktur, die Microsoft Project intern nutzt, sodass ein Gantt-Diagramm, dessen Erstellung in Project einen Nachmittag gedauert hat, nach dem Öffnen in Ingantt immer noch wie dasselbe Gantt-Diagramm aussieht. Freemium-Viewer, die nur Aufgabenlisten zeigen oder das Projekt zu einem Bild flachdrücken, verlieren den Abhängigkeitsgraphen — meist der wichtigste Teil des Plans.

Gibt es einen kostenlosen .MPP-Öffner oder Gantt-Diagramm-Viewer?

Ingantt hat eine 7-tägige kostenlose Testphase, für die zum Start keine Kreditkarte erforderlich ist, und die Testphase umfasst vollständigen .MPP-Import, Ansehen, Bearbeiten, Drucken sowie PDF/PNG/XML-Export — nichts ist hinter einer „Viewer-Edition" vs. „Editor-Edition" verriegelt. Für die meisten, die eine .MPP-Datei nur ein- oder mehrmals öffnen müssen, deckt die kostenlose Testphase die Situation komplett ab. Nach der Testphase beginnen bezahlte Tarife bei 5,99 $ pro Nutzer und Monat. Wenn Sie mit Dienstleistern oder Kunden zu tun haben, die noch Microsoft Project einsetzen, kostet der bezahlte Tarif weniger als ein einzelner Monat Microsoft 365 Project.

Muss ich mich mit Projektmanagement auskennen, um einen Gantt-Viewer zu nutzen?

Nein. Ingantt ist so gestaltet, dass ein Stakeholder ohne PM-Hintergrund — eine Führungskraft, die einen Liefertermin prüft, ein Kunde, der einen Meilenstein checkt, ein neues Teammitglied, das sich orientiert — ein Gantt-Diagramm öffnen und lesen kann, ohne etwas Neues zu lernen. Sie scrollen die Zeitachse, um zu sehen, wann was passiert, fahren über eine Aufgabe, um zu sehen, wer daran arbeitet, und klappen eine Phase auf, um die enthaltenen Aufgaben zu sehen. Kein Fachjargon zum Entschlüsseln und kein Setup-Schritt außer dem Öffnen der Datei.
