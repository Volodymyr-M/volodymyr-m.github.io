.MPP-Dateien öffnen — die häufigsten Fragen

Wie öffne ich eine .MPP-Datei?

Der schnellste Weg ist, die Datei in Ingantt zu öffnen. Eine .MPP-Datei ist das native Binärformat von Microsoft Project, und standardmäßig öffnet nur Microsoft Project sie sauber — aber MS Project läuft nur auf Windows, kostet mehrere hundert Euro und lohnt sich kaum für eine einzelne Datei. Ingantt importiert .MPP direkt, funktioniert auf Mac, iPhone, iPad, Windows, Android und im Browser und bietet eine 7-tägige kostenlose Testversion ohne Kreditkarte. Öffnen Sie die App, ziehen Sie die .MPP hinein, und das vollständige Projekt erscheint in etwa 30 Sekunden.

Wie öffne ich eine .MPP-Datei ohne Microsoft Project?

Verwenden Sie Ingantt. Es liest das .MPP-Binärformat direkt — keine Microsoft-Project-Lizenz, keine virtuelle Windows-Maschine und kein Konvertierungsdienst, der Ihnen ein PDF per E-Mail schickt. Ingantt importiert jede Aufgabe, Teilaufgabe, Abhängigkeit, Ressourcenzuweisung, jeden Meilenstein und den kritischen Pfad und rendert das Projekt als vollständiges Gantt-Diagramm. Die 7-tägige kostenlose Testversion reicht für die meisten einmaligen Fälle, keine Kreditkarte erforderlich.

Wie öffne ich eine .MPP-Datei auf dem Mac?

Microsoft Project läuft nicht auf macOS — es gibt schlicht keine Mac-Version. Die Antwort ist daher nicht „Project auf dem Mac installieren", sondern „etwas nutzen, das .MPP nativ auf dem Mac liest". Ingantt hat eine native macOS-App im Mac App Store, die .MPP-Dateien direkt auf Apple-Silicon- und Intel-Macs öffnet. Laden Sie sie, öffnen Sie die .MPP, und das vollständige Gantt erscheint mit Aufgaben, Abhängigkeiten und kritischem Pfad. Alternativ funktioniert die Browser-Version in Safari, Chrome oder Firefox genauso.

Kann ich eine .MPP-Datei auf iPhone oder iPad öffnen?

Ja. Ingantt hat native iOS-Apps für iPhone und iPad im App Store, die .MPP- und MS-Project-XML-Dateien direkt öffnen. Auf dem iPad erhalten Sie eine breite Zeitachse mit Pinch-to-Zoom und Touch-Scrollen — das Nächste, was Sie zur Desktop-Microsoft-Project-Erfahrung auf einem iOS-Gerät bekommen. Auf dem iPhone passt sich das Layout dem kleineren Bildschirm an, sodass Aufgaben, Termine und Meilensteine lesbar bleiben. Oft der schnellste Weg, eine von einem Kollegen geschickte .MPP unterwegs zu öffnen.

Kann ich eine .MPP-Datei online im Browser öffnen?

Ja. Ingantt bietet eine Browser-Version, die .MPP-Dateien in jedem modernen Browser — Chrome, Safari, Firefox oder Edge — auf Mac, Windows, Linux, Chromebook oder Mobilgerät öffnet. Melden Sie sich mit einem Google-Konto an, laden Sie die .MPP-Datei hoch oder wählen Sie sie aus Google Drive, und das vollständige Projekt öffnet sich im Browser. Nichts wird lokal installiert. Der richtige Weg auf einem Arbeitsrechner, auf dem keine neue Software installiert werden darf, oder auf Linux/Chromebook, wo Microsoft Project ohnehin nie eine Option war.

Gibt es eine kostenlose Möglichkeit, .MPP-Dateien zu öffnen?

Ingantt bietet eine 7-tägige kostenlose Testversion ohne Kreditkarte, und die Testversion umfasst vollständigen .MPP-Import und die Bearbeitung — nichts ist gesperrt. Für die meisten, die eine .MPP nur einmal oder ein paar Mal öffnen müssen, deckt die Testversion die Situation komplett ab. Danach beginnen kostenpflichtige Pläne bei 5,99 € pro Nutzer und Monat. Wenn Sie regelmäßig mit .MPP-Dateien arbeiten, ist der Plan weniger als ein einziger Monat Microsoft 365 Project.

Gehen Abhängigkeiten, Ressourcen oder der kritische Pfad verloren?

In Ingantt nein — der Import ist verlustfrei für die wichtigen Felder. Aufgaben, Teilaufgaben, Finish-to-Start-Abhängigkeiten, Ressourcenzuweisungen, Arbeit und Dauer, Meilensteine und die Berechnung des kritischen Pfads kommen alle aus der Original-.MPP-Datei durch. Ingantt verwendet dieselbe Projektstruktur wie Microsoft Project intern, sodass ein Gantt, das in Project einen Nachmittag gekostet hat, nach dem Öffnen in Ingantt identisch aussieht. Freemium-Viewer, die nur Aufgabenlisten zeigen oder das Projekt zu einem Bild flachlegen, verlieren den Abhängigkeitsgraphen.

Kann ich die .MPP-Datei nach dem Öffnen wieder speichern?

Ja. Ingantt speichert Projekte zurück in das native .MPP-Format von Microsoft Project, außerdem in .XML, PDF und PNG. Sie können eine .MPP auf dem Mac, iPhone oder im Browser öffnen, bearbeiten, als .MPP speichern und die Datei an einen Kollegen mit Microsoft Project weitergeben — er öffnet sie wie gewohnt. Wenn Sie das Diagramm nur ansehen und weder bearbeiten noch neu speichern möchten, deckt unsere Gantt-Viewer-Seite den reinen Lesezugriff ab.
