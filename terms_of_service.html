1. Description of Service

Ingantt is a cross-platform project scheduling application that provides:

  • Project scheduling using the Critical Path Method (CPM) algorithm
  • Task management with dependencies, constraints, and resource assignments
  • Gantt chart visualization and rendering
  • Resource management and resource leveling
  • Calendar and work schedule management
  • Project import and export in XML (Microsoft Project compatible), CSV, PDF, and Markdown formats
  • Microsoft Project (.mpp) file conversion via our server
  • Google Drive integration for file storage and sharing
  • AI-powered project creation ("Create with AI")

The App is available on Web (web.ingantt.com and via Google Workspace Marketplace), iOS, Android, macOS, and Windows.
